Extrair Dados

Com esta ferramenta, você pode selecionar e baixar dados para uma área de interesse específica. As camadas que você selecionar serão adicionadas no arquivo ZIP, CSV ou KMZ.

Camadas para Extrair

Estas são as camadas na sua área de controle das quais você pode extrair feições. As camadas que estão selecionadas são extraídas e incluídas no arquivo ZIP ou CSV resultante.

Para uma camada aparecer na lista de camadas pra extrair, você deve ser proprietário dos dados ou o proprietário dos dados deve habilitar a exportação dos dados. Para habilitar a exportação dos dados de sua propriedade:

  • Abra a descrição de item dos dados.
  • Clique na guia Configurações.
  • Role para baixo até a seção Exportar Dados.
  • Marque Permitir aos outros exportar para diferentes formatos. Clique em Salvar ao finalizar.

Se a sua camada de análise tiver anexos, note que os anexos serão extraídos somente se o formato dos dados de saída for um arquivo geodatabase e as feições de Recorte estiverem desmarcadas.

Área de estudo

A área que define quais feições serão incluídas no arquivo ZIP ou CSV de saída. Você pode escolher do seguinte:

  • A extensão visível atual do seu mapa ( Mesmo da Visualização).
  • Uma das camadas na sua área de controle. Se a camada selecionada for uma feição de ponto ou linha, a extensão da camada será utilizada como área de estudo.
  • Desenhe uma área para definir a área de estudo.
Todas as feições completamente dentro da extensão de mapa serão extraídas. Para feições que cruzam ou interseccionam o limite da extensão do mapa, mas não estão completamente dentro da extensão, você tem duas opções:
  • Selecionar feições—Feições inteiras que cruzam o limite da área de estudo serão extraídas.
  • Recortar feições—Somente as porções de feições que estão dentro da área de estudo serão extraídas.

Formato dos dados de saída

O formato dos dados que serão extraídos. Ao exportar feições de área ou linha para arquivos CSV, somente atributos não espaciais são extraídos. Ao exportar feições de ponto, ambos os atributos espaciais e não espaciais são extraídos.

Os dados extraídos utilizarão a projeção Web Mercator 1984. Para arquivos CSV, campo Lat e Long serão anexados aos dados com coordenadas em Web Mercator 1984.

Os seguintes formatos de saídas são suportados:

  • Valores separados por vírgula (CSV)— .csv ou .zip
  • Arquivo geodatabase— .zip
  • Keyhole Markup Language (KML)— .kmz ou .zip
  • Shapefile— .zip

Nome do arquivo de saída

Este é o nome do item que será adicionado em Meu Conteúdo. O nome do item é baseado no Formato dos dados de saída escolhido. Se o item já existir, você será solicitado para fornecer outro nome.

Ao utilizar a caixa suspensa Salvar resultado em, é possível especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo.
