Geocodificar Locais da Tabela

Geocodificação converte endereços em coordenadas. Esta ferramenta converte endereços de uma tabela de compartilhamento de grandes dados em uma camada de feição com as coordenadas como colunas extras.

Escolher uma tabela de entrada

A tabela de entrada com os endereços a serem geocodificados.

Escolher um localizador

Escolha um serviço do localizador para geocodificar. Se o serviço do localizador que você deseja utilizar não estiver no menu suspenso, entre em contato com seu administrador para ter o serviço do localizador configurado para geocodificação em lote.

País

Se todos os seus endereços forem de um país, selecione este país para garantir os resultados mais precisos. Caso contrário, deixe este parâmetro vazio.

Selecionar Campos de Dados

O serviço do localizador espera determinados campos nos dados, como endereço, cidade, estado ou código postal. Combine os campos de dados no seu arquivo de entrada ou tabela com os campos que o localizador está esperando utilizando as caixas suspensas. Quanto mais campos você mapear, mais precisos serão os resultados. Se a tabela de entrada tiver todos os seus dados em uma coluna, selecione Campo Único, e escolha um campo de dados no menu suspenso. Caso contrário, selecione Múltiplos Campos, e mapeie os campos do localizador apropriados para as colunas de múltiplos campos na sua tabela.

Para cada local

Inclui campos adicionais retornado pelo serviço de localização. Se esta opção estiver marcada, todos os campos criados pelo localizador serão retornados. Se isto estiver desmarcado, seus dados de entrada serão retornados com as coordenadas geocodificadas.

Nome do resultado de saída

O nome da camada que será criada. Se você estiver gravando em um ArcGIS Data Store, seus resultados serão salvos em Meu Conteúdo e adicionados no mapa. Se você estiver gravando em um compartilhamento do arquivo de grandes dados, seus resultados serão armazenados compartilhamento do arquivo de grandes dados e adicionados em seu manifesto. Não serão adicionados no mapa. O nome padrão é baseado no nome da ferramenta e o nome da camada de entrada. Se a camada já existe, a ferramenta falhará.

Ao gravar no ArcGIS Data Store (armazenamento de grandes dados de espaço-tempo ou relacional) utilizando a caixa suspensa Salvar resultados em , você pode especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo.
