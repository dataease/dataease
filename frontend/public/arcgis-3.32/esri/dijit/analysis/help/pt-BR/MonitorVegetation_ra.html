Monitorar Vegetação

Esta ferramenta executa uma operação aritmética nas bandas de uma camada de raster de múltiplas bandas para revelar informações de cobertura da vegetação da área de estudo.

Selecionar os dados de entrada

Escolha a camada de raster de múltiplas bandas. Tenha certeza que o raster de entrada tem as bandas apropriadas disponíveis.

Escolher método para monitorar vegetação

Escolha o método criar a camada de índice da vegetação. Os diferentes índices de vegetação podem ajudar a destacar determinadas feições ou reduzir vários ruídos.

  • Global Environmental Monitoring Index — GEMI é um índice de vegetação não-linear para monitoramento ambiental global a partir de imagens de satélite. É semelhante ao NDVI, mas é menos sensível aos efeitos atmosféricos. Ele é afetado pelo solo bruto; portanto, não é recomendado para uso em áreas de vegetação escassa ou moderadamente densa.
  • Green Vegetation Index - Landsat TM — O GVI estava originalmente projetado de imagens Landsat MSS, mas foi modificado para utilizar com imagens Landsat TM. Também é conhecido como o índice de vegetação verde de Tasseled Cap da Landsat TM. Este índice de monitoramento também pode ser utilizado com imagens cujas bandas compartilham as mesmas características espectrais.
  • Modified Soil Adjusted Vegetation Index — MSAVI2 é um índice de vegetação que tenta minimizar influências do solo bruto do método SAVI.
  • Normalized Difference Vegetation Index — NDVI é um índice padronizado permitindo a você gerar uma imagem exibindo biomassa relativa, verde. Este índice aproveita o contraste das características de duas bandas de um conjunto de dados raster multiespectral: absorções de pigmento da clorofila na banda vermelha e a alta refletividade de materiais vegetais na banda de infravermelho próximo (NIR).
  • Perpendicular Vegetation Index — PVI é semelhante a um índice de vegetação por diferença; porém, é sensível às variações atmosféricas. Ao utilizar este método para comparar diferentes imagens, ele deve ser utilizado em imagens que foram atmosfericamente corrigidas. Estas informações podem ser fornecidas por seu fornecedor de dados.
  • Soil-Adjusted Vegetation Index — SAVI é um índice de vegetação que tenta minimizar influências de brilho do solo utilizando um fator de correção de brilho do solo. Isto é normalmente utilizado em regiões áridas onde a cobertura vegetativa é baixa.
  • Sultan's Formula — O processo da Fórmula de Sultan obtém uma imagem de 8 bits de seis bandas e aplica um algoritmo específico para produzir uma imagem de 8 bits de três bandas. A imagem resultante destaca formações de rochas denominadas ofiolitos em litorais. Esta fórmula foi projetada com base nas bandas TM e ETM de uma cena do Landsat 5 ou 7.
  • Transformed Soil-Adjusted Vegetation Index — SAVI-Transformado é um índice de vegetação que tenta minimizar influências de brilho do solo assumindo que a linha de superfície tem uma declividade arbitrária e intercepta.

Especificar índices das bandas NIR e Red

Especifique os índices de banda das bandas vermelhas e (NIR).

Cada sensor de satélite e câmera aérea capturam informações que são divididas em índices de banda. Cada índice de banda contém as informações para uma parte específica do espectro eletromagnético. Este método de monitoramento da vegetação exige que você especifique o índice de banda que capturou os comprimentos de onda vermelho e NIR.

Declividade da linha de superfície

A declividade da linha de superfície. A declividade é a relação linear aproximada entre as bandas NIR e Vermelha no gráfico de dispersão.

Este parâmetro é válido somente para o método de Índice de Vegetação Ajustado ao Solo Transformado.

Interceptar

O valor do NIR quando o valor de reflexão da banda vermelha (Vermelho) for 0 para as linhas do solo em particular.

(a = NIR - sRed) , quando Vermelho é 0.

Este parâmetro é válido somente para o método de Índice de Vegetação Ajustado ao Solo Transformado.

Quantidade da cobertura de vegetação verde

Especifica a quantia de cobertura de vegetação verde, que é exigido pelo método de SAVI.

Os valores válidos são como seguem:

  • 1 = Áreas sem cobertura de vegetação verde
  • 0.5 = Áreas com cobertura de vegetação verde moderada
  • 0 = Áreas com cobertura de vegetação verde alta

Fator de ajuste

Especifica o fator de ajuste para ajudar minimizar o efeito do solo, que é exigido pelo método Índice de Vegetação Ajustado ao Solo Transformado. O valor padrão é 0.08.

Um valor baixo significa que você está ignorando o efeito do solo. Um valor alto significa que o solo afetará seu resultado.

Nome da camada resultante

O nome da camada que será criada em Meu Conteúdo e adicionada no mapa. O nome padrão é baseado no nome da ferramenta e o nome da camada de entrada. Se a camada já existir, você será solicitado para fornecer outro nome.

Você pode especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo utilizando a caixa suspensa Salvar resultados em .
