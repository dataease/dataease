Converter Raster para Feição

Esta ferramenta converte um raster para um conjunto de dados raster, como pontos, linhas ou polígonos.

O parâmetro Escolher campo permite a você escolher qual campo de atributo do conjunto de dados raster de entrada se tornará um atributo na classe de feição de saída. Este parâmetro está disponível somente para rasters inteiros que têm atributos adicionais na tabela associada.

O raster de entrada pode ter qualquer tamanho de célula.

Os rasters inteiros podem ser convertidas em feições de ponto, linha ou polígono; porém, rasters de ponto flutuantes podem ser convertidos somente em feições de ponto.

Se Utilizar extensão de mapa atual estiver selecionada, somente as células de raster que são visíveis dentro da extensão de mapa atual serão convertidas. Se não selecionada, todas as células na camada de entrada serão convertidas, mesmo que estiverem fora da extensão de mapa atual. Tome nota disto se a camada de raster for grande ou tiver uma resolução muito alta.

Utilizar casos para esta ferramenta inclui o seguinte:

  • Criar feições de polígono a partir de um raster de classificação de uso do solo.
  • Os fluxos identificados em um DEM podem ser convertidos em feições de linha.
  • Um DEM pode ser convertido em feições de ponto que podem ser utilizadas em um fluxo de trabalho baseado em elementos diferentes do raster

Escolher camada raster para converter

A camada de raster de entrada que será convertida em camadas de feição.

Escolher campo

Forneça um campo que especifica qual valor será utilizado para a conversão.

Pode ser qualquer valor de texto ou inteiro.

Um campo contendo valores de ponto flutuante pode ser utilizado somente se a saída for para um conjunto de dados de ponto.

A escolha padrão é utilizar o campo Value , que contém o valor em cada célula de raster.

Escolher tipo de saída

Escolha o tipo de saída entre ponto, linha ou polígono.

  • Ponto—O raster será convertido em um conjunto de dados de ponto.
  • Linha—O raster será convertido em um conjunto de dados de feição de linha.
  • Polígono—O raster será convertido em um conjunto de dados de feição de polígono.

Simplificar linhas ou polígonos

Especifique se as feições devem ser simplificadas.

Se selecionada (o padrão), as feições de linha ou polígono serão suavizadas para produzir um resultado mais generalizado. Se esta opção não estiver selecionada, as feições ineares seguirão os limites da célula do conjunto de dados raster.

Nome da camada resultante

O nome da camada que será criada em Meu Conteúdo e adicionada no mapa. O nome padrão é baseado no nome da ferramenta e o nome da camada de entrada. Se a camada já existir, você será solicitado para fornecer outro nome.

Você pode especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo utilizando a caixa suspensa Salvar resultados em .
