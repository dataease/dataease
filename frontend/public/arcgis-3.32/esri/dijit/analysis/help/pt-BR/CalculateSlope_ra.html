Calcular Declividade

Esta ferramenta cria uma superfície que mostra a declividade dos dados de elevação de entrada. A declividade representa a taxa de alteração da elevação de cada célula do modelo de elevação digital (DEM).

Escolha os dados de entrada para calcular a declividade

A camada de elevação da qual a declividade será calculada.

Especificar fator de escala para elevação

Insira um fator Z.

O fator Z é um fator de escala utilizado para converter os valores de elevação para dois propósitos:

  • Para converter as unidades de elevação (como metros ou pés) em unidades de coordenadas horizontais da camada, que podem ser pés, metros ou graus
  • Para adicionar exagero vertical para efeito visual
