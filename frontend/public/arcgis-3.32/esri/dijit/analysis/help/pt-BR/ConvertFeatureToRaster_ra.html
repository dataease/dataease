Converter Feição para Raster

Esta ferramenta converte feições para um conjunto de dados raster.

Qualquer classe de feição contendo feições de ponto, linha ou polígono pode ser convertida em um conjunto de dados raster.

O centro da célula é utilizado para decidir o valor de pixel do raster de saída.

O tipo do campo de entrada determina o tipo do raster de saída. Se o campo for inteiro, o raster de saída será inteiro; se for ponto flutuante, a saída será ponto flutuante.

Escolher camada de feição para converter

A camada de feição de entrada para converter em um conjunto de dados raster.

Escolher campo

Escolha o campo que será utilizado para atribuir valores para o raster de saída.

Tamanho da Célula de Saída

O tamanho da célula e a unidade para os rasters de saída.

As unidades disponíveis são Milhas, Pés, Quilômetros e Metros. O padrão é Pés.

Nome da camada resultante

O nome da camada que será criada em Meu Conteúdo e adicionada no mapa. O nome padrão é baseado no nome da ferramenta e o nome da camada de entrada. Se a camada já existir, você será solicitado para fornecer outro nome.

Você pode especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo utilizando a caixa suspensa Salvar resultados em .
