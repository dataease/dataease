คำนวณระยะทาง

เครื่องมือนี้คำนวณระยะทาง Euclidean, ทิศทางและการจัดสรรจากต้นทางเดียวหรือชุดของต้นทาง

ตัวอย่างของการใช้งานรวมถึงการตอบคำถามต่าง ๆ ดังต่อไปนี้:

  • ระยะทางไปยังเมืองที่ใกล้ที่สุดคือเท่าใด
  • ระยะทางไปยังถนนที่ใกล้ที่สุดคือเท่าใด
  • ระยะทางไปยังแหล่งน้ำที่ใกล้ที่สุดคือเท่าใด

ถ้า ใช้ขอบเขตแผนที่ปัจจุบัน ถูกเลือก เฉพาะฟีเจอร์เหล่านั้นในชั้นข้อมูลอินพุึตที่มองเห็นได้ ที่อยู่ในขอบเขตแผนที่ปัจจุบันนี้เท่านั้น จะถูกวิเคราะห์ หากไม่เลือก ฟีเจอร์ทั้งหมดในชั้นข้อมูลอินพุตจะถูกนำมาวิเคราะห์ ถึงแม้ว่าฟีเจอร์เหล่านั้นจะอยู่นอกขอบเขตแผนที่ปัจจุบัน

เลือกแรสเตอร์ของชั้นข้อมูลตำแหน่งของฟีเจอร์เพื่อคำนวณระยะทางไปยัง

ชั้นข้อมูลที่กำหนดต้นทางที่จะใช้คำนวณระยะทาง ชั้นข้อมูลอาจเป็นบริการข้อมูลภาพหรือฟีเจอร์เซอร์วิส

สำหรับบริการข้อมูลภาพ ประเภทอินพุตอาจเป็นจำนวนเต็มหรือจุดลอยตัว

สำหรับฟีเจอร์เซอร์วิส อินพุตอาจเป็นจุด เส้น หรือหลายเหลี่ยม

ระยะทางสูงสุดที่จะคำนวณ

ระยะทางสูงสุดที่จะคำนวณ ตัวอย่างเช่น หากคุณป้อนค่าเป็น 10,000 เมตร ระยะทาง Euclidian จะถูกคำนวณจากตำแหน่งต้นทางไปยังค่านั้น (ระยะทาง)

หากไม่ได้ป้อนค่าใด ระยะทางจะถูกคำนวณไปยังขอบข่ายที่กำหนด

ชื่อของชั้นข้อมูลระยะทางผลลัพธ์

ชื่อของชั้นข้อมูลที่จะถูกสร้างขึ้นใน เนื้อหาของฉัน และเพิ่มไปยังแผนที่ ชื่อเริ่มต้นจะขึ้นอยู่กับชื่อเครื่องมือและชื่อชั้นข้อมูลอินพุต ถ้าชั้นข้อมูลนี้มีอยู่แล้ว คุณจะถูกขอให้ตั้งชื่อใหม่

เลือกฟิลด์การจัดสรร

ฟิลด์บนชั้นข้อมูลแหล่งอินพุตที่รวมค่าที่เป็นตัวกำหนดแต่ละแหล่ง ซึ่งอาจเป็นจำนวนเต็มหรือฟิลด์สตริงของชุดข้อมูลแหล่ง

ค่าเริ่มต้นคือ Value หากกำลังใช้ฟีเจอร์เซอร์วิส และไม่มีฟิลด์ Value ใดที่นำมาใช้ได้ จำนวนเต็มหรือฟิลด์สตริงแรกที่ว่างจะถูกนำมาใช้ หากไม่มีฟิลด์ที่ใช้ได้อยู่เลย ฟิลด์ ObjectID (ตัวอย่างเช่น OID หรือ FID ขึ้นอยู่กับประเภทของอินพุตฟีเจอร์) จะถูกนำมาใช้
