สร้างพื้นที่สันปันน้ำ

เครื่องมือนี้สามารถกำหนด พื้นที่สันปันน้ำ หรือพื้นที่ต้นน้ำ สำหรับแต่ละจุดของการนำเข้าชั้นข้อมูลได้

ตัวอย่างเช่น สมมุติว่าคุณมีจุดที่แทนตำแหน่งของการปนเปื้อนที่มากับน้ำ และคุณต้องการค้นหาแหล่งของการปนเปื้อนนั้น โดยแหล่งที่มาของการปนเปื้อนจะต้องอยู่ที่ใดที่หนึ่งเหนือพื้นที่สันปันน้ำจากจุดนี้ คุณสามารถใช้เครื่องมือนี้ในการกำหนดขอบเขตของพื้นที่รับน้ำซึ่งมีแหล่งที่มาของการปนเปื้อนอยู่ภายใน

ตำแหน่งของจุดนำเข้าของคุณสำคัญมาก หากจุดที่คุณวิเคราะห์ไม่ได้อยู่บนเส้่นทางระบายน้ำ พื้นที่รับน้ำที่ได้นั้นอาจเล็กมากและไม่มีประโยชน์ในงานวิเคราะห์ของคุณ ค้นหาระยะทางที่ใกล้ที่สุดในการระบายน้ำ เพื่อใช้ในการปรับตำแหน่งของจุดนำเข้า เส้นทางการไหลในการระบายน้ำ เพื่อหาลุ่มน้ำที่ไหลเข้ามาในจุดที่อยู่บนเส้นระบายน้ำ การปรับตำแหน่ง ใช้เพื่อหาสันปันน้ำ ที่ถูกเขียนขึ้นเป็นผลลัพท์ของชั้นข้อมูล – จุดการนำเข้าของคุณจะไม่ถูกแก้ไข

ผลลัพธ์ที่ได้จึงออกมาเป็นกลุ่มของชั้นข้อมูล ซึ่งมีชั้นข้อมูลอยู่ 2 ชั้นข้อมูล ได้แก่ พื้นที่สันปันน้ำ และจุดที่ถูกปรับตำแหน่งเพื่อใช้วิเคราะห์หาพื้นที่สันปันน้ำ

พื้นที่รับน้ำที่ได้จะมีรายละเอียดทั้งหมดของจุดที่คุณป้อนข้อมูล รวมถึงรายละเอียดเพิ่มเติมเหล่านี้:

  • PourPtID—มีรหัสจากจุดป้อนข้อมูล
  • Area Square Kilometers— —ประกอบด้วยพื้นที่ หน่วยตารางกิโลเมตรของต้นน้ำ สันปันน้ำ
  • Description และ DataResolution—มีข้อมูลที่เกี่ยวกับข้อมูลไฮดรอลิกที่ใช้คำนวณการติดตาม

ถ้า ใช้ขอบเขตแผนที่ปัจจุบัน ถูกเลือกไว้ จะมีการใช้ข้อมูลเพื่อการวิเคราะห์นี้เฉพาะข้อมูลที่อยู่ในพื้นที่จากขอบเขตแผนที่ที่เห็นอยู่ในปัจจุบันมาสร้างพื้นที่รับน้ำ หากไม่เลือก ข้อมูลทั้งหมดในชั้นข้อมูลการนำเข้าจะถูกนำมาวิเคราะห์ ถึงแม้ว่าข้อมูลเหล่านั้น จะอยู่นอกขอบเขตแผนที่ปัจจุบัน

ชิ้นข้อมูลจุด ใช้เพื่อคำนวณหาสันปันน้ำ

สันปันน้ำ (พื้นที่ต้นน้ำ) ที่จะนำมาคำนวณค่าในแต่ละจุด

นอกจากนี้ยังสามารถเลือกชั้นข้อมูลจากแผนที่ของคุณ โดยคุณสามารถเลือก เลือกชั้นข้อมูลวิเคราะห์ ในปุ่มด้านล่างของรายการดรอปดาวน์ ตัวเลือกนี้จะเป็นการเปิดแกลเลอรี ซึ่งคุณจะสามารถเรียกดูและเลือกชุดข้อมูลจากแหล่งต่างๆ เพื่อใช้ในการวิเคราะห์ของคุณได้

ระยะค้นหาแนวเส้นทางน้ำที่ใกล้ที่สุด

หากจุดที่นำเข้าของคุณอยู่ห่างจากเส้นทางน้ำ ผลลัพท์ของสันปันน้ำจะออกมาค่อนข้างเล็ก และไม่ค่อยนำมาใช้วิเคราะห์ เช่น การกำหนดแหล่งที่มาของต้นน้ำที่เป็นมลภาวะ กรณีส่วนใหญ่ คุณต้องการให้จุดนำเข้าของคุณเชื่อมต่อกับเส้นทางน้ำที่ใกล้ที่สุด เพื่อหาสันปันน้ำ ที่ไหลไปตามเส้นทางระบายน้ำ เมื่อต้องการค้นหาทางระบายน้ำที่อยู่ใกล้ที่สุด ให้ระบุระยะทางที่จะใช้ค้นหา ถ้าคุณไม่ได้ระบุระยะในการค้นหา เครื่องมือจะคำนวณและระยะในการค้นหาที่เก็บไว้

หากต้องการใช้ตำแหน่งตามที่คุณระบุเท่านั้น ให้กำหนดค่าระยะในการค้นหาเป็น 0

เพื่อวัตถุประสงค์ในการวิเคราะห์ Esri จะคำนวณเส้นทางระบายไว้ล่วงหน้าโดยใช้รูปแบบอุทกวิทยามาตรฐาน หากไม่มีแนวเส้นทางน้ำไหลในระยะค้นหา ตำแหน่งที่ได้จะวางลงบนตำแหน่งที่มีปริมาณน้ำสะสมสูงสุดภายในพื้นที่ระยะค้นหา

ชื่อของชั้นข้อมูลผลลัพธ์

เป็นชื่อชั้นข้อมูลที่จะถูกสร้างใน เนื้อหาของฉัน และเพิ่มเข้าไปในแผนที่ ชื่อเริ่มต้นจะขึ้นอยู่กับชื่อเครื่องมือและชื่อชั้นข้อมูลอินพุท ถ้าชื่อชั้นข้อมูลนี้มีอยู่แล้ว คุณจะถูกขอให้ตั้งชื่อใหม่

ใช้ บันทึกผลลัพธ์ใน เมนูแบบเลื่อนลง เพื่อระบุชื่อของโฟล์เดอร์ใน เนื้อหาของฉัน เมื่อบันทึกผล

การเรียกใช้เครื่องมือนี้จะไม่ใช้เครดิต
