Área de Separação de Águas

Esta ferramenta determina a área de contribuição acima de um conjunto de células num raster.

Uma watershed (bacia hidrográfica) é a área aclive que contribui com fluxo—geralmente de água—para um escoamento comum, como drenagem concentrada. Pode fazer parte de uma watershed maior e também pode conter watersheds menores, denominadas sub-bacias.

O valor de cada watershed será retirado do valor da origem na camada raster ou de ponto de escoamento de elementos de entrada. Quando o ponto de escoamento +e um conjunto de dados raster, os valores das células são utilizados por predefinição. Quando o ponto de escoamento é um conjunto de dados de elementos de ponto, os valores serão provenientes do campo especificado.

A ferramenta Watershed apenas suporta uma camada raster de fluxo de entrada D8.

Ao especificar os locais de pontos de escoamento de entrada como dados de elementos, o campo predefinido será o primeiro campo válido disponível. Caso não exista um campo válido, o campo ObjectID (por exemplo, OID ou FID) será o predefinido.

Selecionar camada de direção de fluxo

O raster de entrada que exibe a direção do fluxo a partir de cada célula.

O raster de direção de fluxo pode ser criado utilizando a ferramenta Flow Direction ou a tarefa FlowDirection no serviço RasterAnalysisTools, executado utilizando o tipo de direção de fluxo predefinido D8.

Especificar locais de pontos de escoamento de entrada

Os locais de pontos de escoamento de entrada

Para um raster, isto representa células acima das quais a área contribuinte, ou captação, será determinada. Todas as células que não sejam NoData serão utilizadas como células de origem. Para uma entrada de pontos, isto representa locais acima dos quais a área contribuinte, ou captação, será determinada.

Selecionar campo de ponto de escoamento

O campo utilizado para atribuir valores aos locais de pontos de escoamento.

Caso a camada de ponto de escoamento de entrada seja um raster, Value será utilizado. Caso seja um elemento, será utilizado um campo numérico. Caso o campo contenha valores de ponto flutuantes, estes serão truncados para números inteiros.

Nome da camada resultante

O nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Se a camada já existir, ser-lhe-á pedido que introduza outro nome.

É possível especificar o nome de uma pasta em O Meu Conteúdo, onde o resultado será guardado utilizando a opção Guardar Resultado em da caixa suspensa.
