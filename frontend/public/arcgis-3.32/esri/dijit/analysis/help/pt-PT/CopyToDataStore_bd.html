Copiar para Armazenamento de Dados

Esta ferramenta irá copiar uma camada de entrada para um armazenamento de dados. Pode copiar resultados para ArcGIS Data Store (armazenamento relacional ou espaciotemporal de big data) e será criada uma camada no seu portal ou pode copiar resultados para uma partilha de ficheiros de big data.

Por exemplo, poderia copiar elementos que se encontram armazenados numa partilha de dados big data para um armazenamento de dados relaciona e especificar que penas elementos incluídos na atual extensão de mapa serão copiados. Isto criaria um serviço de elementos alojados apenas com os elementos que se encontravam no interior da extensão de mapa definida.

Escolher camada a copiar

A camada ou tabela que será copiada.

Para além de escolher uma camada do seu mapa, pode escolher Escolher Camada de Análise na parte inferior da lista pendente para navegar até aos seus conteúdos de um conjunto de dados de partilha de ficheiros de big data ou camada de elementos.

Pode utilizar todos os elementos ou efetuar uma seleção (opcional)

Opcionalmente, utilize o botão de seleção para identificar as localizações a copiar.

Escolher o armazenamento de dados para o qual copiar

O armazenamento de dados para o qual a camada será copiada. Este parâmetro substituirá a seleção nos Ambientes de Análise. As opções são as seguintes:

  • ArcGIS Data Store ArcGIS Data Store
    • Espaciotemporal—Esta é a predefinição.
    • Relacional
  • Modelos de Produção de Partilha de Ficheiros de Big Data
    • Uma lista de modelos de partilha de ficheiros de big data disponíveis no formato bigDataFileShareName-templateName. Os modelos de partilha de ficheiros de big data têm de ser configurados antes da execução da ferramenta.

Nome da camada resultante

O nome da camada que será criada. Se estiver a escrever para ArcGIS Data Store, os seus resultados serão gravados em O Meu Conteúdo e adicionados ao mapa. Se estiver a escrever para uma partilha de ficheiros big data, os seus resultados serão guardados numa partilha de ficheiros big data e adicionados ao seu manifesto. Não serão adicionados ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Caso a camada já existe, a ferramenta irá falhar.

Ao escrever para ArcGIS Data Store (armazenamento relacional ou espaciotemporal de dados big data) utilizando a caixa suspensa Guardar resultado em pode especificar o nome de uma pasta em O Meu Conteúdo, onde será armazenado o resultado.
