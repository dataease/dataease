Calcular Densidade

Esta ferramenta cria um mapa de densidade a partir de elementos de ponto ou linha, espalhando quantidades conhecidas de determinado fenómeno (representados como atributos dos pontos ou linhas) pelo do mapa. O resultado é uma camada de áreas classificadas da menos densa para a mais densa.

Para a entrada de ponto, cada ponto deve representar a localização de algum evento ou incidente, e a camada resultado representa uma contagem do incidente por unidade de área. Um valor de densidade mais elevado numa nova localização significa que existem mais pontos perto dessa localização. Em muitos casos, a camada de resultado pode ser interpretada como uma superfície de risco para futuros eventos. Por exemplo, se os pontos de entrada representam locais de queda de relâmpagos, a camada resultado pode ser interpretada como uma superfície de risco para quedas de raios futuras.

Para a entrada de linha, a superfície da linha de densidade representa o valor total da linha que está perto de cada local. As unidades dos valores de densidade calculados são o comprimento da área de linhas-por-unidade. Por exemplo, se as linhas representam rios, a camada resultado representará o comprimento total dos rios que se encontram dentro do raio de busca. Este resultado pode ser usado para identificar as áreas que são hospitaleiras para animais de pasto.

Outros casos de uso desta ferramenta incluem o seguinte:

  • Criação de mapas de densidade de crime para ajudar os departamentos de polícia a alocar adequadamente os recursos para áreas de alta criminalidade.
  • Calculando densidades de hospitais dentro de um município. A camada resultado mostrará as áreas com acessibilidades alta e baixa para os hospitais, e esta informação pode ser usada para decidir onde o novo deve ser construído.
  • Identificar áreas que estão em risco elevado de incêndios florestais com base em locais históricos de incêndios florestais.
  • Localizar as comunidades que estão longe de auto-estradas, a fim de planejar onde as novas estradas devem ser construídas.

Selecionar ponto de entrada ou camada de linhas a partir da qual calcular densidade

A camada de entrada, de pontos ou de linhas, que será utilizada para calcular a densidade.

Use um campo de contagem

Fornecer um campo especificando o número de incidentes em cada local. Por exemplo, se tem pontos que representam as cidades, pode usar um campo que representa a população da cidade, como o campo de contagem, e a camada de densidade populacional resultante irá calcular as densidades populacionais maiores perto de cidades com populações maiores.

No caso de a opção predefinida Nenhum ser utilizada, assumir-se-á que cada local representa uma única contagem.

Distância de pesquisa

Introduza uma distância especificando o quão longe deve pesquisar para encontrar o ponto ou linha de elementos ao calcular os valores de densidade. Por exemplo, se proporcionar uma distância de busca de 10.000 metros, a densidade de qualquer local na camada de saída é calculada com base nas características que se encontram dentro dos 10.000 metros em redor do local. Qualquer local que não tenha nenhum incidente dentro de 10.000 metros receberá um valor de densidade de zero.

Se a distância não for fornecida, por omissão será calculado com base nos locais dos elementos de entrada e os valores do campo de contagem (se um campo de contagem é fornecido).

Unidades de área de saída

Especifique a unidade de área de saída. A densidade é dividida por contagem de área, e este parâmetro especifica a unidade de área no cálculo da densidade.

As unidades de área disponíveis são Milhas Quadradas, Pés Quadrados, Metros Quadrados e Quilómetros Quadrados.

Tamanho de célula de saída

O tamanho da célula e unidade para os rasters de saída.

Nome da camada resultante

O nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Se a camada já existir, ser-lhe-á pedido que introduza outro nome.

É possível especificar o nome de uma pasta em O Meu Conteúdo, onde o resultado será guardado utilizando a opção Guardar Resultado em da caixa suspensa.
