GeoAnalytics foca-se na análise de grandes quantidades de dados, enfatizando tanto os padrões espaciais como os temporais.

Summarize Data

The Summarize Data toolset contains tools that calculate total counts, lengths, areas, and basic descriptive statistics of features and their attributes within areas or near other features.

  • Aggregate Points calculates statistics about points that fall within specified areas or bins.
  • Build Multi-Variable Grid generates a grid of square or hexagonal bins and calculates values for each bin based on the proximity of each input layer.
  • Describe Dataset calculates a summary of your input, and can optionally generate a sample of your input layer.
  • Join Features calculates statistics about features that share a spatial, temporal, or attribute relationship with other features.
  • Reconstruct Tracks calculates statistics about points or polygons that belong to the same track and reconstructs inputs into tracks.
  • Summarize Attributes calculates statistics about feature or tabular data that share attributes.
  • Summarize Within calculates statistics for area features and attributes that overlap each other.

Agregar Pontos

Utilizando uma camada de elementos de pontos e uma camada de elementos de área ou caixas definidas por uma distância especificada, esta ferramenta determina os pontos que ficam dentro de cada área ou caixa e calcula estatísticas relativas a todos os pontos dentro de cada área ou caixa. Opcionalmente, pode aplicar divisão temporal com esta ferramenta.

Os seguintes são exemplos:

  • Localizações de determinados pontos de incidentes de crimes, conta o número de crimes por país ou outro distrito administrativo.
  • Encontrar as receitas mensais mais altas e mais baixas para localizações de frachises, utilizando caixas de 100-quilómetros .


Construi Grelha Multivariável

A ferramenta Construir Grelha Multivariável gera uma grelha de caixas quadradas ou hexagonais e calcula variáveis para cada caixa com base na proximidade de uma ou mais camadas de entrada.

Os seguintes são exemplos:

  • Tendo em conta várias camadas de infraestruturas de transportes públicos, que parte da cidade é menos acessível com recurso a transportes públicos?
  • Tendo em conta camadas de vias aquáticas, tais como lagos e rios, qual é o nome do corpo aquático mais próximo de cada localização nos E.U.A.?
  • Tendo em conta a camada de rendimento familiar, em que os E.U.A. representam a variação de rendimento, em que parte dos E.U.A. se encontra a maior variação de rendimento nas 50 milhas circundantes?


Descrever Conjunto de Dados

Resumir elementos de entrada com estatísticas, camadas de amostra e visualização. Pode selecionar emitir uma camada de amostra ou uma camada de extensão.

Os seguintes são exemplos:

  • Referencie o resumo das estatísticas calculadas para compreender que valores estão contidos no seu conjunto de dados.
  • Em vez de desenhar o conjunto de dados completo, crie uma camada subconjunto para visualizar elementos e os seus atributos, quando adicionados ao mapa.
  • Poupe tempo e recursos e corra a análise numa camada de amostra antes de correr a análise no conjunto de dados completo.


Ligar Elementos

Usando camadas de elementos ou dados tabulares, pode ligar elementos e registos com base em relações específicas entre as camadas ou tabelas de entrada. As ligações serão determinadas pelas relações espacial, temporal e de atributos, e, opcionalmente, podem ser calculadas estatísticas resumidas.

Os seguintes são exemplos:

  • Tendo em conta localizações de ponto de incidentes criminais com referência temporal, ligue os dados de crimes a si próprios, especificando uma relação espacial de crimes que ocorreram a 1 quilómetro uns dos outros, e que ocorreram com uma hora de diferença entre si, para determinar se existe uma sequência de crimes próximos uns dos outros, no espaço e no tempo.
  • Tendo em conta uma tabela de Códigos ZIP com informações demográficas e elementos de área que representam edifícios residenciais, ligue as informações demográficas às residências para que cada uma das residências passe a ter as informações.


Reconstruir Trilhos

Utilizando uma camada de elementos de ponto ou polígono com componente temporal ativa que represente um instante no tempo, esta ferramenta determina que elementos de entrada pertencem a um trilho e irá ordenar as entradas sequencialmente no tempo. Opcionalmente, são calculadas estatísticas para os elementos de entrada incluídos em cada trilho.

Os seguintes são exemplos:

  • Tendo em conta localizações de pontos e horas de medições de furacões, calcule a velocidade média do vento e a pressão máxima do vento do furacão.
  • Utilize medições de GPS de navios, para reconstruir rotas de navios como linhas.


Resumir Atributos

Utilizando dados de elementos ou de tabela, esta ferramenta resume estatísticas para campos especificados.

Os seguintes são exemplos:

  • Tendo em conta as localizações de mercearias com um campo COMPANY_NAME, resuma as lojas pelo nome de empresa para determinar estatísticas para cada empresa.
  • Tendo em conta uma tabela com mercearias com campos COMPANY_NAME and COUNTY, resuma as lojas pelo nome de empresa e por condado para determinar estatísticas para cada empresa em cada condado.


Resumir Dentro

Localiza áreas (e partes de áreas) que coincidem entre duas camadas e calcula estatísticas relativas à sobreposição.

Os seguintes são exemplos:

  • Dada uma camada com a área da watershed e uma camada de uso do solo com as áreas por tipo de uso, calcular área total do tipo de uso do solo para cada watershed.
  • Tendo em conta uma camada de parcelas num concelho e uma camada de limites urbanos, resumir o valor médio de parcelas vazias dentro de cada cidade.


Encontrar Locais

Estas ferramentas são utilizadas para identificar áreas que cumprem um determinado número de critérios diferentes definidos por si.

  • Detetar Incidentes funciona com camadas com componente temporal ativa para determinar que elementos correspondem a uma condição especificada.
  • Geocodificar Locais a partir de Tabela converte endereços em coordenadas.
  • Encontrar Locais Semelhantes encontra locais mais semelhantes a um ou mais locais de referência com base em critérios especificados.

Detetar Incidentes

Esta ferramenta funciona com uma camada com componente temporal ativa, de pontos, linhas ou áreas ou tabelas que representem um instante no tempo. Ao utilizar elementos ordenados sequencialmente, a que chamamos trilhos, esta ferramenta determina que elementos são incidentes de interesse. Os incidentes são determinados por condições especificadas por si.

Os seguintes são exemplos:

  • Detetar medições de furacões de Categoria 4 com base no atributo de velocidade do vento.
  • Utilizando medições de GPS de movimentos de navios com um campo que representa a velocidade do navio, encontre medições que sejam superiores a 1,5 a média dos cinco elementos anteriores.


Geocodificar Locais a partir de tabela

Converte endereços em coordenadas. Utilize esta ferramenta em tabelas de partilha de ficheiros big data.


Procurar Localizações Semelhantes

Com base em critérios especificados por si, encontre localizações semelhantes medindo a semelhança de localizações na sua camada de pesquisa de candidatos com uma ou mais localizações de referência.

Os seguintes são exemplos:

  • Encontrar as dez lojas mais semelhantes examinando o número de funcionários e as vendas anuais.
  • Encontrar as 100 cidades mais semelhantes examinando a relação entre população, crescimento anual e receitas fiscais.


Analisar Modelos

Estas ferramentas ajudam-no a identificar, quantificar e visualizar padrões espaciais nos seus dados.

  • Calcular Densidade considera quantidades conhecidas de determinado fenómeno e dispersa-as pelo mapa.
  • Localizar Hot Spots permite identificar clustering com relevância estatística no padrão espacial dos dados.
  • Encontrar Agrupamentos de Pontos encontra agrupamentos de elementos de pontos em ruído circundante com base na respetiva distribuição espacial.
  • Classificação e Regressão Baseadas na Floresta cria modelos e gera previsões utilizando uma adaptação do algoritmo floresta aleatória de Leo Breiman, que é um método de aprendizagem máquina supervisionado.
  • A Regressão Linear Generalizada gera previsões ou modela uma variável dependente no que respeita à sua relação com um conjunto de variáveis explanatórias.

Calcular Densidade

A ferramenta Calcular Densidade cria um mapa de densidade a partir de elementos de ponto espalhando quantidades conhecidas de determinado fenómeno (representados como atributos dos pontos) pelo mapa. O resultado é uma camada de áreas que representa a densidade.

Os seguintes são exemplos:

  • Calculando densidades de hospitais dentro de um município. A camada resultado mostrará as áreas com acessibilidades alta e baixa para os hospitais, e esta informação pode ser usada para decidir onde o novo deve ser construído.
  • Identificar áreas que estão em risco elevado de incêndios florestais com base em locais históricos de incêndios florestais.
  • Localizar as comunidades que estão longe de auto-estradas, a fim de planejar onde as novas estradas devem ser construídas.


Encontrar Clusters de Ponto

A ferramenta Encontrar Clusters de Ponto encontra clusters de elementos de ponto em ruído circundante com base na respetiva distribuição espacial.

Os seguintes são exemplos:

  • Encontrar clusters de agregados familiares afetados por pragas-para ajudar a focar esforços de erradicação.
  • Informar e tomar medidas referentes a necessidades de resgate e evacuação com base no tamanho e localização dos clusters, utilizando tweets geolocalizados que seguem perigos naturais ou ataques terroristas.


Localizar Hot Spots

A ferramenta Localizar Hot Spots permite determinar se existe algum tipo de clustering com relevância estatística no padrão espacial dos seus dados.

  • Os seus pontos (ocorrências de crime, árvores, acidentes de viação) estão efectivamente agrupados num cluster? Como pode ter a certeza?
  • Descobriu efectivamente um hot spot com relevância estatística (para despesas, mortalidade infantil, resultados de testes consistentemente elevados) ou será que o mapa contaria uma história diferente se alterasse a forma como está simbolizado?
A ferramenta Localizar Hot Spots ajuda-o a responder a essas perguntas com confiança.


Classificação e Regressão Baseadas na Floresta

A ferramenta Classificação e Regressão Baseadas na Floresta modela e gera previsões utilizando uma adaptação do algoritmo floresta aleatória de Leo Breiman, que é um método de aprendizagem máquina supervisionado.

Seguem-se exemplos:

  • Tendo em conta os dados sobre a ocorrência de erva marinha, bem como um número de variáveis explanatórias ambientais representadas por ambos os atributos, que foram enriquecidos utilizando uma grelha multivariável para calcular distâncias a fábricas a montante e portos principais, a ocorrência futura de erva marinha pode ser prevista com base em futuras previsões para essas mesmas variáveis explanatórias ambientais.
  • Suponha que tem dados sobre o rendimento das culturas em centenas de quintas através do país, bem como outros atributos sobre cada uma dessas quintas (número de trabalhadores, áreas, etc.). Utilizando estes dados é-lhe possível fornecer um conjunto de elementos representando quintas onde não se tenha o rendimento das culturas (possuindo, no entanto, todas as outras variáveis), e realizar uma previsão sobre o rendimento da cultura.
  • Os preços da habitação podem ser previstos baseados nos preços das casas que foram vendidas no ano presente. O preço das vendas das casas vendidas, bem como informação sobre o número de camas, distância a escolas, proximidade das autoestradas principais, rendimento médio e contagem de crimes pode ser utilizado para prever os preços de venda de casas similares.


Regressão Linear Generalizada

A ferramenta Regressão Linear Generalizada gera previsões ou modela uma variável dependente no que respeita à sua relação com um conjunto de variáveis explanatórias. Esta ferramenta pode ser utilizada para adaptar modelos contínuos (OLS), binários (logística) e de contagem (Poisson).

Seguem-se exemplos:

  • Quais as caraterísticas demográficas que contribuem para elevadas taxas de utilização de transportes públicos?
  • Existirá uma relação positiva entre vandalismo e assaltos?
  • Que variáveis podem efetivamente prever o volume de chamadas do 112? Dadas as projeções futuras, qual será a procura esperada para os recursos de resposta em emergência?
  • Quais as variáveis que afetam os índices de natalidade baixos?


Utilizar Proximidade

Estas ferramentas contribuem para responder a uma das questões mais comummente colocadas no âmbito da análise espacial: O que é que está perto do quê?

  • Criar Buffers cria áreas a uma distância especificada de elementos.

Criar Buffers

Um buffer é uma área que abrange uma determinada distância de um elemento de ponto, linha ou área.

Os seguintes são exemplos:

  • Utilizar elementos de rio lineares, aplicar buffer a cada rio por 50 vezes a largura do rio para determinar um limite ribeirinho proposto.
  • Tendo em conta áreas que representam condados, aplicar buffer a cada condado por 200 milhas náuticas para determinar o limite marítimo.


Manage Data

These tools are used for the day-to-day management of geographic and tabular data.

  • Append Data appends features to an existing hosted feature layer.
  • Calculate Field creates a new field or modifies an existing field and creates a new layer in your contents.
  • Clip Layer copies features that intersect a defined area of interest.
  • Copy to Data Store copies data to your ArcGIS Data Store and creates a layer in your contents.
  • Dissolve Boundaries merges areas that share a common boundary or attribute value.
  • Merge Layers copies all features from two layers into a single result layer.
  • Overlay Layers combines two or more layers into a single layer.

Copiar para Armazenamento de Dados

Copia uma camada de elementos de entrada ou tabela para um ArcGIS Data Store e cria uma camada nos seus conteúdos, em ArcGIS Enterprise.

Os seguintes são exemplos:

  • Copie uma coleção de ficheiros .csv numa partilha de ficheiros big data para o armazenamento de dados espaciotemporal, para visualização.
  • Copiar os elementos presentes na atual extensão do mapa, que se encontram armazenados no armazenamento de dados espaciotemporal para o armazenamento de dados relacional.


Calcular Campo

Calcula valores para um campo novo ou existente e cria uma camada nos seus conteúdos, em ArcGIS Enterprise.

Os seguintes são exemplos:

  • Modifique um campo existente denominado total para que seja a soma das receitas dos campos total_2016, total_2017, e total_2018.
  • Crie um novo campo para categorizar níveis de risco com base nos valores de campo, tais como windspeed e pollutant.


Camada de recorte

Recorta elementos de entrada de áreas de interesse definidas. O resultado de saída será um subconjunto de elementos de entrada.

Os seguintes são exemplos:

  • Cria um subconjunto de edifícios em risco que se encontram nas áreas de inundação.
  • Determina quais as estradas principais, que existem através das suas áreas potenciais de desenvolvimento.


Dissolver Limites

Funde elementos de área que se intersetam ou partilham um valor de campo comum para criar elementos contíguos ou elementos com múltiplas partes.

Os seguintes são exemplos:

  • Cria um elemento único representando o estado do Havai dissolvendo as fronteiras das ilhas do Havai.
  • Combina áreas de estudo similares baseadas no tipo de solo para emitir um conjunto de dados contíguo sobre tipo de solo.


Sobrepor Camadas

Combina duas ou mais camadas numa camada única. A sobreposição é utilizada para responder a uma das mais básicas questões em geografia, O que está por cima de o quê?

Seguem-se exemplos:

  • Que parcelas se encontram dentro da área de inundação de 100 anos? (Dentro é apenas outra maneira de dizer sobre.)
  • Que uso do solo se encontra naquele tipo de solo?
  • Que poços estão dentro de bases militares abandonadas?


Anexar Dados

Anexa elementos a uma camada alojada existente no seus conteúdos em ArcGIS Enterprise.

Os seguintes são exemplos:

  • Atualize a sua camada de elementos 2018_Sales todos os meses à medida que os seus relatórios ficam disponíveis.
  • Anexe conjuntos de dados de registo de furacões dos anos 2016 e 2017 num conjunto master de dados de furacões.
  • Recolha e anexe conjuntos de dados a partir de várias origens com esquemas diferentes utilizando cartografia de campos personalizada.


Juntar Camadas

Combina duas camadas de elementos para criar uma única camada resultante. Por definição todos os campos da camada fundida serão incluídos ou, então, é possível especificar regras de fusão personalizadas para definir o esquema resultante.

Os seguintes são exemplos:

  • Fundir três camadas contendo informação demográfica sobre a Inglaterra, País de Gales e Escócia para criar uma camada única da Grã-Bretanha.
  • Combina duas camadas contendo informação parcelar para municípios contíguos mantendo apenas os campos que, nas duas camadas, possuem o mesmo nome e tipo.


Melhoramento de Dados

Estas ferramentas ajudam a explorar as áreas.

  • Melhorar a Partir de Uma Grelha de Múltiplas Variáveis junta atributos de uma camada de grelha de múltiplas variáveis a uma camada de pontos.

Melhorar a Partir de Uma Grelha de Múltiplas Variáveis

Junta eficientemente os atributos de uma grelha de múltiplas variáveis a uma camada de pontos, permitindo-lhe adicionar rapidamente uma coleção de informações grande e diversificada aos dados de pontos para utilizar em outras análises espaciais.

Os seguintes são exemplos:

  • Dada uma camada contendo milhões de incidentes de quebras de energia, enriquecer os elementos do incidente com informações sobre utilização típica, riscos ambientais, e condições da infraestrutura, de modo a estudar a relação entre estes fatores e a frequência de quebras de energia.
