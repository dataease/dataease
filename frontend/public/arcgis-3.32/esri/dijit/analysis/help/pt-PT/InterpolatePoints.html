Interpolar Pontos

A ferramenta Interpolar Pontos permite prever valores em novos locais baseados em medições de um conjunto de pontos. A ferramenta leva dados de ponto com valores em cada ponto e retorna zonas classificadas pelos valores previstos. Por exemplo

  • A gestão da qualidade do ar de um distrito tem sensores que medem os níveis de poluição. Interpolar Pontos pode ser usados ​​para prever os níveis de poluição em locais que não possuem sensores, tais como locais com populações de risco-escolas ou hospitais, por exemplo.
  • Prever concentrações de metais pesados ​​em culturas com base em amostras colhidas das plantas individuais.
  • Prever os níveis de nutrientes do solo (nitrogénio, fósforo, potássio, etc) e outros indicadores (como condutividade eléctrica), a fim de estudar as suas relações com o rendimento das culturas e prescrever quantidades precisas de fertilizantes para cada local no campo.
  • Aplicações meteorológicas incluem previsão de temperaturas, precipitação e variáveis ​​associadas (como a chuva ácida).

Erros de previsão de saída

Se marcada, será criada uma camada de erros por omissão para as previsões de interpolação. O nome da camada de saída será o nome da camada resultado com 'Erros' anexado.

Os erros por omissão são úteis porque fornecem informações sobre a fiabilidade dos valores previstos. Uma regra simples é que o verdadeiro valor vai cair dentro de dois desvios-padrão do valor previsto de 95 por cento do tempo. Por exemplo, suponha que um novo local recebe um valor previsto de 50 com um erro padrão de 5. Isto significa que a melhor estimativa deste instrumento é que o valor real no local é de 50, mas que poderia razoavelmente ser tão baixo quanto 40 ou tão alto como 60. Para calcular esta gama de valores razoáveis​​, multiplicar o erro padrão por 2, adicionar este valor ao valor previsto para receber a extremidade superior da gama, e subtrair o valor previsto para receber a extremidade inferior do intervalo.

Classificar Por

Escolha como os valores previstos serão classificados em áreas.

  • Intervalos geométricos—Áreas são baseadas em intervalos de classe que têm uma série geométrica. Este método assegura que cada intervalo de classe tem aproximadamente o mesmo número de valores dentro de cada classe, e que a variação entre os intervalos é consistente.
  • Intervalo iguais—Áreas são criadas de tal modo que a gama de valores previstos é igual para cada área.
  • Área igual - as áreas são criadas de forma a que o número de valores de dados em cada área seja igual. Por exemplo, se os dados tem mais valores grandes do que os valores pequenos, mais áreas serão criadas para valores grandes.
  • Manual-Esta opção permite que defina o seu próprio intervalo de valores para as áreas. Estes valores serão inseridos no parâmetro Valores de quebra de classe abaixo.

Número de Classes

Especifica o número de classes (intervalo de valores previstos) na camada de resultado. Cada classe define o limite das áreas. Os valores destas classes serão determinados pelo parâmetro Classifique acima.

Valores de classes de intervalo

Introduza os valores de quebra de classe pretendidos separados por espaços. Por exemplo, se quer quebra de classe nos valores de 1.5, 3 e 7, digite 1.5 3 7. Estes valores irão definir o limite superior de cada classe, de forma a que o número de classes seja igual ao número de valores introduzidos. Não serão criadas áreas para as localizações com valores estimados acima do maior valor de quebra de classe introduzido. Deve digitar pelo menos dois valores e nunca mais de 32.

Recortar saída para

Ou desenhar ou fornecer uma camada especificando a área onde deseja que o resultado a ser desenhado. Por exemplo, se estiver a interpolar medições da poluição de um lago, pode usar o limite do lago neste parâmetro e a saída só vai desenhar dentro do limite do lago.

Prever nestes locais

Ou desenhar ou fornecer uma camada especificando locais de ponto para calcular os valores de previsão. Isso permite que faça previsões em locais específicos de interesse. Por exemplo, se a camada de entrada representa medições de níveis de poluição, pode usar este parâmetro para prever as localizações dos níveis de poluição com elevado risco para as populações, tais como escolas ou hospitais. Pode então usar essas informações para dar recomendações aos funcionários da saúde nesses locais.

Se for fornecido, esta ferramenta irá imprimir uma camada de ponto de previsões, nos locais especificados. O nome da camada de saída será o nome da camada resultado com 'Pontos' anexado.

Nome da camada resultante

Este é o nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Se o nome da camada já existir, ser-lhe-á pedido que introduza outro nome.

Ao utilizar o menu suspenso Guardar resultado em pode definir o nome da pasta em O Meu Conteúdo onde o resultado será guardado.
