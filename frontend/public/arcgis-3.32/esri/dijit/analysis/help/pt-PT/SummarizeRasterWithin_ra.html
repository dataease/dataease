Resumir Raster Em

Esta ferramenta resume um raster com base em áreas (zonas) definidas pela primeira camada de entrada.

Alguns exemplos de aplicações incluem as seguintes:

  • Dada uma camada com os limites das bacias hidrográficas e uma camada de uso do solo com os limites por tipo de uso, calcular área total do tipo de uso do solo para cada bacia hidrográfica.
  • Dada uma camada de parcelas num município e uma camada com o limite das freguesias pertencentes a esse município , resumir o valor médio das parcelas vagas dentro de cada freguesia.

Se Usar extensão de mapa actual é validado, apenas esses elementos na camada de entrada e da camada a ser resumida que são visíveis dentro do mapa de extensão actual serão analisados. Se não validado, todos os elementos tanto na camada de entrada e na camada a serem resumidos vão ser analisados, mesmo que se encontrem fora da extensão do mapa actual.

Selecionar área para resumir uma camada de raster no interior dos limites definidos

A camada que define os limites das áreas, ou zonas, que serão resumidas.

A camada pode ser de dados raster ou de elementos. Para rasters, as zonas são definidas com base em todas as localizações na entrada que possuem o mesmo valor de célula. As áreas não têm de ser contíguas.

Escolher campo para definir os limites

Este é o atributo da camada que será utilizado para definir os limites das áreas.

Por exemplo, suponha que a primeira camada de entrada define os limites de unidades de gestão, com atributos que definem a região, o distrito e a ID de parcela de cada unidade. Tem ainda uma camada raster que define um índice de biodiversidade para cada local. Com o campo que selecionar, pode decidir calcular a biodiversidade média a nível local, distrital ou regional.

Escolher camada de raster para resumir

As células raster nesta camada serão resumidas pelas áreas (zonas) em que se incluem.

Escolher estatística para calcular

Após selecionar uma camada para resumir, pode opcionalmente calcular estatísticas de qualquer atributo numérico dos pontos, linhas ou áreas incluídos na camada de área de entrada.

Os tipos de estatísticas disponíveis quando o campo selecionado é inteiro são Média, Maioria, Máximo, Mediana, Mínimo, Minoria, Intervalo, Desvio padrão, Soma e Variedade.

Caso o campo seja um ponto flutuante, as opções são, Média, Máximo, Mínimo, Intervalo, Desvio padrão e Soma.

Ignorar valores em falta no cálculo

Caso opte por ignorar valores em falta, apenas as células que têm um valor na camada a ser resumido serão utilizadas para determinar o valor de saída para a respetiva área. Caso contrário, caso existam valores em falta em qualquer parte da área, considera-se que a informação existente é insuficiente para efetuar cálculos estatísticos para todas as células nessa zona, e essa área receberá um valor nulo (NoData) na saída.

Nome da camada resultante

O nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Se a camada já existir, ser-lhe-á pedido que introduza outro nome.

É possível especificar o nome de uma pasta em O Meu Conteúdo, onde o resultado será guardado utilizando a opção Guardar Resultado em da caixa suspensa.
