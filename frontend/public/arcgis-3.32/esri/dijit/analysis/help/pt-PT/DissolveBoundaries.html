Dissolver Limites

Áreas que se sobrepõem ou partilham um limite são fundidas de forma a formar uma única área.

Pode controlar quais os limites a serem fundidos, especificando um campo Por exemplo, se tiver uma camada de concelhos e cada concelho tiver um atributo State_Name, pode dissolver limites utilizando o atributo State_Name. Concelhos adjacentes serão combinados se tiverem o mesmo valor de State_Name. O resultado final é uma camada com todos os limites fundidos.

Escolher camada de área cujos limites se dissolverão

A camada contendo elementos de área que será disolvida.

Além de escolher uma camada do seu mapa, pode selecionar Escolher Camada de Análise que se encontra na parte inferior da lista pendente. Esta opção abre uma galeria, na qual pode navegar e selecionar conjuntos de dados de várias fontes para utilizar na sua análise.

Escolher método Dissolver

Pode escolher um dos dois métodos para fundir a área dos elementos:

  • Áreas que se sobrepoem ou são adjacentes—Duas ou mais áreas cujos limites sobrepõem uns aos outros serão fundidas numa área. Além disso, as áreas que são adjacentes (isto é, que partilham um limite) serão fundidos apenas numa área.
  • Áreas com o mesmo campo de valor—Áreas que se sobrepoem ou partilham uma fronteira comum serão fundidas numa área se tiverem o mesmo valor do campo.

    Por exemplo, se tiver uma camada de concelhos, e cada concelho tiver um campo State_Name pode dissolver limites utilizando o campo State_Name . Concelhos adjacentes serão combinados se tiverem o mesmo valor de State_Name. O resultado final é uma camada com todos os limites fundidos.

Se a opção Utilizar a extensão atual do mapa estiver selecionada, serão dissolvidos apenas os elementos da camada de entrada que sejam visíveis na atual extensão de mapa. Se não estiver validado, todos os elementos na camada de entrada serão disolvidas, mesmo que estejam fora da extensão de mapa actual.

Criar Elementos de Várias Partes

Em ArcGIS, uma representação digital de um local ou coisa que tem mais do que uma parte, mas é definida como um elemento porque referencia um conjunto de atributos. Numa camada de estados, por exemplo, o estado do Havai poderia ser considerado um elemento de várias partes porque as respetivas partes geométricas separadas são classificadas como um único estado. Este parâmetro booleano oferece duas opções:

  • Verdadeiro (marcado): o serviço de saída pode conter elementos de várias partes.
  • Falso (não marcado): o serviço de saída apenas conterá elementos únicos, e serão criados elementos individuais para cada parte.

Adicionar estatística

Quando as áreas são fundidas, é possível resumir os respectivos atributos. Por exemplo, se estiver a dissolver o limite entre concelhos com base em State_Name, e cada concelho tiver um campo Population , pode calcular a soma de Population do estado.

Tipos de estatística disponíveis:

  • Soma—Adiciona o valor total para o campo especificado
  • Média—Calcula a média para o campo especificado
  • Minímo—Localiza o menor valor para todos os registros do campo especificado
  • Máximo—Encontra o valor maior para todos os registros do campo especificado
  • Desvio -padrão-Encontra o desvio-padrão dos valores do campo especificado

Nome da camada resultante

Este é o nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Se o nome da camada já existir, ser-lhe-á pedido que introduza outro nome.

Ao utilizar o menu suspenso Guardar resultado em pode definir o nome da pasta em O Meu Conteúdo onde o resultado será guardado.
