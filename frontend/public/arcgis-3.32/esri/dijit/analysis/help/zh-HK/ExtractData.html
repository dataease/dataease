擷取資料

使用此工具可以選擇與下載特定感興趣區域的資料。所選圖層將被新增到 ZIP、CSV 或 KMZ 檔案中。

待擷取圖層

它們是內容表中的圖層，可從中擷取圖徵。勾選的圖層將被擷取到生成的 ZIP 檔案或 CSV 檔案中。

要擷取顯示在圖層列表中的圖層，您必須擁有資料或資料的所有者必須啟用資料匯出功能。要啟用所擁有數據的匯出功能：

  • 請開啟資料的項目描述。
  • 按一下 設定頁籤。
  • 向下捲動至 匯出資料部分。
  • 請勾選 允許其他使用者匯出不同格式. 完成時，按一下 儲存。

如果您的分析圖層包含附件，則請注意，僅當輸出資料格式為檔案地理資料庫並且未勾選 裁剪圖徵時才能擷取附件。

研究區域

該區域定義輸出 ZIP 檔案或 CSV 檔案中所包含的圖徵。可選擇的範圍有：

  • 地圖的目前可見範圍（ 與顯示內容相同）。
  • 內容列表中的某個圖層。如果所選圖層是點或線圖徵，則會將圖層的範圍作為研究區域。
  • 繪製一個區域以定義研究區域。
將擷取所有完全位於地圖範圍內的圖徵。對於跨越地圖範圍邊界或與地圖範圍邊界相交，但沒有完全位於範圍之內的圖徵，您有兩種選擇：
  • 選擇圖徵 - 擷取跨越研究區域邊界的整個圖徵。
  • 裁剪圖徵 - 僅擷取圖徵位於研究區域內的相應部分.

輸出資料格式

待擷取資料的格式。在將線圖徵或區域圖徵匯出為 CSV 檔案時，只擷取非空間屬性。匯出點圖徵時，擷取空間和非空間屬性。

擷取的資料將使用 Web Mercator 1984 投影。對於 CSV 檔案，會將 Lat 和 Long 欄位附加到 Web Mercator 1984 中具有座標的資料。

支援下列輸出格式：

  • 逗號分隔值 (CSV) - .csv 或 .zip
  • 檔案地理資料庫 - .zip
  • Keyhole 標記語言 (KML) - .kmz 或 .zip
  • Shapefile - .zip

輸出檔案名稱

此為將新增到 我的內容中的項目名稱。項目名稱基於所選的 輸出資料格式。如果該項目已存在，則系統將要求提供其他名稱。

使用 將結果儲存到下拉式方塊，可以在 我的內容中指定儲存結果的資料夾名稱。
