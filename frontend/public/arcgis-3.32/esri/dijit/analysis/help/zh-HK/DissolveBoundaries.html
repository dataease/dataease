融合邊界

將重疊區域或分享邊界的區域進行合併，以形成單個區域。

您可以透過指定欄位來控制合併哪些邊界。例如，如果有一個縣圖層，且每個縣具有 State_Name 屬性，則可以使用 State_Name 屬性來融合邊界。如果相鄰的縣具有相同的 State_Name 值，則可對它們進行合併。最終結果是一個州邊界圖層。

選擇其邊界將進行融合的區域圖層

包含將執行融合的區域圖徵的圖層。

除了從地圖選擇圖層，您也可以選擇下拉清單底部的 選擇分析圖層。此選項將開啟圖庫，可讓您瀏覽並選擇各種來源中的資料集，以便在您的分析中使用該資料集。

選擇融合方法

在合併區域圖徵時，可以選擇以下兩種方法之一：

  • 重疊或相鄰的區域 - 將邊界相互重叠的兩個或多個區域合併爲一個區域。此外，相鄰的區域（即共用一個共同的邊界）將合併爲一個區域。
  • 具有相同欄位值的區域 - 如果重疊或共用一個共同邊界的區域具有相同的欄位值，則將其合併爲一個區域。

    例如，如果有一個縣圖層，並且每個縣都具有 State_Name 欄位，則可以使用 State_Name 欄位來融合邊界。如果相鄰的縣具有相同的 State_Name 值，則可對它們進行合併。最終結果是一個州邊界圖層。

如果勾選 使用目前地圖範圍，則僅融合目前地圖範圍中可見的輸入圖層中的圖徵。如果未勾選，則將對輸入圖層中的所有圖徵進行融合，包括超出目前地圖範圍的圖徵。

建立多組件圖徵

為 ArcGIS 中的地點或事物的數位表示方式，其具有多個組件，但由於參考一組屬性而被定義為一個圖徵。例如，在州的圖層中，可將夏威夷州視為多組件圖徵，因為其不同的幾何組件會歸類為單一州。此布林參數提供兩個選項：

  • True (勾選)：輸出服務可包含多組件圖徵。
  • False (取消勾選)：輸出服務僅包含單組件圖徵，並將為每個組件建立個別圖徵。

新增統計資料

合併區域後，可匯總它們的屬性。例如，如果基於 State_Name 來融合各縣的邊界，並且每個縣都具有 Population 欄位，則可以計算州的 Population 總和。

可用的統計資料類型有：

  • 總和 - 新增指定欄位的合計值
  • 平均值 - 計算指定欄位的平均值
  • 最小值 - 尋找指定欄位中所有記錄的最小值
  • 最大值 - 尋找指定欄位中所有記錄的最大值
  • 標準 差 - 尋找指定欄位中的值的標準差

結果圖層名稱

在 我的內容中建立並新增到地圖中的圖層的名稱。預設名稱參照工具名稱以及輸入圖層名稱。如果圖層名稱已存在，系統將提示您提供其他名稱。
