Εύρεση Ακραίων Τιμών

Το εργαλείο "Εύρεση Ακραίων Τιμών" προσδιορίζει αν υπάρχουν στατιστικά σημαντικές ακραίες τιμές στο χωρικό μοτίβο των δεδομένων σας.

  • Υπάρχουν μη κανονικές περιοχές στο μοτίβο των δεδομένων σας (εγκληματικές ενέργειες, δέντρα, τροχαία ατυχήματα); Πώς μπορείτε να είστε βέβαιοι;
  • Ανακαλύψατε πράγματι μια στατιστικά σημαντική ακραία τιμή (για δαπάνες, βρεφική θνησιμότητα, συνέπεια σε υψηλή βαθμολογία σε εξετάσεις) ή μήπως ο χάρτης θα αποκάλυπτε κάτι άλλο, αν αλλάζατε τον τρόπο απεικόνισής του;
Με το εργαλείο "Εύρεση Aκραίων Tιμών" θα μπορέσετε να απαντήσετε σε αυτές τις ερωτήσεις με βεβαιότητα.

Κάθε φορά που κοιτάζουμε ένα χάρτη, είναι φυσιολογικό τα μάτια και το μυαλό μας να προσπαθούν να βρουν μοτίβα ακόμη και όταν δεν υπάρχουν. Συνεπώς, είναι δύσκολο να γνωρίζετε αν τα μοτίβα στα δεδομένα σας είναι το αποτέλεσμα πραγματικών χωρικών διεργασιών ή απλώς ένα τυχαίο αποτέλεσμα. Για το λόγο αυτό, οι ερευνητές και οι αναλυτές χρησιμοποιούν στατιστικές μεθόδους όπως η Εύρεση Aκραίων Tιμών (Anselin Local Moran's I) για την ποσοτικοποίηση χωρικών μοτίβων. Όταν βρίσκετε στατιστικά σημαντικές ακραίες τιμές ή συστάδες στα δεδομένα σας, έχετε πολύτιμες πληροφορίες. Γνωρίζοντας πού και πότε παρουσιάζονται ακραίες τιμές, μπορείτε να αντλήσετε σημαντικά στοιχεία σχετικά με τις διεργασίες που προωθούν τα μοτίβα που βλέπετε. Το επόμενο βήμα θα ήταν να διερευνήσετε γιατί υπάρχουν σημαντικές διαφορές σε αυτές τις περιοχές ακραίων τιμών. Η γνώση ότι οι διαρρήξεις σε σπίτια, για παράδειγμα, είναι σημαντικά περισσότερες σε μια συγκεκριμένη συνοικία ενώ στις γύρω συνοικίες οι διαρρήξεις είναι λιγότερες, αποτελεί πολύ σημαντική πληροφορία όταν θέλετε να σχεδιάσετε αποτελεσματικές στρατηγικές αποτροπής, να κατανείμετε τους περιορισμένους πόρους αστυνόμευσης, να ξεκινήσετε προγράμματα παρακολούθησης συνοικιών, να εγκρίνετε λεπτομερείς έρευνες για εγκλήματα ή να προσδιορίσετε πιθανούς υπόπτους.

Επιλέξτε θεματικό επίπεδο για το οποίο θα υπολογιστούν ακραίες τιμές

Το θεματικό επίπεδο σημείου ή περιοχής για το οποίο θα βρεθούν οι ακραίες τιμές.

Εύρεση Ακραίων Τιμών για

Η ανάλυση αυτή απαντά στην ερώτηση: Πού βρίσκονται οι χωρικές ακραίες τιμές στα δεδομένα μου;

Αν τα δεδομένα σας είναι σημεία και επιλέξετε το Μετρήσεις σημείων, το εργαλείο αυτό θα υπολογίσει τη χωρική κατανομή των σημειακών στοιχείων για να απαντήσει στην ερώτηση: Πού συγκεντρώνονται ή διασκορπίζονται τα σημεία με μη αναμενόμενο τρόπο;

Αν επιλέξετε ένα πεδίο, το εργαλείο αυτό θα υπολογίσει τη χωρική κατανομή των τιμών που σχετίζονται με κάθε στοιχείο για να απαντήσει στις ερωτήσεις: Πού βρίσκονται χαμηλές τιμές οι οποίες περιστοιχίζονται από υψηλές τιμές; Πού βρίσκονται υψηλές τιμές οι οποίες περιστοιχίζονται από χαμηλές τιμές;

Μέτρηση σημείων εντός

Η προεπιλογή είναι να μετρούνται σημεία εντός ενός δικτυωτού πλέγματος, που δημιουργείται από το εργαλείο με βάση τα σημειακά δεδομένα σας. Εναλλακτικά, μπορείτε να επιλέξετε τη μέτρηση σημείων εντός ενός εξάγωνου πλέγματος ή την παροχή ενός θεματικού επιπέδου περιοχής (συνήθως, αυτά αντιπροσωπεύουν διοικητικές περιοχές, όπως περιοχές απογραφής, κοινοτικά όρια ή επαρχίες) για να απαντήσετε στην ερώτηση: Δεδομένου του αριθμού των σημείων που μετρήθηκαν μέσα σε κάθε στοιχείο περιοχής, υπάρχουν τοποθεσίες με στατιστικά σημαντικές υψηλών ή χαμηλών τιμών μέτρησης σημείων συγκριτικά με τις γειτονικές περιοχές;

Καθορίστε πού μπορεί να υπάρχουν σημεία

Είτε σχεδιάστε είτε δώστε ένα θεματικό επίπεδο που ορίζει το χώρο όπου περιστατικά μπορούν να απαντήσουν στην ερώτηση: Μέσα στις περιοχές, υπάρχουν τοποθεσίες με μη αναμενόμενες υψηλές ή χαμηλές συγκεντρώσεις σημείων;

Τα στοιχεία της περιοχής που σχεδιάζετε ή τα στοιχεία στο θεματικό επίπεδο περιοχής που καθορίζετε θα πρέπει να προσδιορίζουν τη θέση όπου θα μπορούσαν να εμφανιστούν σημεία. Για να σχεδιάσετε αυτές τις περιοχές, κάντε κλικ στο κουμπί Σχεδίαση και κάντε κλικ σε μια τοποθεσία στο χάρτη για να δημιουργήσετε ένα σχήμα περιοχής. Για να σχεδιάσετε πρόσθετες περιοχές, κάντε κλικ στο κουμπί σχεδίασης ξανά και μετά κάντε κλικ σε μια τοποθεσία στο χάρτη για να συνεχίστε.

Διαίρεση με

Ορισμένες φορές μπορεί να θέλετε να αναλύσετε μοτίβα που λαμβάνουν υπόψη υποκείμενες κατανομές. Για παράδειγμα, αν τα σημεία σας αντιπροσωπεύουν εγκλήματα, η διαίρεση με το συνολικό πληθυσμό θα είχε ως αποτέλεσμα μια ανάλυση των εγκλημάτων κατά κεφαλή και όχι απλώς το πλήθος των εγκλημάτων. Η επιλογή ενός περιγραφικού γνωρίσματος για διαίρεση με αυτό αναφέρεται συχνά ως "κανονικοποίηση".

Η επιλογή Πληθυσμός Esri θα εμπλουτίσει κάθε στοιχείο περιοχής με τιμές πληθυσμού, οι οποίες στη συνέχεια θα χρησιμοποιηθούν ως το περιγραφικό γνώρισμα για διαίρεση. Αυτή η επιλογή θα χρησιμοποιήσει μονάδες.

Βελτιστοποίηση για

Μπορείτε να επιλέξετε βελτιστοποίηση για ταχύτητα ή ακρίβεια.

Το εργαλείο χρησιμοποιεί παραλλαγές για να προσδιορίσει πόσο διαφορετικό από το τυχαίο είναι το χωρικό μοτίβο των δεδομένων σας. Αυξάνοντας τον αριθμό των παραλλαγών αυξάνεται η ακρίβεια αλλά και ο χρόνος επεξεργασίας.

Παράκαμψη επιλογών

Το εργαλείο βρίσκει τις βέλτιστες ρυθμίσεις για τις προεπιλογές Μέγεθος κελιού και Ζώνη απόστασης με βάση τα χαρακτηριστικά των δεδομένων σας. Ωστόσο, αν έχετε συγκεκριμένες τιμές για Μέγεθος κελιού ή Ζώνη απόστασης που έχουν νόημα για την ανάλυσή σας, μπορείτε να χρησιμοποιήσετε την Παράκαμψη επιλογών για να ορίσετε αυτές τις τιμές.

Η Παράκαμψη επιλογών είναι επίσης χρήσιμη όταν πραγματοποιείτε ανάλυση σε διαφορετικά σύνολα δεδομένων, επειδή σας επιτρέπει να διατηρείτε τις τιμές για Ζώνη απόστασης και Μέγεθος κελιού ίδιες σε πολλαπλά σύνολα δεδομένων. Στη συνέχεια, μπορείτε να συγκρίνετε τα αποτελέσματα (για παράδειγμα, ποσοστά παχυσαρκίας και διαβήτη ή ακόμη και ποσοστά εγκληματικότητας για δύο διαφορετικά έτη).

Μέγεθος κελιού

Το μέγεθος των κελιών του πλέγματος εντός του οποίου γίνεται η μέτρηση σημείων.

Όταν χρησιμοποιείτε ένα εξάγωνο πλέγμα για μέτρηση σημείων, αυτή η απόσταση χρησιμοποιείται ως το ύψος των εξαγώνων.

Ζώνη απόστασης

Κάθε στοιχείο αναλύεται στο πλαίσιο των γειτονικών στοιχείων που βρίσκονται εντός της απόστασης που καθορίζετε. Το εργαλείο θα υπολογίσει μια προεπιλεγμένη απόσταση ή μπορείτε να χρησιμοποιήσετε αυτή την επιλογή για να ορίσετε μια συγκεκριμένη απόσταση που έχει νόημα για την ανάλυσή σας

Για παράδειγμα, αν μελετάτε μοτίβα μετακίνησης και γνωρίζετε ότι η μέση διάρκεια μετακίνησης προς τη δουλειά είναι για παράδειγμα 15 μίλια, ίσως θέλετε να χρησιμοποιήσετε μια ζώνη απόστασης 15 μιλίων.

Όνομα εξαγόμενου θεματικού επιπέδου

Δώστε ένα όνομα για το θεματικό επίπεδο που θα δημιουργηθεί στην ενότητα Το περιεχόμενό μου και θα προστεθεί στο χάρτη. Αυτό το επίπεδο αποτελεσμάτων θα εμφανίσει στατιστικά σημαντικές ακραίες τιμές υψηλών και χαμηλών τιμών ή πλήθους σημείων. Αν το όνομα του εξαγόμενου θεματικού επιπέδου υπάρχει ήδη, θα σας ζητηθεί να το μετονομάσετε.

Με τη χρήση του πτυσσόμενου πλαισίου Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε το όνομα ενός φακέλου στην ενότητα Το περιεχόμενό μου όπου θα αποθηκευτεί το αποτέλεσμα.
