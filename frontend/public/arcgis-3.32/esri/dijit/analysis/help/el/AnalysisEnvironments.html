Ρυθμίσεις περιβάλλοντος ανάλυσης portal

Ρυθμίσεις περιβάλλοντος για ανάλυση στο Portal for ArcGIS και στο ArcGIS Online.

Αντικατάσταση θεματικού επιπέδου αποτελεσμάτων

Καθορίζει αν θα αντικατασταθούν τα υπάρχοντα θεματικά επίπεδα. Όταν είναι επιλεγμένη και υπάρχει ήδη στα περιεχόμενά σας θεματικό επίπεδο με το ίδιο όνομα, αυτό θα αντικατασταθεί.

  • Επιλεγμένη δυνατότητα—Αν ήδη υπάρχει στο περιεχόμενό σας ένα θεματικό επίπεδο με το ίδιο όνομα, θα αντικατασταθεί. Η έξοδος θα έχει το ίδιο σύστημα συντεταγμένων με την είσοδο.
  • Μη επιλεγμένη δυνατότητα—Αν το όνομα που καταχωρίσατε για το εξαγόμενο θεματικό επίπεδο ήδη υπάρχει, θα παρουσιαστεί σφάλμα και θα πρέπει να μετονομάσετε το εξαγόμενο θεματικό επίπεδο πριν την εκτέλεση του εργαλείου. Αυτή είναι η προεπιλεγμένη ρύθμιση.

Κλείσιμο widget ανάλυσης μετά την εκτέλεση της ανάλυσης

Καθορίζει αν το παράθυρο διαλόγου της ανάλυσης θα παραμείνει ανοιχτό και ενεργοποιημένο όση ώρα εκτελείται το εργαλείο. Αυτή η ρύθμιση σας επιτρέψει να κάνετε τροποποιήσεις στις παραμέτρους της ανάλυσής σας, την ώρα που εξακολουθεί να εκτελείται η προηγούμενη εργασία ανάλυσης. Η ανάλυση είναι μια επαναληπτική διαδικασία και διατηρώντας το παράθυρο διαλόγου του εργαλείου πλαίσιο ανοιχτό έχετε τη δυνατότητα να κάνετε αλλαγές στις παραμέτρους και να εκτελέσετε ξανά την ανάλυσή σας.

  • Επιλεγμένη δυνατότητα—Το παράθυρο διαλόγου του εργαλείου ανάλυσης θα κλείσει και η εστίαση θα επιστρέψει στον πίνακα περιεχομένων. Αυτή είναι η προεπιλεγμένη ρύθμιση.
  • Μη επιλεγμένη δυνατότητα—Το παράθυρο διαλόγου του εργαλείου ανάλυσης θα παραμείνει ανοιχτό και ενεργοποιημένο.

Αποθήκευση αποτελεσμάτων ανάλυσης ως hosted feature layer

Καθορίζει αν το εξαγόμενο θεματικό επίπεδο που δημιουργήθηκε από την ανάλυσή σας θα είναι μια συλλογή στοιχείων ή ένα hosted feature layer.

  • Επιλεγμένη δυνατότητα—Τα αποτελέσματα της ανάλυσής σας θα δημιουργήσουν ένα hosted feature layer και θα δημιουργηθεί ένα αντικείμενο στα περιεχόμενά σας. Αυτή είναι η προεπιλεγμένη ρύθμιση.
  • Μη επιλεγμένη δυνατότητα—Τα αποτελέσματα της ανάλυσής σας θα δημιουργήσουν μια συλλογή στοιχείων.

Σύστημα συντεταγμένων εξόδου

Καθορίζει το σύστημα συντεταγμένων του εξαγόμενου θεματικού επιπέδου για ανάλυση raster.

  • Ίδιο με την είσοδο—Το αποτέλεσμα της ανάλυσής σας θα είναι στο ίδιο σύστημα συντεταγμένων με την είσοδο. Αυτή είναι η προεπιλεγμένη ρύθμιση.
  • Όπως έχει καθοριστεί—Το αποτέλεσμα της ανάλυσής σας θα είναι στο σύστημα συντεταγμένων που επιλέγετε. Όταν επιλέξετε τη δυνατότητα αυτή, κάντε κλικ στο κουμπί με το εικονίδιο της υφηλίου και επιλέξτε ένα σύστημα από μια λίστα γνωστών συστημάτων συντεταγμένων ή προσθέστε τη χωρική αναφορά WKID στο παρεχόμενο πλαίσιο.
  • Θεματικό επίπεδο <name>—Το αποτέλεσμα της ανάλυσής σας θα είναι στο ίδιο σύστημα συντεταγμένων με ένα υπάρχον θεματικό επίπεδο που επιλέγετε τον web χάρτη σας.

Επεξεργασία συστήματος συντεταγμένων

Καθορίζει το σύστημα συντεταγμένων που θα χρησιμοποιηθεί στην ανάλυσή σας κατά την εκτέλεση της ανάλυσης GeoAnalytics.

Για το GeoAnalytics, ορισμένες αναλύσεις απαιτούν την εκτέλεση της επεξεργασίας σε προβολικό σύστημα συντεταγμένων. Οποιοδήποτε εργαλείο έχει εφαρμόσει αποθήκευση δεδομένων ή γραμμικές μονάδες απαιτεί τη χρήση ενός προβολικού συστήματος συντεταγμένων.

  • Ίδιο με την είσοδο—Η επεξεργασία της ανάλυσής σας θα γίνει στο ίδιο σύστημα συντεταγμένων με την είσοδο. Αυτή είναι η προεπιλεγμένη ρύθμιση.
  • Όπως έχει καθοριστεί—Η ανάλυσή σας θα είναι στο σύστημα συντεταγμένων που επιλέγετε. Όταν επιλέξετε τη δυνατότητα αυτή, κάντε κλικ στο κουμπί με το εικονίδιο της υφηλίου και επιλέξτε ένα σύστημα από μια λίστα συστημάτων προβαλλόμενων συντεταγμένων ή προσθέστε τη χωρική αναφορά WKID στο παρεχόμενο πλαίσιο.
  • Θεματικό επίπεδο <name>—Η ανάλυσή σας θα είναι στο ίδιο σύστημα συντεταγμένων με ένα υπάρχον θεματικό επίπεδο που επιλέγετε τον web χάρτη σας.

Έκταση

Καθορίζει την έκταση ή το όριο που θα χρησιμοποιηθεί κατά την εκτέλεση της ανάλυσης. Όλα τα στοιχεία εισόδου που βρίσκονται εντελώς μέσα στην καθορισμένη έκταση ή που τέμνουν την καθορισμένη έκταση θα χρησιμοποιηθούν στην ανάλυση.

  • Προεπιλεγμένη ρύθμιση—Η έκταση την οποία παρέχει το εργαλείο.
  • Όπως έχει καθοριστεί—Η έκταση καθορίζεται από τις συντεταγμένες που καταχωρίζετε.
  • Θεματικό επίπεδο <name>—Η έκταση που χρησιμοποιείται κατά την διεργασία της ανάλυσής σας θα είναι η ίδια με τη χωρική έκταση ενός υπάρχοντος θεματικού επιπέδου που επιλέγετε τον web χάρτη σας.

Αγκίστρωση raster

Ρυθμίζει την έκταση του θεματικού επιπέδου raster εξόδου έτσι ώστε να συμφωνεί με την ευθυγράμμιση των κελιών του καθορισμένου θεματικού επιπέδου Αγκίστρωση raster στην ανάλυση raster.

Μέγεθος κελιού

Καθορίζει το μέγεθος κελιού ή την ανάλυση που θα χρησιμοποιηθεί για τη δημιουργία του θεματικού επιπέδου raster εξόδου στην ανάλυση raster. Η προκαθορισμένη ανάλυση εξόδου προσδιορίζεται από το μεγαλύτερο μέγεθος κελιού του θεματικού επιπέδου raster εισόδου.

  • Ελάχιστο όριο δεδομένων εισόδου—Χρησιμοποιήστε το μικρότερο μέγεθος κελιού όλων των θεματικών επιπέδων εισόδου.
  • Μέγιστο όριο δεδομένων εισόδου—Χρησιμοποιήστε το μεγαλύτερο μέγεθος κελιού όλων των θεματικών επιπέδων εισόδου. Αυτή είναι η προεπιλεγμένη ρύθμιση.
  • Όπως έχει καθοριστεί—Καθορίστε μια αριθμική τιμή για να ρυθμίσετε το μέγεθος κελιού. Αν είναι επιλεγμένο, η προκαθορισμένη τιμή είναι 1.
  • Θεματικό επίπεδο <όνομα>—Καθορίστε το μέγεθος κελιού στο πλεγματικό θεματικό επίπεδο που έχετε επιλέξει.

Μάσκα

Καθορίζει ένα θεματικό επίπεδο που θα χρησιμοποιηθεί για τον καθορισμό της περιοχής ενδιαφέροντός σας στην ανάλυση raster. Μόνο τα κελιά που βρίσκονται εντός της μάσκας ανάλυσης θα ληφθούν υπόψη κατά τη λειτουργία της ανάλυσης.

  • Η μάσκα μπορεί να είναι raster ή θεματικό επίπεδο στοιχείων.
  • Αν η μάσκα ανάλυσης είναι raster, όλα τα κελιά με κάποια τιμή θα ληφθούν υπόψη για τον ορισμό της μάσκας. Τα κελιά ενός raster μάσκας με τιμή NoData θα θεωρηθούν ότι βρίσκονται εκτός της μάσκας και θα έχουν την τιμή NoData στο θεματικό επίπεδο αποτελεσμάτων της ανάλυσης.
  • Αν η μάσκα ανάλυσης είναι θεματικό επίπεδο στοιχείων, θα μετατραπεί εσωτερικά σε raster κατά την εκτέλεση. Για τον λόγο αυτό, βεβαιωθείτε ότι έχετε δώσει στις επιλογές Μέγεθος κελιού και Αγκίστρωση Raster τις τιμές που ενδείκνυνται για την ανάλυσή σας.

Datastore

Τα αποτελέσματα του GeoAnalytics μπορούν να αποθηκεύονται είτε σε ένα ArcGIS Data Store και να εκτίθενται ως feature layer στο Portal for ArcGIS είτε σε ένα παραμετροποιημένο κοινόχρηστο αρχείο big data.

Όποτε αποθηκεύετε ένα αποτέλεσμα σε ArcGIS Data Store, στις περισσότερες περιπτώσεις τα αποτελέσματα λογικά θα αποθηκεύονται στο datastore για χωροχρονικά δεδομένα. Αυτή είναι η προεπιλεγμένη ρύθμιση. Ακολουθούν κάποιοι λόγοι για τους οποίους ίσως θελήσετε να αποθηκεύσετε αποτελέσματα σε σχεσιακό data store:

  • Για να χρησιμοποιήσετε τα αποτελέσματα σε συνεργασία μεταξύ portal
  • Για ενεργοποίηση δυνατοτήτων συγχρονισμού με τα αποτελέσματά σας

Δεν πρέπει να χρησιμοποιήσετε σχεσιακό data store αν περιμένετε αύξηση αποτελεσμάτων GeoAnalytics και επιθυμείτε να εκμεταλλευτείτε τις δυνατότητες του χωροχρονικού big data store για διαχείριση μεγάλων όγκων δεδομένων.

Παράγοντας παράλληλης επεξεργασίας

Κατανέμει την ανάλυση σε πολλά instance της υπηρεσίας επεξεργασίας raster.

Τα εργαλεία που τηρούν τις απαιτήσεις του περιβάλλοντος της επιλογής Παράγοντας παράλληλης επεξεργασίας σας επιτρέπουν να ελέγχετε τον αριθμό των instance της υπηρεσίας επεξεργασίας raster που μπορούν να χρησιμοποιηθούν για την επεξεργασία των δεδομένων σας.

Αν το εργαλείο δεν τηρεί τις απαιτήσεις του περιβάλλοντος της επιλογής Τύπος επεξεργαστή ή αν το περιβάλλον της επιλογής Τύπος επεξεργαστή έχει τη ρύθμιση CPU, το περιβάλλον της επιλογής Παράγοντας παράλληλης επεξεργασίας ελέγχει τα instance της υπηρεσίας επεξεργασίας raster (μέσω της CPU). Αν η επιλογή Τύπος επεξεργαστή έχει τη ρύθμιση GPU, το περιβάλλον της επιλογής Παράγοντας παράλληλης επεξεργασίας ελέγχει τον αριθμό των instance της υπηρεσίας επεξεργασίας raster μέσω της GPU.

Καθορίζοντας τη ρύθμιση για την επιλογή Παράγοντας παράλληλης επεξεργασίας, μπορείτε να ζητήσετε τον αριθμό των παράλληλων εργασιών τις οποίες χρησιμοποιεί το image server ανάλυσης του raster για την επεξεργασία μίας εργασίας ανάλυσης raster. Ωστόσο, αν ο αριθμός των παράλληλων διεργασιών ξεπερνά τον ανώτατο αριθμό των instance της υπηρεσίας επεξεργασίας raster (μέσω της CPU ή της GPU), οι πρόσθετες παράλληλες διεργασίες θα ενταχθούν στην ουρά.

Με βάση τις προεπιλεγμένες ρυθμίσεις δεν είναι καθορισμένη η ρύθμιση για την επιλογή Παράγοντας παράλληλης επεξεργασίας και στην περίπτωση αυτή το εργαλείο θα χρησιμοποιήσει το 80% του ανώτατου αριθμού των instance της υπηρεσίας επεξεργασίας raster. Ως παράγοντας παράλληλης επεξεργασίας μπορεί να καθοριστεί είτε ένας ακέραιος αριθμός είτε ένα ποσοστό.

Τύπος επεξεργαστή

Τα εργαλεία που τηρούν τις απαιτήσεις του περιβάλλοντος της επιλογής Τύπος επεξεργαστή σας επιτρέπουν να επιλέγετε πού και πώς θέλετε να γίνει η επεξεργασία των δεδομένων σας. Μπορείτε να επεξεργαστείτε τα δεδομένα σας χρησιμοποιώντας τη CPU ή την GPU του υπολογιστή του διακομιστή. Αν το περιβάλλον της επιλογής Τύπος επεξεργαστή είναι κενό, το εργαλείο χρησιμοποιεί τη CPU για την επεξεργασία των δεδομένων. Αυτή είναι η προεπιλεγμένη ρύθμιση.

Η διαδικασία επεξεργασίας μέσω της CPU μπορεί να εκτελεστεί παράλληλα σε πολλαπλούς πυρήνες και πολλαπλά instance επειδή η διαχείριση γίνεται από την επιλογή Παράγοντας παράλληλης επεξεργασίας.

Οι GPU είναι αποδοτικές στη διαδικασία επεξεργασίας γραφικών και εικόνων, όπου η άκρως παράλληλη δομή τους τις καθιστά αποτελεσματικές κατά την επανειλημμένη επεξεργασία μεγάλων μπλοκ. Τα εργαλεία ανάλυσης raster που τηρούν τις απαιτήσεις του περιβάλλοντος αυτού μπορούν να κατανείμουν την εργασία του σε διάφορα instance της GPU σε πολλαπλά μηχανήματα του διακομιστή ανάλυσης raster επειδή η διαχείριση γίνεται από την επιλογή Παράγοντας παράλληλης επεξεργασίας.

  • GPU—Για την επεξεργασία των δεδομένων θα χρησιμοποιηθεί η μονάδα επεξεργασίας γραφικών.
  • CPU—Για την επεξεργασία των δεδομένων θα χρησιμοποιηθεί η κεντρική μονάδα επεξεργασίας του υπολογιστή.
