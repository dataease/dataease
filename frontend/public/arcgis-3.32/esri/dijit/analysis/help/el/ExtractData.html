Εξαγωγή Δεδομένων

Με αυτό το εργαλείο μπορείτε να επιλέξετε και να κάνετε λήψη δεδομένων για μια καθορισμένη περιοχή ενδιαφέροντος. Τα επίπεδα που επιλέγετε προστίθενται σε ένα αρχείο ZIP, CSV ή KMZ.

Θεματικά επίπεδα προς εξαγωγή

Είναι τα θεματικά επίπεδα στον πίνακα περιεχομένων από τα οποία μπορείτε να εξαγάγετε στοιχεία Τα θεματικά επίπεδα που επιλέγονται, εξάγονται και προστίθενται στο αρχείο ZIP ή CSV που προκύπτει.

Για να εμφανιστεί ένα θεματικό επίπεδο στη λίστα των θεματικών επιπέδων προς εξαγωγή, πρέπει είτε να είστε ο κάτοχος των δεδομένων είτε ο κάτοχος των δεδομένων να ενεργοποιήσει την εξαγωγή των δεδομένων. Για να ενεργοποιήσετε την εξαγωγή δεδομένων σε δεδομένα των οποίων είστε κάτοχοι:

  • Ανοίξτε την περιγραφή αντικειμένου των δεδομένων.
  • Κάντε κλικ στην καρτέλα Ρυθμίσεις.
  • Μεταβείτε προς τα κάτω με κύλιση στην ενότητα Εξαγωγή δεδομένων.
  • Επιλέξτε Να επιτρέπεται σε άλλους η εξαγωγή σε διαφορετικά μορφότυπα. Κάντε κλικ στην επιλογή Αποθήκευση όταν ολοκληρώσετε.

Αν το θεματικό επίπεδο ανάλυσης περιέχει συνημμένα, θα πρέπει να έχετε υπόψη σας ότι θα γίνει εξαγωγή των συνημμένων μόνο αν το μορφότυπο των εξαγόμενων δεδομένων είναι file geodatabase και δεν είναι επιλεγμένη η δυνατότητα Αποκοπή στοιχείων.

Περιοχή μελέτης

Η περιοχή που προσδιορίζει τα στοιχεία που θα συμπεριληφθούν στο αρχείο εξόδου ZIP ή CSV. Μπορείτε να επιλέξετε από τα εξής:

  • Την τρέχουσα ορατή έκταση του χάρτη ( Ίδια με την εμφανιζόμενη).
  • Ένα από τα θεματικά επίπεδα του πίνακα περιεχομένων σας. Αν το επιλεγμένο θεματικό επίπεδο είναι σημειακά ή γραμμικά στοιχεία, ως περιοχή μελέτης θα χρησιμοποιηθεί η έκταση του θεματικού επιπέδου.
  • Σχεδιάστε μια περιοχή για να καθορίσετε την περιοχή μελέτης.
Θα εξαχθούν όλα τα στοιχεία που βρίσκονται εντελώς μέσα στην έκταση του χάρτη. Για στοιχεία που διασχίζουν ή τέμνουν το όριο της έκτασης του χάρτη, αλλά δεν είναι εντελώς εντός της έκτασης, έχετε δύο επιλογές:
  • Επιλογή Στοιχείων—Θα εξαχθούν ολόκληρα τα στοιχεία που διασταυρώνονται με το όριο της περιοχής μελέτης.
  • Αποκοπή Στοιχείων—Θα εξαχθούν μόνο τα τμήματα των στοιχείων εκείνων που βρίσκονται εντός της περιοχής μελέτης.

Μορφότυπο εξαγόμενων δεδομένων

Το μορφότυπο των δεδομένων που θα εξαχθούν. Όταν γίνεται εξαγωγή γραμμικών ή πολυγωνικών στοιχείων σε αρχεία CSV, εξάγονται μόνο τα μη χωρικά γνωρίσματα. Όταν γίνεται εξαγωγή σημειακών στοιχείων, εξάγονται τόσο τα χωρικά όσο και τα μη χωρικά γνωρίσματα.

Τα εξαγόμενα δεδομένα θα χρησιμοποιούν την προβολή Web Mercator 1984. Για αρχεία CSV τα πεδία Lat και Long θα προσαρτώνται στα δεδομένα με συντεταγμένες στο Web Mercator 1984.

Υποστηρίζονται τα εξής μορφότυπα εξόδου:

  • Αρχείο τιμών διαχωρισμένων με κόμματα (CSV)— .csv ή .zip
  • File geodatabase— .zip
  • Keyhole Markup Language (KML)— .kmz ή .zip
  • Shapefile— .zip

Όνομα αρχείου εξόδου

Αυτό είναι το όνομα του αντικειμένου που θα προστεθεί στην ενότητα Το Περιεχόμενό Μου. Το όνομα αντικειμένου βασίζεται στην επιλεγμένη ρύθμιση Μορφότυπο εξαγόμενων δεδομένων. Αν το αντικείμενο υπάρχει ήδη, θα σας ζητηθεί να δώσετε άλλο όνομα.

Χρησιμοποιώντας το πτυσσόμενο πλαίσιο Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε το όνομα ενός φακέλου στην ενότητα Το περιεχόμενό μου όπου θα αποθηκευτεί το αποτέλεσμα.
