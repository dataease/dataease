Δημιουργία Ζωνών

Η ζώνη είναι μια περιοχή που καλύπτει μια δεδομένη απόσταση από ένα σημειακό, γραμμικό ή πολυγωνικό στοιχείο.

Οι ζώνες συνήθως χρησιμοποιούνται για τη δημιουργία περιοχών, οι οποίες μπορούν να αναλυθούν περισσότερο με χρήση άλλων εργαλείων. Για παράδειγμα, αν η ερώτηση είναι "Ποια κτίρια υπάρχουν σε απόσταση 1 μιλίου από το σχολείο;", η απάντηση μπορεί να βρεθεί, αν δημιουργηθεί μια ζώνη 1 μιλίου γύρω από το σχολείο και επικαλυφθεί η ζώνη με το θεματικό επίπεδο που περιέχει τα ίχνη των κτιρίων. Το τελικό αποτέλεσμα είναι ένα θεματικό επίπεδο με τα κτίρια που βρίσκονται σε απόσταση 1 μιλίου από το σχολείο.

Διαλέξτε θεματικό επίπεδο για εφαρμογή ζώνης

Τα σημειακά, γραμμικά ή πολυγωνικά στοιχεία για τα οποία θα δημιουργηθεί ζώνη. Τα θεματικά επίπεδα εισόδου πρέπει να βρίσκονται σε προβολικό σύστημα συντεταγμένων ή η χωρική αναφορά της επεξεργασίας πρέπει να είναι ορισμένη σε προβολικό σύστημα συντεταγμένων χρησιμοποιώντας τα Περιβάλλοντα Ανάλυσης.

Εκτός από την επιλογή ενός θεματικού επιπέδου από το χάρτη σας, μπορείτε επίσης να χρησιμοποιήσετε την Επιλογή θεματικού επιπέδου ανάλυσης στο κάτω μέρος της αναπτυσσόμενης λίστας για περιήγηση στο περιεχόμενό σας και αναζήτηση ενός κοινόχρηστου συνόλου δεδομένων αρχείων big data ή ενός feature layer.

Επιλέξτε τον τύπο της ζώνης προς εφαρμογή

Υπάρχουν τρεις τρόποι με τους οποίους μπορείτε να προσδιορίσετε το μέγεθος της ζώνης για τα στοιχεία εισόδου:

  • Προσδιορίστε μια απόσταση που θα εφαρμοστεί σε όλα τα στοιχεία.
  • Προσδιορίστε ένα πεδίο στο θεματικό επίπεδο εισόδου το οποίο περιέχει την τιμή απόστασης. Μπορείτε να χρησιμοποιήσετε ένα πεδίο συμβολοσειράς ή ένα αριθμητικό πεδίο. Αν δεν οριστεί γραμμική μονάδα, θα χρησιμοποιηθούν οι μονάδες της χωρικής αναφοράς. Αν χρησιμοποιείτε σύστημα γεωγραφικών συντεταγμένων, το πεδίο χωρίς μονάδες θα θεωρηθεί ότι είναι μέτρα.
  • Δημιουργήστε μια έκφραση με πολλαπλά πεδία και μαθηματικούς τελεστές. Για παράδειγμα, αν θέλετε να δημιουργήσετε ζώνη 10 φορές μεγαλύτερη από την τιμή ενός πεδίου με το όνομα wind_speed, προσθέστε την έκφραση $feature["wind_speed"] x 10.

Επιλέξτε τη μέθοδο ζώνης

Μπορείτε να επιλέξετε να χρησιμοποιήσετε μια Επιπεδική ή Γεωδαιτική μέθοδο. Η Επιπεδική μέθοδος μπορεί να είναι ταχύτερη και είναι κατάλληλη για τοπική ανάλυση σε προβολικά δεδομένα. Η Γεωδαιτική μέθοδος είναι κατάλληλη για μεγάλες περιοχές και οποιοδήποτε σύστημα γεωγραφικών συντεταγμένων.

Επιλέξτε τον τύπο της μεθόδου ομαδοποίησης

Επιλογές για τον προσδιορισμό της μεθόδου ομαδοποίησης. Αν επιλεχθεί μια μέθοδος ομαδοποίησης, θα έχετε τη δυνατότητα να δημιουργήσετε περιοχές πολλαπλών ή ενιαίων τμημάτων και θα μπορείτε να υπολογίσετε στατιστικά με βάση τα παρεχόμενα πεδία.

  • Όλα—Θα ομαδοποιηθούν όλα τα στοιχεία. Αν επιτρέπονται στοιχεία πολλαπλών τμημάτων, όλα τα στοιχεία θα ομαδοποιηθούν σε ένα και μόνο στοιχείο. Αν δεν επιτρέπονται στοιχεία πολλαπλών τμημάτων, θα ομαδοποιηθούν τα γειτονικά ή τα επικαλυπτόμενα στοιχεία.
  • Πεδία—Θα ομαδοποιηθούν τα στοιχεία που μοιράζονται το ίδιο καθορισμένο πεδίο ή τον καθορισμένο συνδυασμό πεδίων. Αν επιτρέπονται στοιχεία πολλαπλών τμημάτων, όλα τα στοιχεία με το ίδιο πεδίο θα ομαδοποιηθούν σε ένα και μόνο στοιχείο. Αν δεν επιτρέπονται πολλαπλών τμημάτων, θα ομαδοποιηθούν τα γειτονικά ή τα επικαλυπτόμενα στοιχεία με την ίδια τιμή πεδίου.
  • Κανένα—Κανένα στοιχείο δεν θα ομαδοποιηθεί. Αυτή είναι η προεπιλογή.

Να επιτρέπονται στοιχεία πολλαπλών τμημάτων.

Επιλογή που προσδιορίζει αν τα αποτελέσματά σας θα περιλαμβάνουν στοιχεία πολλαπλών ή ενιαίων τμημάτων.

  • Επιλεγμένη—Τα αποτελέσματα της ανάλυσής σας θα περιέχουν στοιχεία πολλαπλών τμημάτων.
  • Μη επιλεγμένη—Τα αποτελέσματα της ανάλυσής σας θα περιέχουν στοιχεία ενιαίων τμημάτων. Αυτή είναι η προεπιλεγμένη ρύθμιση.

Προσθήκη στατιστικών (προαιρετικά)

Μπορείτε να υπολογίσετε στατιστικά για τα στοιχεία που συνοψίζονται.Μπορείτε να υπολογίσετε τα στοιχεία που ακολουθούν στα αριθμικά πεδία:

  • Αριθμός—Υπολογίζει τον αριθμό τιμών που δεν είναι μηδενικές (null). Μπορεί να χρησιμοποιηθεί σε αριθμικά πεδία ή σε συμβολοσειρές. Το πλήθος των [null, 0, 2] είναι 2.
  • Άθροισμα—Το άθροισμα των αριθμητικών τιμών σε ένα πεδίο. Το άθροισμα των [null, null, 3] είναι 3.
  • Μέση τιμή—Η μέση τιμή αριθμητικών τιμών. Η μέση τιμή των [0, 2, null] είναι 1.
  • Ελάχιστο—Η ελάχιστη τιμή ενός αριθμητικού πεδίου. Το ελάχιστο των [0, 2, null] είναι 0.
  • Μέγιστο—Η μέγιστη τιμή ενός αριθμητικού πεδίου. Η μέγιστη τιμή των [0, 2, null] είναι 2.
  • Εύρος—Το εύρος ενός αριθμητικού πεδίου. Αυτό υπολογίζεται ως οι ελάχιστες τιμές που αφαιρούνται από την μέγιστη τιμή. Το εύρος των [0, null, 1] είναι 1. Το εύρος των [null, 4] είναι 0.
  • Διακύμανση—Η διακύμανση ενός αριθμητικού πεδίου σε ένα ίχνος. Η διακύμανση του [1] είναι null. Η διακύμανση των τιμών [null, 1,0,1,1] είναι 0,25.
  • Τυπική απόκλιση—Η τυπική απόκλιση ενός αριθμητικού πεδίου. Η τυπική απόκλιση του [1] είναι null. Η τυπική απόκλιση των τιμών [null, 1,0,1,1] είναι 0,5.

Μπορείτε να υπολογίσετε τα στοιχεία που ακολουθούν στα πεδία συμβολοσειρών:

  • Πλήθος—Ο αριθμός των συμβολοσειρών που δεν είναι null.
  • Οποιαδήποτε—Αυτό το στατιστικό στοιχείο είναι ένα τυχαίο δείγμα μιας τιμής συμβολοσειράς στο καθορισμένο πεδίο.
Όλα τα στατιστικά στοιχεία υπολογίζονται σε τιμές που δεν είναι null. Το εξαγόμενο θεματικό επίπεδο θα περιέχει ένα νέο πεδίο για κάθε υπολογιζόμενο στατιστικό στοιχείο. Μπορεί να προστεθεί οποιοσδήποτε αριθμός στατιστικών στοιχείων, επιλέγοντας γνώρισμα και στατιστικό στοιχείο.

Επιλέξτε ένα ArcGIS Data Store για αποθήκευση των αποτελεσμάτων

Τα αποτελέσματα GeoAnalytics αποθηκεύονται σε ένα datastore και εκτίθενται ως feature layer στο Portal for ArcGIS. Στις περισσότερες περιπτώσεις, τα αποτελέσματα λογικά θα αποθηκεύονται στο datastore για χωροχρονικά δεδομένα και αυτή είναι η προεπιλεγμένη ρύθμιση. Σε ορισμένες περιπτώσεις, μια καλή επιλογή είναι η αποθήκευση αποτελεσμάτων στο σχεσιακό data store. Ακολουθούν ορισμένοι λόγοι για τους οποίους θα θέλατε να αποθηκεύσετε αποτελέσματα στο σχεσιακό data store:

  • Μπορείτε να χρησιμοποιήσετε αποτελέσματα σε συνεργασία portal με portal.
  • Μπορείτε να ενεργοποιήσετε τις δυνατότητες συγχρονισμού με τα αποτελέσματά σας.

Δεν πρέπει να χρησιμοποιείτε το σχεσιακό data store αν περιμένετε αύξηση των αποτελεσμάτων GeoAnalytics και επιθυμείτε να εκμεταλλευτείτε τις δυνατότητες του χωροχρονικού data store μαζικών δεδομένων για τη διαχείριση μεγάλων όγκων δεδομένων.

Όνομα εξαγόμενου θεματικού επιπέδου

Το όνομα του θεματικού επιπέδου που θα δημιουργηθεί. Αν πραγματοποιείτε εγγραφή σε ένα ArcGIS Data Store, τα αποτελέσματά σας θα αποθηκευτούν στον φάκελο Το περιεχόμενό μου και θα προστεθούν στον χάρτη. Αν πραγματοποιείτε εγγραφή σε ένα κοινόχρηστο αρχείο big data, τα αποτελέσματά σας θα αποθηκευτούν στο κοινόχρηστο αρχείο big data και θα προστεθούν στη διακήρυξή του. Δεν θα προστεθούν στον χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο υπάρχει ήδη, το εργαλείο θα αποτύχει.

Όποτε πραγματοποιείτε εγγραφή σε ArcGIS Data Store (datastore για σχεσιακά ή χωροχρονικά big data) χρησιμοποιώντας το πτυσσόμενο πλαίσιο Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε μέσα στον φάκελο Το περιεχόμενό μου το όνομα ενός άλλου φακέλου όπου θα αποθηκευτεί το αποτέλεσμα.
