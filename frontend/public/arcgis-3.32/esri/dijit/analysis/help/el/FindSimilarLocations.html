Εύρεση Παρόμοιων Τοποθεσιών

Ανάλογα με τα κριτήρια που καθορίζετε, το εργαλείο Εύρεση Παρόμοιων Τοποθεσιών μετρά την ομοιότητα των τοποθεσιών στο υποψήφιο θεματικό επίπεδο αναζήτησης έναντι μίας ή περισσότερων τοποθεσιών αναφοράς. Αυτό το εργαλείο μπορεί να δώσει απάντηση σε ερωτήσεις, όπως

  • Ποιο από τα καταστήματά σας μοιάζει περισσότερο με εκείνα που έχουν τις περισσότερες πωλήσεις, όσον αφορά τα προφίλ πελατών;
  • Με βάση τα χαρακτηριστικά των χωριών που είχαν τα περισσότερα κρούσματα ασθένειας, ποια άλλα χωριά αντιμετωπίζουν μεγάλο κίνδυνο;
Για να απαντηθούν τέτοιες ερωτήσεις, παρέχετε τις τοποθεσίες αναφοράς, τις υποψήφιες τοποθεσίες αναζήτησης και τα πεδία που αντιπροσωπεύουν τα κριτήρια που θέλετε να πληρούνται. Το πρώτο θεματικό επίπεδο εισόδου θα πρέπει να περιέχει τις τοποθεσίες ή τα σημεία αναφοράς. Για παράδειγμα, μπορεί να είναι ένα θεματικό επίπεδο που περιέχει τα καταστήματα με τις καλύτερες πωλήσεις ή τα χωριά με τα περισσότερα κρούσματα της ασθένειας. Στη συνέχεια, καθορίζετε το θεματικό επίπεδο με τις προς αναζήτηση τοποθεσίες. Αυτό μπορεί να είναι όλα τα καταστήματά σας ή όλα τα άλλα χωριά. Τέλος, προσδιορίζετε ένα ή περισσότερα πεδία που θα χρησιμοποιηθούν για τη μέτρηση της ομοιότητας. Κατόπιν, το εργαλείο Εύρεση Παρόμοιων Τοποθεσιών θα κατατάξει όλες τις υποψήφιες τοποθεσίες αναζήτησης ανάλογα με την ομοιότητά τους με τις τοποθεσίες αναφοράς σε όλα τα πεδία που έχετε επιλέξει.

Επιλέξτε το θεματικό επίπεδο που περιέχει τις τοποθεσίες αναφοράς

Το θεματικό επίπεδο σημείου, γραμμής ή περιοχής που περιέχει τις τοποθεσίες αναφοράς για συμφωνία.

Μπορείτε να επιλέξετε ένα θεματικό επίπεδο από τον χάρτη σας αλλά και να χρησιμοποιήσετε και τη δυνατότητα Επιλογή θεματικού επιπέδου ανάλυσης που βρίσκεται στο κάτω μέρος της πτυσσόμενης λίστας. Με την επιλογή αυτή ανοίγει μία συλλογή στην οποία μπορείτε να περιηγηθείτε και να επιλέξετε από διάφορες πηγές σύνολα δεδομένων που θα χρησιμοποιηθούν στην ανάλυσή σας.

Μπορείτε να χρησιμοποιήσετε όλες τις τοποθεσίες ή να επιλέξετε ορισμένες

Αν χρειάζεται, χρησιμοποιήστε τα κουμπιά επιλογής, για να προσδιορίσετε τις τοποθεσίες αναφοράς. Για παράδειγμα, αν το θεματικό επίπεδο εισόδου περιέχει όλες τις τοποθεσίες —τις τοποθεσίες αναφοράς και τις υποψήφιες τοποθεσίες αναζήτησης— θα πρέπει να χρησιμοποιήσετε ένα από τα εργαλεία επιλογής για να προσδιορίσετε τις τοποθεσίες αναφοράς. Αν δημιουργήσετε δύο ξεχωριστά θεματικά επίπεδα, ένα με τις τοποθεσίες αναφοράς και το άλλο με όλες τις υποψήφιες τοποθεσίες αναζήτησης, δεν χρειάζεται να επιλέξετε κάτι.

Αν υπάρχουν περισσότερες από μία τοποθεσίες αναφοράς, δημιουργείται μία τοποθεσία με βάση το μέσο όρο των τιμών κάθε πεδίου που χρησιμοποιείται για την ανάλυση της ομοιότητας.

Αναζήτηση παρόμοιων τοποθεσιών σε

Οι υποψήφιες τοποθεσίες αναζήτησης αυτού του θεματικού επιπέδου θα καταταχθούν από την περισσότερο όμοια προς τη λιγότερο όμοια. Η ομοιότητα βασίζεται στο πόσο ταιριάζει κάθε τοποθεσία με την τοποθεσία αναφοράς στα πεδία που καθορίζετε.

Μπορείτε να επιλέξετε ένα θεματικό επίπεδο από τον χάρτη σας αλλά και να χρησιμοποιήσετε και τη δυνατότητα Επιλογή θεματικού επιπέδου ανάλυσης που βρίσκεται στο κάτω μέρος της πτυσσόμενης λίστας. Με την επιλογή αυτή ανοίγει μία συλλογή στην οποία μπορείτε να περιηγηθείτε και να επιλέξετε από διάφορες πηγές σύνολα δεδομένων που θα χρησιμοποιηθούν στην ανάλυσή σας.

Η ομοιότητα να βασιστεί σε

Τα πεδία που επιλέγετε θα είναι τα κριτήρια για την αξιολόγηση της ομοιότητας. Αν για παράδειγμα, επιλέξετε ένα πεδίο πληθυσμού και εισοδήματος, οι υποψήφιες τοποθεσίες αναζήτησης με τη χαμηλότερη (καλύτερη) κατάταξη θα είναι εκείνες που έχουν παρόμοιες τιμές πληθυσμού και εισοδήματος με τις τοποθεσίες αναφοράς.

Εμφάνιση

Μπορείτε να δείτε όλες τις υποψήφιες τοποθεσίες αναζήτησης, καταταγμένες από την περισσότερο έως τη λιγότερο παρόμοια, ή μπορείτε να καθορίσετε τον αριθμό των αποτελεσμάτων που θέλετε να εμφανίζονται.

  • Όλες οι τοποθεσίες από την περισσότερο στη λιγότερο όμοια—Όλα τα στοιχεία του υποψήφιου θεματικού επιπέδου αναζήτησης θα συμπεριληφθούν στη σειρά κατάταξης στο εξαγόμενο θεματικό επίπεδο.
  • Ο αριθμός των επιστρεφόμενων τοποθεσιών—Καθορίζετε πόσες από τις περισσότερο όμοιες υποψήφιες τοποθεσίες θα πρέπει να περιλαμβάνονται στο εξαγόμενο θεματικό επίπεδο. Η προεπιλεγμένη τιμή εισόδου είναι 1.

Όνομα εξαγόμενου θεματικού επιπέδου

Δώστε ένα όνομα για το θεματικό επίπεδο που θα δημιουργηθεί στην ενότητα Το περιεχόμενό μου και θα προστεθεί στο χάρτη. Αυτό το εξαγόμενο θεματικό επίπεδο θα περιέχει τις τοποθεσίες αναφοράς και τον αριθμό των καταταγμένων υποψήφιων τοποθεσιών αναζήτησης που καθορίσατε. Αν το όνομα του εξαγόμενου θεματικού επιπέδου υπάρχει ήδη, θα σας ζητηθεί να το μετονομάσετε.

Χρησιμοποιώντας το πτυσσόμενο πλαίσιο Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε το όνομα ενός φακέλου στην ενότητα Το περιεχόμενό μου όπου θα αποθηκευτεί το αποτέλεσμα.
