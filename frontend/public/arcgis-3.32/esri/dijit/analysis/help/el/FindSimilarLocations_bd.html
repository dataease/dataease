Εύρεση Παρόμοιων Τοποθεσιών

Ανάλογα με τα κριτήρια που καθορίζετε, το εργαλείο Εύρεση Παρόμοιων Τοποθεσιών μετρά την ομοιότητα των τοποθεσιών στο υποψήφιο θεματικό επίπεδο αναζήτησης έναντι μίας ή περισσότερων τοποθεσιών αναφοράς. Αυτό το εργαλείο μπορεί να δώσει απάντηση σε ερωτήσεις, όπως

  • Ποιο από τα καταστήματά σας μοιάζει περισσότερο με εκείνα που έχουν τις περισσότερες πωλήσεις, όσον αφορά τα προφίλ πελατών;
  • Με βάση τα χαρακτηριστικά των χωριών που είχαν τα περισσότερα κρούσματα ασθένειας, ποια άλλα χωριά αντιμετωπίζουν μεγάλο κίνδυνο;
Για να απαντηθούν τέτοιες ερωτήσεις, παρέχετε τις τοποθεσίες αναφοράς, τις υποψήφιες τοποθεσίες αναζήτησης και τα πεδία που αντιπροσωπεύουν τα κριτήρια που θέλετε να πληρούνται. Το πρώτο θεματικό επίπεδο εισόδου θα πρέπει να περιέχει τις τοποθεσίες ή τα σημεία αναφοράς. Για παράδειγμα, μπορεί να είναι ένα θεματικό επίπεδο που περιέχει τα καταστήματα με τις καλύτερες πωλήσεις ή τα χωριά με τα περισσότερα κρούσματα της ασθένειας. Στη συνέχεια, καθορίζετε το θεματικό επίπεδο με τις προς αναζήτηση τοποθεσίες. Αυτό μπορεί να είναι όλα τα καταστήματά σας ή όλα τα άλλα χωριά. Τέλος, προσδιορίζετε ένα ή περισσότερα πεδία που θα χρησιμοποιηθούν για τη μέτρηση της ομοιότητας. Κατόπιν, το εργαλείο Εύρεση Παρόμοιων Τοποθεσιών θα κατατάξει όλες τις υποψήφιες τοποθεσίες αναζήτησης ανάλογα με την ομοιότητά τους με τις τοποθεσίες αναφοράς σε όλα τα πεδία που έχετε επιλέξει.

Επιλέξτε το θεματικό επίπεδο που περιέχει τις τοποθεσίες αναφοράς

Το θεματικό επίπεδο που περιέχει τις τοποθεσίες αναφοράς προς αντιστοίχιση.

Εκτός από την επιλογή ενός θεματικού επιπέδου από το χάρτη σας, μπορείτε επίσης να χρησιμοποιήσετε την Επιλογή θεματικού επιπέδου ανάλυσης στο κάτω μέρος της αναπτυσσόμενης λίστας για περιήγηση στο περιεχόμενό σας και αναζήτηση ενός κοινόχρηστου συνόλου δεδομένων αρχείων big data ή ενός feature layer.

Μπορείτε να χρησιμοποιήσετε όλες τις τοποθεσίες ή να επιλέξετε ορισμένες

Αν χρειάζεται, χρησιμοποιήστε τα κουμπιά επιλογής, για να προσδιορίσετε τις τοποθεσίες αναφοράς. Αυτή η επιλογή είναι διαθέσιμη μόνο αν τα στοιχεία σχεδιάζονται στο χάρτη (δεν είναι διαθέσιμη για κοινόχρηστο αρχείο μαζικών δεδομένων). Για παράδειγμα, αν το θεματικό επίπεδο εισόδου περιέχει όλες τις τοποθεσίες —τις τοποθεσίες αναφοράς και τις υποψήφιες τοποθεσίες αναζήτησης— θα πρέπει να χρησιμοποιήσετε ένα από τα εργαλεία επιλογής για να προσδιορίσετε τις τοποθεσίες αναφοράς. Αν δημιουργήσετε δύο ξεχωριστά θεματικά επίπεδα, ένα με τις τοποθεσίες αναφοράς και το άλλο με όλες τις υποψήφιες τοποθεσίες αναζήτησης, δεν χρειάζεται να επιλέξετε κάτι.

Αν υπάρχουν περισσότερες από μία τοποθεσίες αναφοράς, δημιουργείται μία τοποθεσία με βάση το μέσο όρο των τιμών κάθε πεδίου που χρησιμοποιείται για την ανάλυση της ομοιότητας.

Αναζήτηση παρόμοιων τοποθεσιών σε

Τα υποψήφια στοιχεία αναζήτησης αυτού του θεματικού επιπέδου θα καταταχθούν από τα περισσότερο όμοια προς τα λιγότερο όμοια.

Εκτός από την επιλογή ενός θεματικού επιπέδου από το χάρτη σας, μπορείτε επίσης να χρησιμοποιήσετε την Επιλογή θεματικού επιπέδου ανάλυσης στο κάτω μέρος της αναπτυσσόμενης λίστας για περιήγηση στο περιεχόμενό σας και αναζήτηση ενός κοινόχρηστου συνόλου δεδομένων αρχείων big data ή ενός feature layer.

Η ομοιότητα να βασιστεί σε

Τα πεδία που επιλέγετε θα είναι τα κριτήρια για την αξιολόγηση της ομοιότητας. Αν για παράδειγμα, επιλέξετε ένα πεδίο πληθυσμού και εισοδήματος, οι υποψήφιες τοποθεσίες αναζήτησης με τη χαμηλότερη (καλύτερη) κατάταξη θα είναι εκείνες που έχουν παρόμοιες τιμές πληθυσμού και εισοδήματος με τις τοποθεσίες αναφοράς.

Προσδιορίστε τα περισσότερο και λιγότερο παρόμοια χρησιμοποιώντας

Η μέθοδος που επιλέγετε προσδιορίζει τον τρόπο προσδιορισμού της αντιστοίχισης.

  • Η μέθοδος με τις τιμές πεδίου χρησιμοποιεί τη διαφορά των τυποποιημένων τιμών στο τετράγωνο. Αυτή είναι η προεπιλογή.
  • Η μέθοδος με τα προφίλ πεδίων χρησιμοποιεί μαθηματικά ομοιότητας συνημίτονου για τη σύγκριση του προφίλ τυποποιημένων τιμών. Η χρήση προφίλ γνωρισμένων απαιτεί τη χρήση τουλάχιστον δύο πεδίων ανάλυσης.

Εμφάνιση

Μπορείτε να δείτε όλες τις υποψήφιες τοποθεσίες αναζήτησης, καταταγμένες από την περισσότερο έως τη λιγότερο όμοια, ή μπορείτε να καθορίσετε τον αριθμό των αποτελεσμάτων που θέλετε να εμφανίζονται.

  • Αν καθορίσετε όλες τις τοποθεσίες από την περισσότερο έως τη λιγότερο όμοια, όλα τα στοιχεία του υποψήφιου θεματικού επιπέδου αναζήτησης θα συμπεριληφθούν στη σειρά κατάταξης του εξαγόμενου θεματικού επιπέδου. Θα επιστραφούν 10.000 αποτελέσματα το ανώτερο.
  • Αν καθορίσετε τον αριθμό των περισσότερο, λιγότερο, ή των περισσότερο και λιγότερο όμοιων τοποθεσιών, προσδιορίζετε πόσα από τα περισσότερο, περισσότερο και λιγότερο, και λιγότερο όμοια υποψήφια στοιχεία πρέπει να συμπεριληφθούν στο θεματικό επίπεδο των αποτελεσμάτων. Η μέγιστη επιτρεπόμενη τιμή είναι 10.000.

Επιλέξτε τα πεδία για προσθήκη αποτελεσμάτων

Προαιρετικά, προσθέστε πεδία στα δεδομένα σας από το θεματικό επίπεδο αναζήτησης. Η προεπιλογή είναι να προστίθενται όλα τα πεδία.

Όνομα εξαγόμενου θεματικού επιπέδου

Το όνομα του θεματικού επιπέδου που θα δημιουργηθεί. Αν πραγματοποιείτε εγγραφή σε ένα ArcGIS Data Store, τα αποτελέσματά σας θα αποθηκευτούν στον φάκελο Το περιεχόμενό μου και θα προστεθούν στον χάρτη. Αν πραγματοποιείτε εγγραφή σε ένα κοινόχρηστο αρχείο big data, τα αποτελέσματά σας θα αποθηκευτούν στο κοινόχρηστο αρχείο big data και θα προστεθούν στη διακήρυξή του. Δεν θα προστεθούν στον χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο υπάρχει ήδη, το εργαλείο θα αποτύχει.

Αυτό το εξαγόμενο θεματικό επίπεδο θα περιέχει τις τοποθεσίες αναφοράς και τον αριθμό των καταταγμένων υποψήφιων στοιχείων αναζήτησης που καθορίσατε. Αν το όνομα του εξαγόμενου θεματικού επιπέδου υπάρχει ήδη, θα σας ζητηθεί να το μετονομάσετε.

Όποτε πραγματοποιείτε εγγραφή σε ArcGIS Data Store (datastore για σχεσιακά ή χωροχρονικά big data) χρησιμοποιώντας το πτυσσόμενο πλαίσιο Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε μέσα στον φάκελο Το περιεχόμενό μου το όνομα ενός άλλου φακέλου όπου θα αποθηκευτεί το αποτέλεσμα.
