Ομαδοποίηση Ορίων

Οι περιοχές που επικαλύπτονται ή έχουν ένα κοινό όριο συγχωνεύονται για να σχηματίσουν μια ενιαία περιοχή.

Μπορείτε να καθορίσετε ένα πεδίο για να ελέγξετε ποια όρια συγχωνεύονται. Για παράδειγμα, αν έχετε ένα θεματικό επίπεδο κομητειών και κάθε κομητεία έχει ένα γνώρισμα State_Name, μπορείτε να ομαδοποιήσετε τα όρια χρησιμοποιώντας το γνώρισμα State_Name. Οι γειτονικές κομητείας θα συγχωνευθούν αν έχουν την ίδια τιμή για το State_Name. Το τελικό αποτέλεσμα είναι ένα θεματικό επίπεδο με τις επαρχίες του κράτους.

Επιλέξτε το θεματικό επίπεδο της περιοχής της οποίου τα όρια θα ομαδοποιηθούν

Το θεματικό επίπεδο που περιέχει τα στοιχεία περιοχής που θα ομαδοποιηθούν

Μπορείτε να επιλέξετε ένα θεματικό επίπεδο από τον χάρτη σας αλλά και να χρησιμοποιήσετε και τη δυνατότητα Επιλογή θεματικού επιπέδου ανάλυσης που βρίσκεται στο κάτω μέρος της πτυσσόμενης λίστας. Με την επιλογή αυτή ανοίγει μία συλλογή στην οποία μπορείτε να περιηγηθείτε και να επιλέξετε από διάφορες πηγές σύνολα δεδομένων που θα χρησιμοποιηθούν στην ανάλυσή σας.

Επιλογή μεθόδου ομαδοποίησης

Υπάρχουν δύο μέθοδοι για την ομαδοποίηση των γνωρισμάτων μιας περιοχής:

  • Περιοχές που επικαλύπτονται ή είναι γειτονικές—Δύο ή περισσότερες περιοχές τα όρια των οποίων επικαλύπτονται θα συγχωνευθούν σε μία περιοχή. Επιπλέον, οι περιοχές που είναι γειτονικές (έχουν δηλαδή ένα κοινό όριο) θα συγχωνευθούν σε μία περιοχή.
  • Περιοχές με την ίδια τιμή πεδίου—Περιοχές που επικαλύπτονται ή έχουν ένα κοινό όριο θα συγχωνευθούν σε μία περιοχή, αν έχουν την ίδια τιμή πεδίου.

    Για παράδειγμα, αν έχετε ένα θεματικό επίπεδο κομητειών και κάθε κομητεία έχει ένα πεδίο State_Name, μπορείτε να ομαδοποιήσετε τα όρια χρησιμοποιώντας το πεδίο State_Name. Οι γειτονικές κομητείας θα συγχωνευθούν αν έχουν την ίδια τιμή για το State_Name. Το τελικό αποτέλεσμα είναι ένα θεματικό επίπεδο με τις επαρχίες του κράτους.

Αν είναι επιλεγμένη η Χρήση τρέχουσας έκτασης χάρτη, θα απαλειφθούν τα όρια μόνο όσων στοιχείων του θεματικού επιπέδου εισόδου είναι ορατά στην τρέχουσα έκταση του χάρτη. Αν δεν είναι επιλεγμένο, θα ομαδοποιηθούν όλα τα στοιχεία του θεματικού επιπέδου εισόδου, ακόμη και αν βρίσκονται έξω από την τρέχουσα έκταση του χάρτη.

Δημιουργία Πολυτμηματικών Στοιχείων

Στο ArcGIS μια ψηφιακή αναπαράσταση ενός τόπου ή πράγματος που έχει περισσότερα από ένα τμήματα αλλά καθορίζεται ως ένα στοιχείο επειδή παραθέτει ένα σύνολο γνωρισμάτων. Για παράδειγμα, σε ένα θεματικό επίπεδο πολιτειών η πολιτεία της Χαβάης θα μπορούσε να θεωρηθεί πολυτμηματικό στοιχείο επειδή τα χωριστά του γεωμετρικά τμήματα ταξινομούνται ως ενιαία πολιτεία. Αυτή η boolean παράμετρος προσφέρει δύο επιλογές:

  • Αληθής (επιλεγμένη): η υπηρεσία που θα προκύψει μπορεί να περιέχει πολυτμηματικά στοιχεία.
  • Ψευδής (μη επιλεγμένη): η υπηρεσία που θα προκύψει θα περιέχει μόνο μονοτμηματικά στοιχεία και θα δημιουργηθούν μεμονωμένα στοιχεία για κάθε τμήμα.

Προσθήκη στατιστικών στοιχείων

Όταν συγχωνεύονται περιοχές, μπορείτε να συνοψίσετε τα γνωρίσματά τους. Για παράδειγμα, αν θέλετε να ομαδοποιήσετε το όριο μεταξύ κομητειών με βάση το State_Name και κάθε κομητεία έχει ένα πεδίο Population, θα μπορούσατε να υπολογίσετε το άθροισμα του στοιχείου Population για την πολιτεία.

Οι διαθέσιμοι τύποι στατιστικών στοιχείων είναι οι εξής:

  • Άθροισμα—Προσθέτει τη συνολική τιμή για το καθορισμένο πεδίο
  • Μέσος όρος—Υπολογίζει το μέσο όρο για το καθορισμένο πεδίο
  • Ελάχιστο—Βρίσκει τη μικρότερη τιμή για όλες τις εγγραφές του καθορισμένου πεδίου
  • Μέγιστο—Βρίσκει τη μεγαλύτερη τιμή για όλες τις εγγραφές του καθορισμένου πεδίου
  • Τυπικό. απόκλιση—Βρίσκει την τυπική απόκλιση των τιμών του καθορισμένου πεδίου

Όνομα εξαγόμενου θεματικού επιπέδου

Αυτό είναι το όνομα του θεματικού επιπέδου που θα δημιουργηθεί στην ενότητα Το περιεχόμενό μου και θα προστεθεί στον χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το όνομα του θεματικού επιπέδου ήδη υπάρχει, θα εμφανιστεί προτροπή για να καταχωρίσετε κάποιο άλλο όνομα.

Χρησιμοποιώντας το πτυσσόμενο μενού Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε το όνομα ενός φακέλου στην ενότητα Το περιεχόμενό μου όπου θα αποθηκευτεί το αποτέλεσμα.
