Συνάθροιση Σημείων

Το εργαλείο αυτό λειτουργεί με ένα σημειακό θεματικό επίπεδο και ένα θεματικό επίπεδο περιοχής. Τα στοιχεία περιοχής εισόδου μπορεί να προέρχονται από ένα πολυγωνικό θεματικό επίπεδο ή μπορεί να είναι τετράγωνα ή εξάγωνα bin που υπολογίζονται κατά την εκτέλεση του εργαλείου. Το εργαλείο αρχικά προσδιορίζει τα σημεία που ανήκουν σε κάθε καθορισμένη περιοχή. Αφού προσδιοριστεί αυτή η χωρική σχέση σημείου και περιοχής, υπολογίζονται τα στατιστικά στοιχεία για όλα τα σημεία της περιοχής και αντιστοιχίζονται στην περιοχή. Το πιο βασικό στατιστικό στοιχείο είναι η μέτρηση του αριθμού των σημείων εντός της περιοχής, αλλά μπορείτε, επίσης, να υπολογίσετε και άλλα στατιστικά στοιχεία.

Για παράδειγμα, ας υποθέσουμε ότι έχετε στοιχεία σημείου για τοποθεσίες με καφετέριες και στοιχεία περιοχής για επαρχίες και θέλετε να συνοψίσετε τις πωλήσεις καφέ ανά επαρχία. Θεωρώντας ότι οι καφετέριες διαθέτουν ένα περιγραφικό γνώρισμα TOTAL_SALES, μπορείτε να βρείτε το άθροισμα όλων των TOTAL_SALES κάθε επαρχίας, τις μέγιστες ή ελάχιστες τιμές TOTAL_SALES κάθε επαρχίας, ή άλλα στατιστικά στοιχεία, όπως το πλήθος, την τυπική απόκλιση και την διακύμανση.

Το εργαλείο μπορεί επίσης να λειτουργεί με δεδομένα που έχουν χρονική πληροφορία. Αν είναι ενεργοποιημένος ο χρόνος στα σημεία εισόδου, τότε είναι διαθέσιμες οι επιλογές χρονικής κατάτμησης. Η χρονική κατάτμηση σάς επιτρέπει να υπολογίσετε τη σχέση σημείου και περιοχής, εξετάζοντας ένα συγκεκριμένο χρονικό τμήμα. Για παράδειγμα, θα μπορούσατε να εξετάσετε ωριαία διαστήματα, πράγμα που θα είχε ως αποτέλεσμα εξόδους για κάθε ώρα.

Για ένα παράδειγμα που αφορά το χρόνο, ας υποθέσουμε ότι είχατε σημειακά στοιχεία κάθε συναλλαγής που έγινε σε διάφορες τοποθεσίες με καφετέριες και κανένα πολυγωνικό θεματικό επίπεδο. Τα δεδομένα έχουν καταγραφεί για πάνω από ένα χρόνο και κάθε συναλλαγή έχει μια τοποθεσία και χρονική σήμανση. Θεωρώντας ότι κάθε συναλλαγή έχει ένα περιγραφικό γνώρισμα TOTAL_SALES, μπορείτε να βρείτε το άθροισμα όλων των TOTAL_SALES εντός του χώρου και του χρόνου ενδιαφέροντος. Αν αυτές οι συναλλαγές αφορούν μία μόνο πόλη, θα μπορούσαμε να δημιουργήσουμε περιοχές που αποτελούν πλέγματα 1 χιλιομέτρου και να εξετάσουμε εβδομαδιαία χρονικά τμήματα ώστε να συνοψίσουμε τις συναλλαγές τόσο χρονικά όσο και χωρικά.

Επιλέξτε ένα θεματικό επίπεδο που να περιέχει σημεία τα οποία θα συναθροιστούν σε περιοχές

Το σημειακό θεματικό επίπεδο για το οποίο θα γίνει συνάθροιση στις περιοχές. Για τα σημειακά θεματικά επίπεδα που συναθροίζονται σε τετράγωνα ή εξάγωνα bin, η χωρική αναφορά της επεξεργασίας πρέπει να είναι ορισμένη σε προβολικό σύστημα συντεταγμένων μέσα από τα Περιβάλλοντα ανάλυσης.

Εκτός από την επιλογή ενός θεματικού επιπέδου από το χάρτη σας, μπορείτε επίσης να χρησιμοποιήσετε την Επιλογή θεματικού επιπέδου ανάλυσης στο κάτω μέρος της αναπτυσσόμενης λίστας για περιήγηση στο περιεχόμενό σας και αναζήτηση ενός κοινόχρηστου συνόλου δεδομένων αρχείων big data ή ενός feature layer.

Διαλέξτε ένα θεματικό επίπεδο περιοχής για συνάθροιση σε

Οι περιοχές στις οποίες θα συναθροιστεί το σημειακό θεματικό επίπεδο. Μπορείτε να επιλέξετε είτε ένα θεματικό επίπεδο περιοχής είτε bin τα οποία θα δημιουργηθούν κατά την εκτέλεση του εργαλείου. Κατά τη δημιουργία των bin, για Τετράγωνο, ο αριθμός και οι μονάδες που καθορίζονται προσδιορίζουν το ύψος και το μήκος του τετραγώνου. Για Εξάγωνο, ο αριθμός και οι μονάδες που καθορίζονται προσδιορίζουν την απόσταση μεταξύ των παράλληλων πλευρών.

Για την ανάλυση με χρήση τετράγωνων ή εξάγωνων bin, απαιτείται προβολικό σύστημα συντεταγμένων. Μπορείτε να ρυθμίσετε το Σύστημα συντεταγμένων επεξεργασίας στα Περιβάλλοντα ανάλυσης. Αν το σύστημα επεξεργασίας συντεταγμένων που χρησιμοποιείτε δεν έχει καθοριστεί να είναι προβολικό σύστημα συντεταγμένο, θα σας παρουσιαστεί προτροπή για να το καθορίσετε κατά την Εκτέλεση ανάλυσης .

Διαλέξτε το πολυγωνικό θεματικό επίπεδο για συνάθροιση

Το πολυγωνικό θεματικό επίπεδο όπου θα συναθροιστούν τα σημεία. Τα σημεία που βρίσκονται εντός των ορίων των περιοχών αυτού του επιπέδου θα μετρηθούν και, προαιρετικά, θα συνοψιστούν με την επιλογή στατιστικής που κάνατε.

Πρέπει να καθορίσετε είτε Μέγεθος bin για τα bin είτε Θεματικό επίπεδο περιοχής.

Διαλέξτε απόσταση για να δημιουργήσετε bin και να κάνετε συνάθροιση σε

Η απόσταση που χρησιμοποιείται για τη δημιουργία bin στα οποία θα συναθροιστούν τα σημεία εισόδου. Για ένα τετράγωνο bin, το μέγεθος αντιπροσωπεύει το ύψος του τετραγώνου. Για εξάγωνο, το μέγεθος αντιπροσωπεύει το ύψος του εξαγώνου (από τις παράλληλες πλευρές).

Πρέπει να καθορίσετε είτε Μέγεθος bin για τα bin είτε Θεματικό επίπεδο περιοχής.

Συνάθροιση χρησιμοποιώντας χρονικά βήματα (προαιρετικό)

Αν είναι ενεργοποιημένος ο χρόνος στο σημειακό θεματικό επίπεδο εισόδου, και ως τύπος έχει επιλεχθεί στιγμή, η ανάλυση μπορεί να γίνει χρησιμοποιώντας χρονική κατάτμηση. Υπάρχουν τρεις παράμετροι που μπορείτε να ορίσετε με χρήση του χρόνου:

  • Διάστημα χρονικού βήματος
  • Συχνότητα επανάληψης χρονικού βήματος
  • Χρόνος για ευθυγράμμιση των χρονικών βημάτων

Για παράδειγμα, αν έχετε δεδομένα που αντιπροσωπεύουν ένα έτος και θέλετε να τα αναλύσετε χρησιμοποιώντας εβδομαδιαία βήματα, δώστε στην επιλογή Διάστημα χρονικού βήματος τη ρύθμιση 1 εβδομάδα.

Για παράδειγμα, αν έχετε δεδομένα που αντιπροσωπεύουνs ένα έτος και θέλετε να τα αναλύσετε χρησιμοποιώντας την πρώτη εβδομάδα του μήνα, δώστε στην επιλογή Διάστημα χρονικού βήματος τη ρύθμιση 1 εβδομάδα, στην επιλογή Συχνότητα επανάληψης χρονικού βήματος τη ρύθμιση 1 μήνας και στην επιλογή Χρόνος για ευθυγράμμιση των χρονικών βημάτων τη ρύθμιση 1 Ιανουαρίου στις 12:00 π.μ.

Διάστημα χρονικού βήματος

Το χρονικό διάστημα για τη δημιουργία χρονικών βημάτων. Η επιλογή Διάστημα χρονικού βήματος μπορεί χρησιμοποιηθεί μόνη της ή με την παράμετρο Συχνότητα επανάληψης χρονικού βήματος ή με την παράμετρο Χρόνος για ευθυγράμμιση των χρονικών βημάτων.

Για παράδειγμα, αν θέλετε να δημιουργήσετε χρονικά βήματα που θα πραγματοποιούνται κάθε Δευτέρα από τις 9:00 π.μ. ως τις 10:00 π.μ., δώστε στην επιλογή Διάστημα χρονικού βήματος τη ρύθμιση 1 ώρα, στην επιλογή Συχνότητα επανάληψης χρονικού βήματος τη ρύθμιση 1 εβδομάδα και στην επιλογή Χρόνος για ευθυγράμμιση των χρονικών βημάτων τη ρύθμιση 9:00:00 π.μ. κάθε Δευτέρα.

Συχνότητα επανάληψης χρονικού βήματος

Το βήμα που χρησιμοποιείται για τον υπολογισμό ενός χρονικού βήματος. Η επιλογή Συχνότητα επανάληψης χρονικού βήματος μπορεί να χρησιμοποιηθεί μόνη της, με την επιλογή Διάστημα χρονικού βήματος, με την επιλογή Χρόνος αναφοράς ή με την επιλογή Διάστημα χρονικού βήματος και την επιλογή Χρόνος για ευθυγράμμιση των χρονικών βημάτων ταυτόχρονα.

Για παράδειγμα, αν θέλετε να δημιουργήσετε χρονικά βήματα που θα πραγματοποιούνται κάθε Δευτέρα από τις 9:00 π.μ. ως τις 10:00 π.μ., δώστε στην επιλογή Διάστημα χρονικού βήματος τη ρύθμιση 1 ώρα, στην επιλογή Συχνότητα επανάληψης χρονικού βήματος τη ρύθμιση 1 εβδομάδα και στην επιλογή Χρόνος για ευθυγράμμιση των χρονικών βημάτων τη ρύθμιση 9:00:00 π.μ. κάθε Δευτέρα.

Χρόνος για ευθυγράμμιση των χρονικών βημάτων

Η ημερομηνία και η ώρα που χρησιμοποιούνται για ευθυγράμμιση της χρονικής κατάτμησης. Η διαδικασία των χρονικών βημάτων θα αρχίσει στις και θα συνεχιστεί αντίστροφα από εκείνη την ώρα. Αν δεν έχει επιλεγεί χρόνος αναφοράς, η χρονική κατάτμηση θα ευθυγραμμιστεί στην 1η Ιανουαρίου 1970.

Για παράδειγμα, αν θέλετε να δημιουργήσετε χρονικά βήματα που θα πραγματοποιούνται κάθε Δευτέρα από τις 9:00 π.μ. ως τις 10:00 π.μ., δώστε στην επιλογή Διάστημα χρονικού βήματος τη ρύθμιση 1 ώρα, στην επιλογή Συχνότητα επανάληψης χρονικού βήματος τη ρύθμιση 1 εβδομάδα και στην επιλογή Χρόνος για ευθυγράμμιση των χρονικών βημάτων τη ρύθμιση 9:00:00 π.μ. κάθε Δευτέρα.

Προσθήκη στατιστικών (προαιρετικά)

Μπορείτε να υπολογίσετε στατιστικά για τα στοιχεία που συνοψίζονται.Μπορείτε να υπολογίσετε τα στοιχεία που ακολουθούν στα αριθμικά πεδία:

  • Αριθμός—Υπολογίζει τον αριθμό τιμών που δεν είναι μηδενικές (null). Μπορεί να χρησιμοποιηθεί σε αριθμικά πεδία ή σε συμβολοσειρές. Το πλήθος των [null, 0, 2] είναι 2.
  • Άθροισμα—Το άθροισμα των αριθμητικών τιμών σε ένα πεδίο. Το άθροισμα των [null, null, 3] είναι 3.
  • Μέση τιμή—Η μέση τιμή αριθμητικών τιμών. Η μέση τιμή των [0, 2, null] είναι 1.
  • Ελάχιστο—Η ελάχιστη τιμή ενός αριθμητικού πεδίου. Το ελάχιστο των [0, 2, null] είναι 0.
  • Μέγιστο—Η μέγιστη τιμή ενός αριθμητικού πεδίου. Η μέγιστη τιμή των [0, 2, null] είναι 2.
  • Εύρος—Το εύρος ενός αριθμητικού πεδίου. Αυτό υπολογίζεται ως οι ελάχιστες τιμές που αφαιρούνται από την μέγιστη τιμή. Το εύρος των [0, null, 1] είναι 1. Το εύρος των [null, 4] είναι 0.
  • Διακύμανση—Η διακύμανση ενός αριθμητικού πεδίου σε ένα ίχνος. Η διακύμανση του [1] είναι null. Η διακύμανση των τιμών [null, 1,0,1,1] είναι 0,25.
  • Τυπική απόκλιση—Η τυπική απόκλιση ενός αριθμητικού πεδίου. Η τυπική απόκλιση του [1] είναι null. Η τυπική απόκλιση των τιμών [null, 1,0,1,1] είναι 0,5.

Μπορείτε να υπολογίσετε τα στοιχεία που ακολουθούν στα πεδία συμβολοσειρών:

  • Πλήθος—Ο αριθμός των συμβολοσειρών που δεν είναι null.
  • Οποιαδήποτε—Αυτό το στατιστικό στοιχείο είναι ένα τυχαίο δείγμα μιας τιμής συμβολοσειράς στο καθορισμένο πεδίο.
Όλα τα στατιστικά στοιχεία υπολογίζονται σε τιμές που δεν είναι null. Το εξαγόμενο θεματικό επίπεδο θα περιέχει ένα νέο πεδίο για κάθε υπολογιζόμενο στατιστικό στοιχείο. Μπορεί να προστεθεί οποιοσδήποτε αριθμός στατιστικών στοιχείων, επιλέγοντας γνώρισμα και στατιστικό στοιχείο.

Χωρική αναφορά (wkid)

Είναι μια προσωρινή παράμετρος για προέκδοση με στόχο τον καθορισμό της χωρικής αναφοράς της επεξεργασίας. Πολλά εργαλεία big data απαιτούν τη χρήση ενός προβολικού συστήματος συντεταγμένων ως τη χωρική αναφορά της επεξεργασίας. Από προεπιλογή, το εργαλείο θα χρησιμοποιεί το σύστημα συντεταγμένων εισόδου αλλά θα αποτύχει εάν πρόκειται για σύστημα γεωγραφικών συντεταγμένων. Για να ορίσετε ένα προβολικό σύστημα συντεταγμένων, εισαγάγετε το WKID. Για παράδειγμα, για το Web Mercator πρέπει να εισαγάγετε 3857.

Επιλέξτε ένα ArcGIS Data Store για αποθήκευση των αποτελεσμάτων

Τα αποτελέσματα GeoAnalytics αποθηκεύονται σε ένα datastore και εκτίθενται ως feature layer στο Portal for ArcGIS. Στις περισσότερες περιπτώσεις, τα αποτελέσματα λογικά θα αποθηκεύονται στο datastore για χωροχρονικά δεδομένα και αυτή είναι η προεπιλεγμένη ρύθμιση. Σε ορισμένες περιπτώσεις, μια καλή επιλογή είναι η αποθήκευση αποτελεσμάτων στο σχεσιακό data store. Ακολουθούν ορισμένοι λόγοι για τους οποίους θα θέλατε να αποθηκεύσετε αποτελέσματα στο σχεσιακό data store:

  • Μπορείτε να χρησιμοποιήσετε αποτελέσματα σε συνεργασία portal με portal.
  • Μπορείτε να ενεργοποιήσετε τις δυνατότητες συγχρονισμού με τα αποτελέσματά σας.

Δεν πρέπει να χρησιμοποιείτε το σχεσιακό data store αν περιμένετε αύξηση των αποτελεσμάτων GeoAnalytics και επιθυμείτε να εκμεταλλευτείτε τις δυνατότητες του χωροχρονικού data store μαζικών δεδομένων για τη διαχείριση μεγάλων όγκων δεδομένων.

Όνομα εξαγόμενου θεματικού επιπέδου

Το όνομα του θεματικού επιπέδου που θα δημιουργηθεί. Αν πραγματοποιείτε εγγραφή σε ένα ArcGIS Data Store, τα αποτελέσματά σας θα αποθηκευτούν στον φάκελο Το περιεχόμενό μου και θα προστεθούν στον χάρτη. Αν πραγματοποιείτε εγγραφή σε ένα κοινόχρηστο αρχείο big data, τα αποτελέσματά σας θα αποθηκευτούν στο κοινόχρηστο αρχείο big data και θα προστεθούν στη διακήρυξή του. Δεν θα προστεθούν στον χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο υπάρχει ήδη, το εργαλείο θα αποτύχει.

Όποτε πραγματοποιείτε εγγραφή σε ArcGIS Data Store (datastore για σχεσιακά ή χωροχρονικά big data) χρησιμοποιώντας το πτυσσόμενο πλαίσιο Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε μέσα στον φάκελο Το περιεχόμενό μου το όνομα ενός άλλου φακέλου όπου θα αποθηκευτεί το αποτέλεσμα.
