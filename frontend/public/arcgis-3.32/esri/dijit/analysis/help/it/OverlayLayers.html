Sovrapponi layer

Consente di combinare due o più layer in un unico layer. La sovrapposizione è un processo analogo all'analisi di un insieme di mappe ed alla creazione di un unica mappa contenente tutte le informazioni trovate nel gruppo. La sovrapposizione non è una semplice unione di linee; nel prodotto finale vengono infatti inclusi tutti gli attributi delle feature coinvolte nella sovrapposizione. La sovrapposizione costituisce la risposta ad una delle più elementari domande di geografia, ovvero "Cosa sta sopra e cosa sta sotto?"Ad esempio:

  • Quali lotti sono compresi nella pianura alluvionaleda 100 anni?
  • Quali strade sono comprese in determinate regioni?
  • Quale uso del suolo si fa di un determinato tipo di terreno?
  • Quali pozzi si trovano in basi militari abbandonate?

Se l'opzione Usa estensione mappa corrente è selezionata, verranno sovrapposte solo le feature nel layer di input e di sovrapposizione visibili nell'estensione mappa corrente. Se è deselezionata, verranno sovrapposte tutte le feature sia nel layer di input che in quello di sovrapposizione, anche se esterne all'estensione mappa corrente.

Scegli il layer di input

Il layer di punti, linea o area che verrà sovrascritto.

Oltre a scegliere un layer dalla mappa, è possibile selezionare Scegli layer di analisi in fondo all'elenco a discesa. Con questa opzione si apre una galleria in cui è possibile individuare e selezionare dataset di varie fonti da utilizzare nell'analisi.

Scegliere il metodo di sovrapposizione

Il metodo di sovrapposizione definisce in che modo vengono combinati il layer di input e quello di sovrapposizione.

  • Incrocia: il risultato conterrà solo le feature o le parti di feature del layer di sovrapposizione che risultano sovrapposte a feature del layer di input. Quando si intersecano feature lineari o di area, è possibile specificare il tipo di feature di output.

  • Unione: il risultato conterrà feature sia del layer di input che di quello di sovrapposizione. Questa opzione è disponibile solo se il layer di input e quello di sovrapposizione contengono entrambi feature di area.
  • Cancella: il risultato conterrà le feature o le parti di feature del layer di sovrapposizione che non risultano sovrapposte ad aree del layer di input. Il layer di sovrapposizione può contenere feature area, linea o punto.

Se l'opzione Usa estensione mappa corrente è selezionata, verranno sovrapposte solo le feature nel layer di input e di sovrapposizione visibili nell'estensione mappa corrente. Se è deselezionata, verranno sovrapposte tutte le feature sia nel layer di input che in quello di sovrapposizione, anche se esterne all'estensione mappa corrente.

Nome layer risultati

Questo è il nome del layer che verrà creato in I miei contenuti e aggiunto alla mappa. Il tipo di feature del layer, ovvero area, linea o punto, è uguale a quello di sovrapposizione. Il nome predefinito è basato sul metodo di sovrapposizione e sul nome del layer di input. Se il layer esiste già, verrà chiesto di fornire un altro nome.

Utilizzare la casella a discesa Salva risultato in per specificare il nome di una cartella di I miei contenuti in cui salvare il risultato.
