尋找相似位置

根據指定條件，「尋找相似位置」工具將測量候選搜尋圖層中的位置與一個或多個參考位置之間的相似性。此工具可回答以下的問題

  • 在客戶概况方面，您的哪些商店與績效最佳的商店最爲相似？
  • 根據遭受此疾病侵襲最嚴重的村莊特徵，其他哪些村莊有較高的風險？
要回答此類問題，應提供參考位置、候選搜尋位置和代表匹配條件的欄位。第一個輸入圖層應包含參考位置或基準位置。例如，這可能是包含績效最佳的商店或遭受此疾病侵襲最嚴重的村莊所在的圖層。然後指定包含候選搜尋位置的圖層。這可能是所有商店或所有其他村莊。最後，標識出一個或多個用來測量相似性的欄位。隨後，「尋找相似位置」工具將在所有已選欄位中，按候選搜尋位置與參考位置之間的相近程度，對所有候選搜尋位置排序。

選擇包含參考位置的圖層

包含待匹配參考位置的圖層。

除了從地圖選擇圖層外，您也可以選擇下拉式清單底部的 選擇分析圖層，以瀏覽到大數據檔案分享資料集或圖徵圖層的內容。

您可以使用所有位置或進行選擇

如有必要，請使用選擇按鈕標識參考位置。只有在地圖上繪製圖徵時，才能使用這個選項（不適用於大資料檔案分享）。例如，如果輸入圖層包含所有位置（參考位置以及候選搜尋位置），則需要使用其中一個選擇工具來標識參考位置。如果建立兩個獨立的圖層，其中一個包含參考位置而另一個包含所有候選搜尋位置，則不需要進行選擇。

如果存在多個參考位置，則透過對每個用於分析相似性的欄位求平均值來建立單個位置。

相似性依據

所選欄位將成為用於評估相似性的條件。例如，如果選擇人口和收入欄位，則具有最低（最佳）等級的候選搜尋位置將是那些與參考位置具有相似人口和收入值的位置。

使用下列方式判定最相似與最不相似

選擇的方法決定了判定的符合程度。

  • 欄位值方法使用標準化值的平方差。這是預設情況。
  • 欄位設定檔方法使用餘弦相似性數學來比較標準化值的設定檔。使用屬性設定檔至少需要使用兩個分析欄位。

詳情介紹

您可以查看從最相似到最不相似排序的所有候選搜尋位置，或指定想要查看的結果數。

  • 如果指定 從最相似到最不相似的所有位置，候選搜尋圖層的所有圖徵將以排序順序加入結果圖層中。最多將傳回 10,000 個結果。
  • 如果指定 最相似、最不相似，或最相似和最不相似 位置的 數量，則確定應包括在結果圖層中的最相似、最不相似或最相似，及最不相似的候選數量。允許的最大值為 10,000。

結果圖層名稱

將建立之圖層的名稱。若是寫入到 ArcGIS Data Store，您的結果將儲存於我的內容並新增至地圖中。若是寫入到大數據檔案分享，您的結果將儲存於大數據檔案分享並新增至其資訊清單中。不會將它新增到地圖。預設名稱參照工具名稱以及輸入圖層名稱。如果圖層已存在，工具將失敗。

該結果圖層將包含參考位置和所指定的已排序之候選搜尋圖徵的數量。如果結果圖層名稱已存在，則系統會要求您重新命名該圖層。

使用 將結果儲存到下拉式方塊寫入到 ArcGIS Data Store（關聯式或大型的時空資料儲存庫）時，可以在我的內容中指定將儲存結果的資料夾名稱。
