데이터 추출(Extract Data)

이 도구를 사용하면 지정된 관심 영역에 대한 데이터를 선택하여 다운로드할 수 있습니다. 선택한 레이어는 ZIP, CSV 또는 KMZ 파일에 추가됩니다.

추출할 레이어

콘텐츠 목록에 있는 레이어 중에서 피처를 추출할 레이어입니다. 선택한 레이어는 추출되어 결과 ZIP 또는 CSV 파일에 포함됩니다.

추출할 레이어 목록에 레이어를 나타내려면 데이터를 소유하거나 데이터 소유자가 데이터를 내보내도록 허용해야 합니다. 소유한 데이터에 대해 데이터 내보내기를 활성화하려면

  • 데이터의 항목 설명을 엽니다.
  • 설정 탭을 클릭합니다.
  • 데이터 내보내기 섹션이 나올 때까지 화면을 아래로 스크롤합니다.
  • 다른 사용자가 다른 형식으로 내보내도록 허용을 선택합니다. 작업을 마치면 저장을 클릭합니다.

분석 레이어에 첨부 파일이 포함된 경우 결과 데이터 형식이 파일 지오데이터베이스이고 클립 피처의 선택이 취소되어 있을 때만 첨부 파일이 추출됩니다.

처리 범위

결과 ZIP 또는 CSV 파일에 포함할 피처를 정의하는 영역입니다. 다음 중에서 선택할 수 있습니다.

  • 맵의 현재 가시 범위( 화면 범위와 동일함)
  • 콘텐츠 목록에 있는 레이어 중 하나 선택한 레이어가 포인트나 라인 피처인 경우 레이어의 범위가 연구 영역으로 사용됩니다.
  • 처리 범위를 정의하는 영역 그리기
맵 범위 내에 완전히 포함된 모든 피처가 추출됩니다. 완전히 범위 내에 있지 않고 맵 범위 경계를 가로지르거나 교차하는 피처의 경우 다음 두 가지 옵션 중에서 선택할 수 있습니다.
  • 피처 선택 - 처리 범위의 경계를 가로지르는 피처 전체가 추출됩니다.
  • 클립 피처 - 처리 범위 내에 있는 피처 부분만 추출됩니다.

결과 데이터 형식

추출할 데이터의 형식입니다. 라인 또는 영역 피처를 CSV 파일로 내보낼 경우 비공간 속성만 추출되고, 포인트 피처를 내보낼 경우 공간 속성과 비공간 속성이 둘 다 추출됩니다.

추출된 데이터는 웹 메르카토르 1984(Web Mercator 1984) 투영을 사용합니다. CSV 파일의 경우 Lat 및 Long 필드가 웹 메르카토르 1984 좌표계를 사용하여 데이터에 추가됩니다.

지원되는 결과 형식은 다음과 같습니다.

  • 쉼표로 구분된 값(CSV) - .csv 또는 .zip
  • 파일 지오데이터베이스 - .zip
  • 키홀 마크업 언어(KML) - .kmz 또는 .zip
  • 쉐이프파일 - .zip

결과 파일 이름

내 콘텐츠에 추가되는 항목의 이름입니다. 항목 이름은 선택한 출력 데이터 형식을 기반으로 합니다. 항목이 이미 있는 경우 다른 이름을 제공하라는 메시지가 나타납니다.

결과 저장: 드롭다운 상자를 사용하여 내 콘텐츠에서 결과를 저장할 폴더의 이름을 지정할 수 있습니다.
