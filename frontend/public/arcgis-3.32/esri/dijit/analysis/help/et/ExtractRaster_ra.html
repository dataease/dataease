Ekstrakti raster

Selle tööriistaga saab ekstraktida piiri rastri nelinurkse alaga või ekraanil interaktiivselt vaba käega markeeritud ulatusega. Saate ekstraktida parajasti kuvatava kaardiala ulatuses või polügooniga määratletud uuringuala ulatuses.

Määrake kärpimise geomeetria

Valige ulatuse tüüp kihi geomeetria kärpimiseks.

  • Kasuta praegust ulatust – kihi kärpimiseks kasutatakse kaardi praegust ulatust. See on vaikimisi valik.
  • Vali uuringu ala – saate valida, kas tulemuses näidatakse ala geomeetria ulatuse piires või väljaspool geomeetria ulatust. Kui valite valiku Geomeetria sisse jäävad andmed, eemaldatakse ulatusest välja jäävad pildid. Kui valite valiku Geomeetriast väljapoole jäävad andmed, eemaldatakse ulatuse sisse jäävad pildid. Kärpimise geomeetria saab määratleda ka joonistamise tööriistaga.

Tulemkihi nimi

Selle kihi nimi, mis luuakse jaotises Minu sisu ja lisatakse kaardile. Vaikimisi nimi põhineb tööriista nimel ja sisendkihi nimel. Kui kiht on juba olemas, palutakse teil sisestada mõni muu nimi.

You can specify the name of a folder in My Content where the result will be saved using the Save result in drop-down box.
