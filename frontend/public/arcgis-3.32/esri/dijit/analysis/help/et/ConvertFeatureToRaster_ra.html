Teisenda vektorist rastriks

See tööriist teisendab vektorobjektid rasterkihiks.

Iga objektiklassi, mis sisaldab punkt-, joon- või pindobjekte, saab teisendada rasterandmestikuks.

Väljundi rasterpiksli väärtus määratakse piksli keskpunkti järgi.

Väljundrastri tüübi määrab sisendvälja tüüp. Kui väli on täisarv, on väljundraster täisarv; kui väli on ujukomaarv, on ka väljund ujukomaarv.

Valige teisendatav objektikiht

Rasterandmestikuks teisendatav sisendobjektide kiht.

Valige väli

Valige väli, mida kasutatakse väljundrastrile väärtuste määramiseks.

Väljundpiksli suurus

The cell size and unit for the output rasters.

Saadaolevad ühikud on miil, jalg, kilomeeter ja meeter. Vaikeühik on miil.

Tulemkihi nimi

Selle kihi nimi, mis luuakse jaotises Minu sisu ja lisatakse kaardile. Vaikimisi nimi põhineb tööriista nimel ja sisendkihi nimel. Kui kiht on juba olemas, palutakse teil sisestada mõni muu nimi.

You can specify the name of a folder in My Content where the result will be saved using the Save result in drop-down box.
