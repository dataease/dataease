Melakukan Geocode alamat dari Tabel

Geocoding adalah cara untuk mengubah alamat menjadi koordinat. Alat ini mengonversi alamat dari tabel, file CSV, atau file Excel menjadi feature layer atau file output dengan koordinat sebagai kolom tambahan.

Pilih tabel input

Tabel input dengan alamat akan di-geocode. Input harus berupa tabel, file CSV, atau file Excel.

Pilih penentu lokasi

Pilih service penentu lokasi untuk melakukan geocode. Jika service penentu lokasi yang ingin Anda gunakan tidak berada di menu tarik-turun, hubungi administrator agar service penentu lokasi dikonfigurasi untuk geocoding kelompok.

Negara

Jika semua alamat Anda berasal dari satu negara, pilih negara tersebut untuk memastikan hasil yang paling akurat. Jika tidak, kosongkan parameter ini.

Pilih Kolom Data

Service penentu lokasi membutuhkan kolom tertentu di data, seperti alamat, kota, negara, dan kode pos. Cocokkan kolom data di file input atau tabel dengan kolom yang diharapkan penentu lokasi dengan menggunakan kotak tarik-turun. Semakin banyak kolom yang Anda petakan, hasilnya akan semakin akurat. Jika semua data tabel input berada di satu kolom, pilih Kolom Tunggal dan pilih kolom data dari menu tarik-turun. Jika tidak, pilih Beberapa Kolom dan petakan kolom penentu lokasi yang sesuai dengan kolom dengan banyak bidang di tabel.

Pilih format output

Format hasil output. Jika Anda memilih untuk membuat sebuah feature service, alamat dari tabel input akan ditambahkan ke peta sebagai layer fitur titik. Opsi CSV dan XLS menambahkan file dengan hasil tugas geocoding ke Konten Saya ketika tugas selesai.

Nama hasil output

Ini adalah nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama layer input. Jika nama layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Dengan menggunakan Simpan hasil di menu drop-down, Anda dapat menentukan nama folder di Konten Saya sebagai tempat penyimpanan hasil.
