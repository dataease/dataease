Temukan Klaster Titik

Alat ini menemukan klaster fitur titik dalam kebisingan di sekitar berdasarkan distribusi spasialnya.

Jika Gunakan jangkauan peta saat ini dicentang, hanya fitur dalam layer titik input dalam jangkauan peta saat ini yang akan dianalisis. Jika tidak dicentang, semua fitur pada layer titik input akan dianalisis, meskipun berada di luar jangkauan peta saat ini.

Pilih layer yang memiliki klaster

Layer titik tempat klaster akan ditemukan.

Jumlah titik minimal untuk dapat dianggap sebagai klaster

Jumlah fitur minimum untuk dapat dianggap sebagai klaster. Klaster apa pun dengan fitur yang lebih sedikit daripada yang disediakan akan dianggap sebagai kebisingan.

Batasi rentang pencarian menjadi (opsional)

Jarak maksimum untuk mempertimbangkan fitur menjadi bagian dari klaster yang sama. Jika dibiarkan kosong, alat akan menjalankan HDBSCAN, yang menggunakan beragam jarak untuk memisahkan klaster beragam densitas dari derau tersebar. Jika nilai disediakan, alat akan menjalankan DBSCAN, yang menggunakan jarak tertentu untuk memisahkan klaster padat dari derau tersebar.

Nama layer hasil

Ini adalah nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama layer input. Jika nama layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Dengan menggunakan Simpan hasil di menu drop-down, Anda dapat menentukan nama folder di Konten Saya sebagai tempat penyimpanan hasil.
