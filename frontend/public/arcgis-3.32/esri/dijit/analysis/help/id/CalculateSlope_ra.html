Tentukan faktor penskalaan untuk elevasi

Masukkan faktor z.

Faktor z adalah faktor penskalaan yang digunakan untuk mengubah nilai elevasi untuk dua tujuan:

  • Untuk mengubah satuan elevasi (seperti meter atau kaki) ke satuan koordinasi horizontal dari layer, yang mungkin berupa kaki, meter, atau derajat
  • Untuk menambahkan pembesaran vertikal untuk efek visual

Nama layer hasil
