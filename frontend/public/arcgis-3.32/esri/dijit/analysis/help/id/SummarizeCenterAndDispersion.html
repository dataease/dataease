Ringkas Pusat dan Penyebaran

Alat ini menemukan fitur pusat dan distribusi arah. Alat ini dapat digunakan untuk menjawab pertanyaan seperti:

  • Di mana pusatnya?
  • Fitur mana yang paling dapat diakses dari semua fitur lainnya?
  • Seberapa tersebar, ringkas, atau terintegrasi fitur tersebut?
  • Apakah ada tren arah?

Jika Gunakan jangkauan peta saat ini dicentang, hanya fitur di layer input tersebut yang visibel dalam jangkauan peta saat ini yang akan dianalisis. Jika tidak dicentang, seluruh fitur pada layer input akan dianalisis, meskipun berada di luar jangkauan peta saat ini.

Pilih layer yang akan diuraikan

Layer titik, garis, atau area akan digunakan untuk menemukan fitur utama dan distribusi arah.

Jenis ringkasan

Metode untuk meringkas pusat dan penyebaran.

  • Fitur pusat - Mengidentifikasi fitur yang berada paling pusat di set data.
  • Pusat rata-rata - Mengidentifikasi pusat geografis (atau pusat konsentrasi) untuk serangkaian fitur.
  • Pusat median - Mengidentifikasi lokasi yang meminimalkan keseluruhan jarak Euclidean ke fitur set data.
  • Elips - Membuat elips deviasional standar untuk meringkas karakteristik spasial fitur geografis: kecenderungan pusat, penyebaran, dan tren arah. Elips dapat diberi ukuran sebagai simpangan baku 1, 2, atau 3.

Hitung bobot menurut (opsional)

Kolom ini digunakan untuk menghitung bobot lokasi menurut kepentingan relatifnya.

Kelompokkan menurut (opsional)

Kolom ini digunakan untuk mengelompokkan fitur untuk penghitungan terpisah dari fitur atau penyebaran pusat.

Nama layer hasil

Ini adalah nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama layer input. Jika nama layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Dengan menggunakan Simpan hasil di menu drop-down, Anda dapat menentukan nama folder di Konten Saya sebagai tempat penyimpanan hasil.
