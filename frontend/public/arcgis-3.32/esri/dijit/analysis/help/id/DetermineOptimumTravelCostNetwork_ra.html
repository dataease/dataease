Tentukan Jaringan Biaya Perjalanan Optimal

Alat ini menghitung jaringan biaya optimal dari serangkaian daerah input.

Satu contoh aplikasi alat ini adalah menemukan jaringan terbaik untuk kendaraan darurat.

Jika Gunakan jangkauan peta saat ini dicentang, hanya lokasi di layer input yang terlihat dalam jangkauan peta saat ini yang akan dianalisis. Jika tidak dicentang, seluruh lokasi pada layer input akan dianalisis, meskipun berada di luar jangkauan peta saat ini.

Pilih raster atau layer lokasi fitur untuk menentukan jalur perjalanan di antaranya

Wilayah input yang akan dihubungkan oleh jaringan dengan biaya paling sedikit.

Wilayah dapat ditentukan oleh layanan gambar atau feature service.

Untuk layanan gambar, wilayah ditentukan menurut kelompok sel yang berdekatan dengan nilai yang sama. Setiap wilayah harus diberi nomor secara unik. Sel yang bukan bagian dari wilayah apa pun harus berupa NoData. Jenis raster harus berupa bilangan bulat, dan nilai dapat berupa nilai positif atau negatif.

Untuk feature service, input dapat berupa titik, garis, atau poligon.

Pilih layer raster biaya perjalanan

Sebuah raster mendefinisikan impedansi atau biaya untuk bergerak secara planimetri melalui setiap sel.

Nilai di setiap lokasi sel mewakili jarak biaya-per-unit untuk bergerak melalui sel. Setiap nilai lokasi sel dikalikan dengan resolusi sel, sementara juga mengkompensasi gerakan diagonal untuk mendapatkan biaya total melewati sel.

Nilai-nilai dari raster biaya dapat berupa bilangan bulat atau bilangan titik mengambang, tetapi tidak boleh negatif atau nol (Anda tidak dapat memiliki biaya negatif atau nol).

Nama layer polyline jalur optimal hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan menggunakan kotak drop-down Simpan hasil di.

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan menggunakan kotak drop-down Simpan hasil di.
