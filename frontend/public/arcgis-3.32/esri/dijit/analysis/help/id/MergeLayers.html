Gabungkan Layer

Alat ini menyalin fitur dari dua layer ke dalam satu layer baru. Layer yang akan digabung semuanya harus berisi tipe fitur yang sama (titik, garis, atau area). Anda dapat mengontrol cara menggabung dan menyalin kolom dari layer input. Contoh:

  • Saya memiliki tiga layer untuk Inggris, Wales, dan Skotlandia dan saya menginginkan satu layer Britania Raya.
  • Saya memiliki 12 layer dan setiap layer berisi informasi persil untuk daerah kota yang berdampingan. Saya ingin menggabungkan semuanya menjadi satu layer, dengan mempertahankan hanya kolom yang memiliki nama dan tipe sama dari 12 layer input tersebut.

Jika Gunakan jangkauan peta saat ini dicentang, hanya fitur yang terlihat dalam jangkauan peta saat ini yang akan digabung. Jika tidak dicentang, semua fitur pada kedua layer akan digabung, meskipun fitur tersebut berada di luar jangkauan peta saat ini.

Pilih layer

Fitur titik, garis atau area yang akan digabung dengan layer yang ditentukan di bawah ini.

Pilih Layer untuk digabung

Pilih layer yang akan digabung dengan layer pertama.

Ubah kolom penggabungan

Secara default, semua kolom dari kedua input akan dibawa ke output. Anda dapat mengubah cara kolom dibawa ke output sebagai berikut:

  • Ubah nama-Kolom pada output akan diberi nama baru.
  • Hapus-Kolom akan dihapus dari output.
  • Cocokkan—Nama kolom akan disamakan. Nilai kolom dari layer gabungan akan disalin ke kolom output yang sesuai.

Jika kedua layer berisi kolom dengan nama dan jenis data yang sama, konten kedua kolom akan digabungkan menjadi satu kolom dengan nama tersebut. Ini tidak diterapkan untuk kolom di mana Cocokkan digunakan di parameter Modifikasi kolom gabungan.

Nama layer hasil

Ini adalah nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama layer input. Jika nama layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Dengan menggunakan Simpan hasil di menu drop-down, Anda dapat menentukan nama folder di Konten Saya sebagai tempat penyimpanan hasil.
