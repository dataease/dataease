Klasifikuoti pikselius naudojant gilųjį mokymąsi

Šis įrankis įvesties rastrui naudoja apmokytą giliojo mokymosi modelį, kad sukurtų klasifikuotą rastrą, ir kiekvienam tinkamam pikseliui priskiriama klasės žymė.

Jei parinktis Naudoti esamą žemėlapio aprėptį pažymėta, bus analizuojama tik dabartinėje žemėlapio aprėptyje matomas rastro plotas. Jei nepažymėta, bus analizuojamas visas rastras, net jei jis nepatenka į esamą žemėlapio aprėptį.

Pasirinkite, kuris vaizdas bus naudojamas klasifikuoti pikselius

Klasifikuotinas įvesties vaizdas.

Tai gali būti vaizdų paslaugos URL, rastrinis sluoksnis arba vaizdų paslaugos sluoksnis.

Pasirinkite, kokį gilaus mokymosi modelį naudoti klasifikuojant pikselius

Įvesties giliojo mokymosi paketo ( .dlpk) elementas.

Giliojo mokymosi paketą sudaro Esri modelio aprašo JSON failas ( .emd), giliojo mokymosi dvinario modelio failas ir (nebūtinai) naudojama Python rastrinė funkcija.

Nurodykite gilaus mokymosi modelio argumentus

Funkcijos argumentai apibrėžiami įvesties modelyje nurodytoje Python rastro funkcijos klasėje. Būtent čia nurodote papildomus giliojo mokymosi parametrus ir argumentus, skirtus eksperimentuoti ir tobulinti, pavyzdžiui, pasitikėjimo ribą jautrumo korekcijoms.

Argumentų pavadinimus įrankis išplatina skaitydamas Python modulį rastrų analizės serveryje.

Rezultatų sluoksnio pavadinimas

Sluoksnio, kuris bus sukurtas puslapyje Turinys ir įtrauktas į žemėlapį, pavadinimas. Numatytasis pavadinimas sukuriamas pagal įrankio ir įvesties sluoksnio pavadinimus. Jei toks sluoksnis jau egzistuoja, būsite paprašyti pateikti kitą pavadinimą.

Galite nurodyti aplanko Turinyje, kuriame rezultatai bus įrašyti naudojant iššokantįjį meniu Rezultatą įrašyti į, pavadinimą.
