Summera inom

Det här verktyget används för att hitta geoobjekt (och delar av geoobjekt) som ligger inom gränserna för områden i det första indatalagret. Exempel:

  • Med ett lager för avrinningsgränser och ett lager med markanvändningsgränser enligt markanvändningstyp kan du beräkna den totala arealen för de olika markanvändningstyperna för varje avrinningsområde.
  • Med ett lager för fastigheter i ett län och ett lager kommungränser kan du summera snittvärdet för de tomma fastigheterna inom varje kommungräns.

Om Använd aktuell kartutbredning markeras, analyseras bara de geoobjekt i indatalagret och lagret som ska summeras som visas i den aktuella kartutbredningen. Om alternativet inte är markerat, analyseras alla geoobjekt i både indatalagret och lagret som ska summeras, även om de ligger utanför den aktuella kartutbredningen.

Välj områdeslager där andra geoobjekt ska summeras inom dess gränser

Det områdeslager som ska användas för att summera geoobjekt inom dess gränser.

Du kan välja mellan ett områdeslager som du anger eller lagringsplatser som genereras när verktyget körs. När lagringsplatser skapas avgör siffran och enheterna som angivits för Kvadrat kvadratens höjd och längd. För Hexagon avgör siffran och enheterna som angivits avståndet mellan parallella sidor.

Förutom att välja ett lager på din karta kan du välja Välj analyslager, som du hittar längst ned i listrutan. Det alternativet öppnar ett galleri där du kan bläddra till och välja datauppsättningar från diverse källor som du kan använda i din analys.

Lägg till statistik från lagret som ska summeras

Alternativt kan du beräkna statistik för numeriska attribut hos punkter som hamnar inom geoobjektens gränser.

Om du markerar kryssrutan Antal punkter beräknas det totala antalet punkter inom varje områdesgräns. Dessutom kan du beräkna statistik för numeriska attribut hos de punkter som ligger inom varje områdesgräns.

  • Summa – Totalvärdet för det angivna fältet.
  • Genomsnitt – Snittvärde eller medelvärde för det angivna fältet.
  • Minimum – Det lägsta värdet för det angivna fältet.
  • Maximum – Det högsta värdet för det angivna fältet.
  • Std. avvikelse – Standardavvikelsen för värdena i det angivna fältet.

Om lagret som ska summeras inte har några nummerfält kommer ingen ytterligare statistik att vara tillgänglig.

Välj fält att gruppera efter

Det här är ett attribut för funktionen Summera geoobjekt som du kan använda för att beräkna statistik separat för varje unikt attributvärde. Anta till exempel att det första indatalagret innehåller kommungränser och att summeringsgeoobjekten är fastigheter. Ett av attributen för fastigheterna är Status, som innehåller två värden: LEDIG och UPPTAGEN. Om du vill beräkna den totala ytan för lediga och upptagna fastigheter inom kommungränserna använder du Status som attribut för grupperingen. Statistiken för varje grupp visas i resultatlagrets popupfönster tillsammans med antalet geoobjekt inom varje områdesgräns.

Lägg till minoritet, majoritet

Den här kryssrutan aktiveras när du väljer ett fält att gruppera efter. Om du vill ta reda på vilket attributvärde inom varje grupp som är i minoritet (minst dominant) eller majoritet (mest dominant) inom varje gräns i det första indatalagret väljer du Lägg till minoritet, majoritet. När du väljer Lägg till minoritet, majoritet läggs två nya fält till i resultatlagret. Om du även väljer Lägg till procenttal läggs två ytterligare fält till i resultatlagret som innehåller procenttalen för minoritetens och majoritetens attributvärden inom varje grupp.

  • Om du summerar punkter beräknas minoritetens och majoritetens värden baserat på antalet punkter inom varje begränsningsområde i indataområdeslagret.
  • Om du summerar linjer beräknas minoritetens och majoritetens värden baserat på längden för linjerna inom varje begränsningsområde i indataområdeslagret.
  • Om du summerar områden beräknas minoritetens och majoritetens värden baserat på områdena inom varje begränsningsområde i indataområdeslagret.

Lägg till procenttal

Den här kryssrutan aktiveras när du väljer ett fält att gruppera efter. Välj Lägg till procenttal om du vill ta reda på procenttalet för varje attributvärde inom varje grupp. Ett nytt fält läggs till i resultattabellen som innehåller procenttalet för varje attributvärde inom varje grupp. Om du även markerar Lägg till minoritet, majoritet läggs två ytterligare fält till i resultatlagret som innehåller procenttalen för minoritetens och majoritetens attributvärden inom varje grupp.

Resultatlagrets namn

Det här är namnet på lagret som skapas i Mitt innehåll och läggs till i kartan. Standardnamnet baseras på verktygets namn och indatalagrets namn. Om lagernamnet redan finns uppmanas du ange ett annat namn.

Om du använder listrutan Spara resultat i kan du ange namnet på en mapp i Mitt innehåll där resultatet ska sparas.
