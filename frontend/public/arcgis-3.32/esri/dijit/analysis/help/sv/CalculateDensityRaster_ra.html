Beräkna densitet

Det här verktyget skapar en densitetskarta av punkt- eller linjegeoobjekt genom att sprida ut kända antal av något fenomen (representerade som attribut för punkterna eller linjerna) över kartan. Resultatet blir ett lager med områden klassificerade från lägsta till högsta densitet.

För punktdata bör varje punkt representera platsen för en händelse eller incident, medan resultatlagret representerar antalet incidenter per enhetsområde. Ett större densitetsvärde på en ny plats innebär att det finns fler punkter nära den platsen. I många fall kan resultatlagret tolkas som ett riskområde för framtida händelser. Om indatapunkterna till exempel representerar platser för blixtnedslag, kan resultatlagret tolkas som ett riskområde för framtida blixtnedslag.

För linjedata representerar linjetäthetsområdet det totala antal linjer som finns nära varje plats. Enheterna för de beräknade densitetsvärdena är längden på linjerna per enhetsområde. Om linjerna till exempel representerar floder, representerar resultatlagret den totala längden för de floder som ligger inom sökradien. Det här resultatet kan användas för att identifiera områden som är gynnsamma för betesdjur.

Andra exempel på användningsområden för det här verktyget är bland annat:

  • Skapa kartor över brottslighet som hjälper polisen att fördela sina resurser till områden med hög brottslighet.
  • Beräkna densitet för sjukhus inom en kommun. Resultatlagret visar områden med hög och låg tillgång till sjukhus, och den här informationen kan användas för att avgöra var nya sjukhus bör byggas.
  • Identifiera områden där risken för skogsbränder är stor baserat på platsen för tidigare skogsbränder.
  • Hitta orter som ligger långt från större motorvägar för att kunna planera var nya vägar bör byggas.

Använd ett antalsfält

Ange ett fält som visar antalet incidenter på respektive plats. Om du till exempel har punkter som representerar städer, kan du använda ett fält som representerar befolkningen i staden som ett antalsfält. Då räknar det resulterande befolkningstäthetslagret med större befolkningstäthet nära städer med större befolkning.

Om standardalternativet Inget används, anses varje plats representera ett enda antal.

Sök avstånd

Ange ett avstånd som visar hur långt du vill söka för att hitta punkt- eller linjegeoobjekt när du beräknar densitetsvärden. Om du till exempel anger ett sökavstånd på 10 000 meter, beräknas densiteten för alla platser i utdatalagret baserat på geoobjekt som ligger inom 10 000 meter från platsen. Alla platser som inte har några incidenter inom 10 000 meter får densitetsvärdet noll.

Om inget avstånd anges kommer att standardvärde att beräknas som bygger på platserna för indatageoobjekten och värdena i antalsfältet (om ett antalsfält anges).

Areaenheter för utdata

Ange areaenheten för utdata. Densiteten är antalet delat med området, och den här parametern anger areaenheten i densitetsberäkningen.

De tillgängliga areaenheterna är engelska kvadratmil, kvadratfot, kvadratmeter och kvadratkilometer.

Resultatlagrets namn

Namnet på lagret som skapas i Mitt innehåll och läggs till i kartan. Standardnamnet baseras på verktygets namn och indatalagrets namn. Om lagret redan finns uppmanas du ange ett annat namn.

Du kan ange namnet på en mapp i Mitt innehåll där resultatet sparas med listrutan Spara resultat i.
