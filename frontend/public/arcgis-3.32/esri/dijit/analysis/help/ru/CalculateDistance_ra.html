Вычислить расстояние

Этот инструмент вычисляет Евклидово расстояние, направление и размещение на основании одного или нескольких источников.

Примеры задач, в которых необходимо вычислять расстояния:

  • Каково расстояние до ближайшего города?
  • Каково расстояние до ближайшей дороги?
  • Каково расстояние до ближайшего источника водоснабжения?

Если опция Использовать текущий экстент карты включена, будут анализироваться только объекты входного слоя, которые видны в момент анализа в текущем экстенте. Если опция отключена, анализ будет выполнен для всех объектов входного слоя, даже если они находятся вне текущего экстента карты.

Выбор растра слоя расположения объектов для вычисления расстояний

Слой, который определяет, к чему будут вычисляться расстояния. Это может быть сервис изображений или сервис объектов.

Для сервиса изображений тип входных данных может быть целым или с плавающей запятой.

Для сервиса объектов входными данными могут быть точки, линии или полигоны.

Максимальное расстояние для расчета

Максимально возможное рассчитываемое расстояние. Например, если будет указано значение 10 000м, то Евклидово расстояние будет рассчитано от исходных объектов в пределах этого значения (расстояния).

Если это значение не будет указано, то вычисления будут производиться в пределах указанного экстента.

В качестве единиц измерения можно использовать километры, метры, мили и футы.

Размер выходной ячейки

Размер ячеек и единицы измерения для выходных растров.

Единицами измерения могут быть километры, метры, мили или футы.

Имя слоя результата расстояний

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

Вы можете указать имя папки в Моих ресурсах, где будет сохранен результат, с помощью ниспадающего списка Сохранить результат в.

Имя слоя результата направлений

Это имя слоя, который будет создан в Моих ресурсах. Необходимо указать имя, которое до сих пор не использовалось в Моих ресурсах.

Вы можете указать имя папки в Моих ресурсах, где будет сохранен результат, с помощью ниспадающего списка Сохранить результат в.

Имя слоя результата распределения

Это имя слоя, который будет создан в Моих ресурсах. Необходимо указать имя, которое до сих пор не использовалось в Моих ресурсах.

Вы можете указать имя папки в Моих ресурсах, где будет сохранен результат, с помощью ниспадающего списка Сохранить результат в.

Выберите поле распределения

Поле в исходных данных, в котором хранятся значения, определяющие каждый источник. Это может быть целочисленное или строковое поле исходного набора данных.

По умолчанию – Value. Если используется сервис объектов и нет доступного поля Value, будет использоваться первое доступное целочисленное или текстовое поле. Если доступных полей нет, используется поле ObjectID (например, OID или FID, в зависимости от типа входных данных объекта).
