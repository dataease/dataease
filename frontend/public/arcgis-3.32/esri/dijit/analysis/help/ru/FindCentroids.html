Найти центроиды

Этот инструмент работает со слоем мультиточек, линейными и площадными объектами для определения репрезентативного центра каждого объекта.

Если опция Использовать текущий экстент карты включена, будут анализироваться только объекты входного линейного, площадного и мультиточечного слоев, которые отображаются в момент анализа в текущем экстенте. Если опция отключена, анализ будет выполнен для всех объектов входного слоя, даже если они находятся вне текущего экстента карты.

Выбрать слой, объекты которого будут использоваться для построения центроидов

Для нахождения центроидов будут использоваться мультиточечные, линейные и полигональные слои.

Выбирая слой для вашей карты, вы можете также щелкнуть Выбрать слои анализа в нижней части ниспадающего списка. Эта опция открывает галерею, где вы можете найти и выбрать наборы данных из разных источников для своего анализа.

Показать выходные местоположения

Если отмечена опция В истинном центроиде (по умолчанию), в выходных данных будет истинное местоположение центроида для каждого входного объекта. Центроиды должны располагаться в соответствии со следующим:

  • Мультиточечные объекты – выходной центроид будет в средней точке, вычисленной на основе координат x и y всех точек мультиточечного объекта.
  • Линейные объекты – выходной центроид будет в взвешенных средних координатах x и y для всех линейных сегментов линейного объекта. Вес определяется на основе длины сегмента.
  • Площадные объекты – выходной центроид будет располагаться в центре площадного объекта.

Если установлена опция внутри входных объектов, выходные точечные объекты будут ограничены или находиться внутри каждого входного объекта. Полученные в результате точечные объекты будут выглядеть следующим образом:

  • Мультиточечные объекты – выходной центроид будет совмещается с одной из точек в группе мультиточек.
  • Линейные объекты – выходной центроид будет находиться на линии. Если линия имеет изгибы, выходная точка будет в средней точке линии.
  • Площадные объекты – выходная точка будет внутри площадного объекта.

Имя слоя результата

Это имя слоя, который будет создан в разделе Мои ресурсы и добавлен на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
