Создать водоразделы

Этот инструмент определяет водораздел, водосборную область, для каждой точки входного слоя.

Например, предположим, у вас есть точечные объекты водных загрязнений, и вы хотите определить наиболее вероятные источники загрязнения. Поскольку источник загрязнения должен быть где-то выше по течению в пределах водосборной области, используйте этот инструмент для определения водосборных областей, содержащих источники загрязнения.

Расположение входных точек имеет важное значение. Если они расположены не точно на линии стока, полученные водосборные области могут оказаться малы и непригодны для анализа. Можно использовать параметр Расстояние поиска до ближайшего водостока, чтобы настроить расположение входных точек точно на линии водостока и определить водосборную область, из которой вода поступает в точку на линии водостока. Настроенные местоположения, используемые для поиска водоразделов, записываются в выходной слой – входные точки не изменяются.

В результате получается составной слой, содержащий два слоя: водоразделы и настроенные точки для вычисления водоразделов.

Полученные водоразделы будут иметь все атрибуты входных точек, а также дополнительные:

  • PourPtID – содержит ID из входной точки
  • Area Square Kilometers – содержит площадь водораздела в квадратных километрах
  • Description и DataResolution – содержит сведения, касающиеся гидрологических данных, используемых для вычисления

Если опция Использовать текущий экстент карты включена, для вычисления водоразделов будут использоваться только те объекты, которые находятся в текущем экстенте. Если опция отключена, анализ будет выполнен для всех объектов входного слоя, даже если они находятся вне текущего экстента карты.

Точечные объекты, используемые для вычисления водоразделов

Инструмент Водораздел будет подсчитывать каждую точку.

Выбирая слой для вашей карты, вы можете также щелкнуть Выбрать слои анализа в нижней части ниспадающего списка. Эта опция открывает галерею, где вы можете найти и выбрать наборы данных из разных источников для своего анализа.

Расстояние поиска до ближайшего водостока

Если входные точки расположены в стороне от линии стока, полученные водосборные области скорее всего будут малы и слабо пригодны для определения расположенного выше источника загрязнения. В большинстве случаев желательно, чтобы входные точки замыкались на ближайшей линии водостока для определения водосборных областей, протекающих через точку на линии водостока. Чтобы найти ближайшую линию стока, задайте расстояние поиска. Если его не задать, инструмент вычислит и использует умеренное расстояние поиска.

Чтобы использовать точное положение входной точки, задайте расстояние поиска, равное нулю.

Для анализа линии водостоков предварительно вычислены в Esri с использованием стандартных гидрологических моделей. Если на указанном расстоянии линий водостоков не найдено, будет использовано местоположение с наибольшим суммарным стоком в пределах заданного расстояния.

Имя слоя результата

Это имя слоя, который будет создан в разделе Мои ресурсы и добавлен на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.

Запуск этого инструмента не использует кредиты.
