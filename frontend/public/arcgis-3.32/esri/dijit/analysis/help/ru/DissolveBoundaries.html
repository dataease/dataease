Слияние границ

Перекрывающиеся области или области с общей границей соединяются вместе, формируя единую область.

Вы можете указать, какие границы будут объединены, используя атрибутивное поле. Например, если у вас есть слой округов, и каждый округ имеет атрибут State_Name, вы можете убрать границы, используя этот атрибут State_Name Смежные округа будут объединены, если значение State_Name совпадает. В результате вы получите слой с границами штатов.

Выберите площадной слой, в котором необходимо выполнить слияние границ

Слой, содержащий полигональные объекты для слияния.

Выбирая слой для вашей карты, вы можете также щелкнуть Выбрать слои анализа в нижней части ниспадающего списка. Эта опция открывает галерею, где вы можете найти и выбрать наборы данных из разных источников для своего анализа.

Выберите метод слияния

Вы можете выбрать один из двух методов слияния областей:

  • Перекрывающиеся или смежные области – две или больше областей, границы которых перекрываются, будут слиты в одну. Кроме того, смежные области (т.е. с общей границей) будут слиты в одну.
  • Области с одинаковым значением в атрибутивном поле – перекрывающиеся области или области с общей границей будут слиты в одну, если они имеют одинаковое значение в атрибутивном поле.

    Например, если у вас есть слой округов, и каждый округ имеет поле State_Name, вы можете убрать границы, используя поле State_Name. Смежные округа будут объединены, если значение State_Name совпадает. В результате вы получите слой с границами штатов.

Если опция Использовать текущий экстент карты включена, будут анализироваться только объекты входного слоя, которые видны в момент анализа в текущем экстенте. Если опция отключена, слияние будет выполнено для всех объектов входного слоя, даже если они находятся вне текущего экстента карты.

Создание составных объектов

В ArcGIS это цифровое представление объекта или места, в котором может быть несколько фрагментов или отдельных частей, но определяемое как единый пространственный объект, так как ссылается на один набор атрибутивных значений. В классе, содержащем штаты США, например, штат Гавайи может рассматриваться, как один полигон из нескольких частей, поскольку его отдельные геометрические части классифицируются, как один штат. С помощью булева параметра можно выбрать одну из двух опций:

  • Истина (отмечено): выходной сервис может содержать составные объекты.
  • Ложь (не отмечено): выходной сервис может содержать только отдельные объекты, и для каждой части будет создан отдельный объект.

Добавление статистики

При слиянии областей можно сложить их атрибуты. Например, если вы убираете границы между округами на базе State_Name, и каждый округ имеет поле Population, можно вычислить сумму для Population и определить численность населения штата.

Доступные типы статистики:

  • Сумма – вычисляет суммарное значение атрибута в заданном поле.
  • Среднее – вычисляет среднее арифметическое указанного поля.
  • Минимум – находит наименьшее значение для всех записей указанного атрибута.
  • Максимум – находит наибольшее значение для всех записей указанного атрибута.
  • Средне-квадратическое отклонение – находит средне-квадратическое отклонение для значений в указанном поле

Имя слоя результата

Это имя слоя, который будет создан в разделе Мои ресурсы и добавлен на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
