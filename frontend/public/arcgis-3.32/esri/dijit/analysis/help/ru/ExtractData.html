Извлечь данные

С помощью этого инструмента можно выбрать и загрузить данные для указанной области интереса. Выбранные слои будут добавлены в файлы форматов ZIP, CSV или KMZ.

Слои для извлечения

Это слои в таблице содержания, из которых можно извлечь данные. Отмеченные слои извлекаются и добавляются в итоговый файл ZIP или CSV.

Чтобы слой был виден в списке слоев для извлечения данных, вы должны быть владельцем данных, или владелец данных должен разрешить экспортировать свои данные. Чтобы ваши данные можно было экспортировать, выполните следующие шаги:

  • Откройте диалоговое окно Описание элемента.
  • Перейдите на вкладку Настройки.
  • Прокрутите вниз до раздела Экспорт данных.
  • Отметьте Разрешить пользователям экспорт в другие форматы. По окончании щелкните Сохранить.

Если слой анализа содержит вложения, они будут извлекаться, только если выходным форматом данных является база геоданных, и опция Вырезать объекты не отмечена.

Область интереса

Область, определяющая, какие объекты будут включаться в выходной файл ZIP или CSV. Имеются следующие варианты:

  • Текущий видимый экстент карты ( Как у отображаемой области).
  • Один из слоев в таблице содержания. Если выбранный слой содержит точечные или линейные объекты, экстент слоя будет использоваться как изучаемая область.
  • Обведите область, чтобы задать область изучения.
Все объекты, находящиеся целиком внутри экстента карты, будут извлечены. Для объектов, которые находятся на границе экстента карты или пересекаются с ней, но не полностью попадают в экстент, есть два способа обработки:
  • Выбрать объекты – извлекаются все объекты, пересекающие границу изучаемой области.
  • Вырезать объекты – извлекаются только части объектов, расположенные в пределах изучаемой области.

Формат выходных данных

Формат извлекаемых данных. При экспорте линейных или площадных объектов в файлы CSV, извлекаются только непространственные атрибуты. При экспорте точечных объектов, извлекаются и пространственные и непространственные атрибуты.

Извлеченные данные будут в проекции Web Mercator 1984. Для файлов CSV, к данным будут добавлены поля Lat и Long с координатами в проекции Web Mercator 1984.

Поддерживаются следующие выходные форматы:

  • Значения, разделенные запятыми (CSV) ­ .csv или .zip
  • Файловая база геоданных - .zip
  • Язык разметки Keyhole Markup Language (KML) - .kmz или .zip
  • Шейп-файл - .zip

Имя выходного файла

Это имя элемента, который будет добавлен в Мои ресурсы. Имя элемента зависит от выбранного Формата выходных данных. Если имя элемента уже используется, появится запрос ввести новое имя.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
