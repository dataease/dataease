Конвертировать объекты в растр

Этот инструмент конвертирует объекты в набор растровых данных.

Любой класс пространственных объектов, содержащий точки, линии или полигоны, может быть конвертирован в набор растровых данных.

Центр ячейки используется для определения значения выходного пиксела растра.

Входной тип поля определяет тип выходного растра. Если поле целочисленное, выходной растр будет целочисленным; если оно представлено значениями с плавающей точкой, выходные данные будут представлены значениями с плавающей точкой.

Выберите векторный слой для конвертации

Входной векторный слой, который будет конвертирован в набор растровых данных.

Выбрать поле

Выберите поле, которое будет использоваться для присвоения значений выходному растру.

Размер выходной ячейки

Размер ячеек и единицы измерения для выходных растров.

Доступные единицы измерения - мили, футы, километры и метры. По умолчанию - футы.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

Вы можете указать имя папки в Моих ресурсах, где будет сохранен результат, с помощью ниспадающего списка Сохранить результат в.
