Найти существующие местоположения

Этот инструмент выбирает в изучаемой области существующие объекты, соответствующие набору критериев. Эти критерии могут основываться на атрибутивных запросах (например, свободные земельные участки) и пространственных запросах (например, на расстоянии одной мили от реки).

Если опция Использовать текущий экстент карты включена, будут анализироваться только объекты входного слоя, которые видны в момент анализа в текущем экстенте. Если опция отключена, будут обрабатываться все объекты обоих слоев, даже если они находятся вне текущего экстента карты.

Выберите слой, содержащий объекты, по которым вы хотите выполнить поиск с использованием атрибутивного или пространственного запроса.

Точечный, линейный или площадной слой, в котором вы хотите выполнить поиск объектов с использованием комбинации пространственных и атрибутивных запросов, приведенных далее.

Выбирая слой для вашей карты, вы можете также щелкнуть Выбрать слои анализа в нижней части ниспадающего списка. Эта опция открывает галерею, где вы можете найти и выбрать наборы данных из разных источников для своего анализа.

Построение запроса для поиска объектов

Щелкните кнопку Добавить выражение, чтобы создать выражение. Вы можете добавить несколько выражений. После ввода выражения и щелчка на кнопке Запустить анализ объекты входного слоя будут фильтроваться по каждому выражению. Те объекты, которые удовлетворяют всем выражениям, записываются в итоговый слой.

Существует два основных типа выражений – атрибутивные и пространственные. Атрибутивные выражения обращаются к значениям полей, например, где есть слово 'San Francisco'. Существует четыре типа пространственных выражений:

  • в пределах расстояния: объект должен быть в пределах заданного расстояния от объекта из другого слоя. Например, можно найти земельные участки, находящиеся не далее 500 метров от реки Лос-Анджелес.
  • полностью содержит: объект должен полностью содержать объект из другого слоя. Например, поиск водосборной области, в которой целиком помещаются земельные участки, предназначенные для промышленного использования. Ни одна часть участка не может быть вне водосборной области.
  • полностью внутри: объект должен быть полностью окружен объектом из другого слоя. Например, можно найти школьные округа, которые целиком входят в границы города. Ни одна часть школьного округа не может располагаться вне границ города.
  • пересекает: объект содержит часть объекта из другого слоя. Например, можно найти земельные участки, пересекающиеся с реками. Любая часть реки, которая проходит по участку или граничит с участком, пересекается с ним.
  • ближайший к: объект должен быть ближайшим к объекту из другого слоя. Например, поиск ресторана, который является ближайшим к каждому отелю.

Каждое пространственное выражение имеет соответствующее отрицание; за исключением опций ближайший к, вне пределов расстояния, не полностью содержится в, не полностью внутри и не пересекает.

Имя слоя результата

Это имя слоя, который будет создан в разделе Мои ресурсы и добавлен на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
