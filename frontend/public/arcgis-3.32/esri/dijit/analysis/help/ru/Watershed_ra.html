Водосборная область

Инструмент геообработки, который определяет область, расположенную выше набора ячеек растра.

Водораздел – это область вверх по склону, которая распределяет сток – особенно воду – в общую «точку слива» в качестве концентрированного дренажа. Он может быть частью большего водораздела, а также может содержать меньшие водоразделы, называемые подбассейны.

Значение для каждой водосборной области будет определено, исходя из источника во входном растре или векторных данных по точкам устьев. Когда точки устьев представлены в виде набора растровых данных, по умолчанию будут использованы значения ячеек. Когда точки устьев даны в виде набора точечных данных, значения будут взяты из заданного пользователем поля.

Инструмент Водосборная область поддерживает только входной растровый слой направлений стоковD8.

При указании входных положений точек устьев в качестве данных объектов полем по умолчанию будет первое доступное и допустимое поле. Если допустимых полей нет, по умолчанию используется поле ObjectID (например, OID или FID).

Выбор слоя направления стока

Входной растр, показывающий направление стока для каждой ячейки.

Растр направления стока можно вычислить с помощью инструмента Направление стока или задачи FlowDirection в сервисе RasterAnalysisTools, запущенного с типом направления стока по умолчанию D8.

Указание входных положений точек устьев

Входные положения точек устьев.

Для растра этот параметр представляет ячейки, выше которых будет определяться водосборная область. В качестве входных ячеек будут использоваться все ячейки, значение которых отличается от значения NoData. Для набора данных точечных объектов этот параметр представляет местоположения, для которых будет определяться расположенная выше них водосборная область.

Выбор поля точек устьев

Поле, используемое для присвоения значений местоположениям точек устьев.

Если входной слой точек слоя является растром, то используется Value. Если же это объекты, то используется числовое поле. Если поле содержит значения с плавающей точкой, они будут округлены до целых чисел.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

Вы можете указать имя папки в Моих ресурсах, где будет сохранен результат, с помощью ниспадающего списка Сохранить результат в.
