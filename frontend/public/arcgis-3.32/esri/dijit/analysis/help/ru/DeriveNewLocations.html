Получить новые местоположения

Этот инструмент создает новые объекты в изучаемой области, которые соответствуют набору критериев. Эти критерии могут основываться на атрибутивных запросах (например, свободные земельные участки) и пространственных запросах (например, участки, находящиеся в зонах затопления).

Если опция Использовать текущий экстент карты включена, будут анализироваться только те объекты, которые отображаются в текущем экстенте. Если опция отключена, будут обрабатываться все объекты обоих слоев, даже если они находятся вне текущего экстента карты.

Получить новые местоположения, которые удовлетворяют следующему выражению

Щелкните кнопку Добавить выражение, чтобы создать выражение. Вы можете добавить несколько выражений. После ввода выражения и щелчка на кнопке Запустить анализ объекты на слое анализа будут фильтроваться по каждому выражению. Объекты, которые удовлетворяют всем выражениям, записываются в итоговый слой.

Существует два основных типа выражений – атрибутивные и пространственные. Атрибутивные выражения обращаются к значениям полей, например, где есть слово 'San Francisco'. Существует четыре типа пространственных выражений:

  • в пределах расстояния: объекты, находящиеся в пределах заданного расстояния от объекта из другого слоя. Например, можно найти объекты, находящиеся не далее 500 метров от реки Лос-Анджелес.
  • полностью содержит: объект должен полностью содержать объект из другого слоя. Например, поиск водосборной области, в которой целиком помещаются земельные участки, предназначенные для промышленного использования. Ни одна часть участка не может быть вне водосборной области.
  • полностью внутри: объект должен быть полностью окружен объектом из другого слоя. Например, можно найти школьные округа, которые целиком входят в границы города. Ни одна часть школьного округа не может располагаться вне границ города.
  • пересекает: объект или части объекта, должны перекрывать или пересекать объект из другого слоя. Например, найти области в границах национального парка, в которых в результате множественных пожаров уничтожена растительность. Области уничтоженной растительности за пределами национального парка не включаются в поиск.
  • ближайший к: объект должен быть ближайшим к объекту из другого слоя. Например, поиск зеленой зоны, которая является ближайшей к каждой школе.

Каждое пространственное выражение имеет соответствующее отрицание; за исключением опций ближайший к, вне пределов расстояния, не полностью содержится в, не полностью внутри и не пересекает.

Имя слоя результата

Это имя слоя, который будет создан в разделе Мои ресурсы и добавлен на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
