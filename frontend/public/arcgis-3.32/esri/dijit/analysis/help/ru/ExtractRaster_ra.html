Извлечь растр

Этот инструмент вырезает растр по границе, заданной либо как прямоугольная область, либо в соответствии с фигурой, обведенной на экране. Вы можете вырезать растр по текущему экстенту или по области изучения, заданной полигоном.

Входной слой для вырезания.

Задать вырезающую геометрию

Выберите тип экстента, использующегося при вырезании геометрии слоя.

  • Использовать текущий экстент – для вырезания слоя используется текущий экстент карты. Используется по умолчанию.
  • Выберите область изучения – позволяет выбрать область в пределах экстента геометрии или вне ее. Если выбрана опция Данные внутри геометрии изображение вне экстентов будет удалено. Если выбрана опция Данные вне геометрии изображение внутри экстентов будет удалено. Также можно задать вырезающую геометрию с помощью инструмента рисования.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

Вы можете указать имя папки в Моих ресурсах, где будет сохранен результат, с помощью ниспадающего списка Сохранить результат в.
