Слияние слоев

Этот инструмент копирует объекты из двух слоев в новый слой. Объединяемые слои должны содержать объекты одного типа (полигоны, линии или точки). Способом объединения и копирования полей можно управлять. Пример:

  • У вас имеется три слоя: Англия, Уэльс и Шотландия, а вы хотите создать один слой - Великобритания.
  • У вас есть 12 слоев, каждый из которых содержит информацию о земельных участках в смежных районах. Я хочу объединить их в один слой, сохранив только поля с одинаковыми названиями и типами из всех 12 слоев.

Если опция Использовать текущий экстент карты включена, объединяются только те объекты, которые отображаются в текущем экстенте. Если опция отключена, будут объединены все объекты обоих слоев, даже если они находятся вне текущего экстента карты.

Обновить объединенные поля

По умолчанию, все поля обоих слоев переносятся в выходные данные. Вы можете управлять способом переноса полей, используя следующие опции:

  • Переименовать – поле в выходных данных будет переименовано.
  • Удалить – поле будет удалено из выходных данных.
  • Сопоставить – имена полей приведены в соответствие друг другу. Значения поля из слоя для объединения будут скопированы в соответствующие выходные поля.

Если оба слоя содержат поле с идентичным именем и типом данных, содержимое этих полей будет объединено в одно поле с тем же именем. Это не применимо к полям, где используется Сопоставление в параметре Обновить объединенные поля.

Имя слоя результата

Это имя слоя, который будет создан в разделе Мои ресурсы и добавлен на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
