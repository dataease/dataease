Суммированный центр и дисперсия

Этот инструмент находит центральные объекты и пространственное распределение. Это может помочь вам ответить на вопросы, такие как:

  • Где находится центр?
  • Какой объект является наиболее доступным для всех остальных?
  • Насколько разбросаны, компактно расположены или интегрированы пространственные объекты?
  • Имеются ли тренды направления?

Если опция Использовать текущий экстент карты включена, анализ будет выполнен только для тех объектов входного слоя, которые видны в момент анализа в текущем экстенте. Если опция отключена, анализ будет выполнен для всех объектов входного слоя, даже если они находятся вне текущего экстента карты.

Выбрать слой для описания

Точечный, линейный или площадной слой, который будет использоваться для поиска центральных объектов и пространственных распределений.

Типы суммирования

Метод суммирования центра и дисперсии.

  • Центральный объект – определяет наиболее центральной расположенный объект в наборе данных.
  • Средний центр – определяет географический центр (или центр концентрации) для набора объектов.
  • Медианный центр – идентифицирует местоположение, минимизирующее среднее Евклидово расстояние до пространственных объектов в наборе данных.
  • Эллипс – создает эллипс стандартного отклонения для обобщения пространственных характеристик географических объектов: центральная тенденция, дисперсность и тренды, обладающие направленностью. Эллипсы могут быть размером в 1, 2 или 3 стандартных отклонения.

Вес по (дополнительно)

Числовое поле, используемое для взвешивания местоположений согласно их относительной важности.

Группировать по (дополнительно)

Поле, используемое для группировки объектов для отдельных расчетов центрального объекта или дисперсии.

Имя слоя результата

Это имя слоя, который будет создан в разделе Мои ресурсы и добавлен на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
