Конвертировать растр в объекты

Этот инструмент конвертирует растр в векторный набор данных с точечными, линейными и полигональными объектами.

Параметр Выберите поле позволяет выбирать, какое атрибутивное поле входного набора растровых данных станет атрибутом в выходном классе объектов. Этот параметр доступен только для целочисленных растров, содержащих дополнительные атрибуты в связанной таблице.

Входной растр может иметь ячейки любого размера.

Целочисленные растры могут конвертироваться в точечные, линейные или полигональные объекты, однако растры с плавающей точкой могут конвертироваться только в точечные объекты.

Если включена опция Использовать текущий экстент карты, конвертируются только ячейки растра, которые отображаются в текущем экстенте карты. Если опция отключена, будут конвертироваться все ячейки растра, даже если они находятся вне текущего экстента карты. Помните об этом при использовании большого растрового слоя или слоя с высоким разрешением.

Этот инструмент используется в следующих случаях:

  • Создание полигональных объектов из растра классификации землепользования.
  • Потоки, найденные в ЦМР, могут конвертироваться в линейные объекты.
  • ЦМР может конвертироваться в точечные объекты, которые можно применять в рабочих процессах, не требующих использования растров.

Выберите растровый слой для конвертации

Входной растровый слой, который будет конвертирован в слои объектов.

Выбрать поле

Укажите поле со значением, которое будет использоваться для конвертации.

Это может быть любое целочисленное или текстовое значение.

Поле, содержащее значения с плавающей точкой, могут использоваться, только если выходной набор данных является точечным.

По умолчанию используется поле Value, содержащее значение каждой ячейки растра.

Выберите тип выходных данных

Выберите тип выходных данных: точка, линия или полигон.

  • Точка –– растр будет конвертирован в набор точечных данных.
  • Линия – растр будет конвертирован в набор линейных данных.
  • Полигон – растр будет конвертирован в набор полигональных данных.

Упростить линии или полигоны

Укажите, следует ли упрощать объекты.

Если опция включена (по умолчанию), линейные или полигональные объекты будут сглажены, полученный результат будет более обобщенным. Если опция отключена, линейные объекты будут соответствовать границам ячеек набора растровых данных.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

Вы можете указать имя папки в Моих ресурсах, где будет сохранен результат, с помощью ниспадающего списка Сохранить результат в.
