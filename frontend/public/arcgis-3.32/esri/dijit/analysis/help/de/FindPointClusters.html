Punkt-Cluster suchen

Dieses Werkzeug findet Cluster aus Punkt-Features im Datenrauschen des Umfelds basierend auf deren räumlicher Verteilung.

Wenn die Option Aktuelle Kartenausdehnung verwenden aktiviert ist, werden nur die Features im Eingabe-Punkt-Layer analysiert, die sich in der aktuellen Kartenausdehnung befinden. Wenn diese Option deaktiviert ist, werden alle Features im Eingabe-Punkt-Layer analysiert, auch diejenigen, die sich außerhalb der aktuellen Kartenausdehnung befinden.

Wählen Sie den Layer aus, für den nach Clustern gesucht wird.

Der Punkt-Layer, in dem nach Clustern gesucht wird.

Minimale Anzahl von Punkten, die als Cluster gelten

Die minimale Anzahl an Features, die als Cluster betrachtet werden sollen. Ein Cluster mit weniger Features als die angegebene Anzahl wird als Rauschen betrachtet.

Suchbereich begrenzen auf (optional)

Die maximale Entfernung, bei der Features demselben Cluster angehören können. Wird nichts angegeben, führt das Werkzeug den Algorithmus "HDBSCAN" aus, wobei Cluster variierender Dichten anhand von unterschiedlichen Entfernungen vom schwächeren Rauschen getrennt werden. Wird ein Wert angegeben, führt das Werkzeug den Algorithmus "DBSCAN" aus, wobei dichte Cluster anhand einer angegebenen Entfernung vom schwächeren Rauschen getrennt werden.

Name des Ergebnis-Layers

Dies ist der Name des Layers, der in Eigene Inhalte erstellt und der Karte hinzugefügt wird. Der Standardname basiert auf dem Werkzeugnamen und dem Namen des Eingabe-Layers. Wenn der Layer-Name bereits vorhanden ist, werden Sie dazu aufgefordert, einen anderen Namen einzugeben.

Über das Dropdown-Menü Ergebnis speichern in können Sie den Namen eines Ordners in Eigene Inhalte angeben, in dem das Ergebnis gespeichert werden soll.
