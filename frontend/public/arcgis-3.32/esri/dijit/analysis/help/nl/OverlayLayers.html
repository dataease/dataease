Overlay lagen

Overlay maakt van twee of meer lagen één enkele laag. Bij overlay lijkt het alsof u door een stapel kaarten kijkt en één kaart maakt die alle informatie bevat die in de stapel is gevonden. Overlay is meer dan het samenvoegen van lijnwerk. Alle attributen van de objecten die deel uitmaken van de overlay, worden overgedragen naar het definitieve product. Overlay wordt gebruikt om een antwoord te bieden op een van de basisvragen in de geografie: "Wat ligt boven op wat?"For example:

  • Welke percelen bevinden zich binnen het 100-jarige overstromingsgebied?(Binnen betekent hierbij hetzelfde als boven op.)
  • Welke wegen bevinden zich binnen welke provincies?
  • Welk bodemgebruik ligt boven op welk bodemtype?
  • Welke bronnen bevinden zich binnen verlaten militaire basissen?

Als Huidige omvang van de kaart gebruiken is aangevinkt, worden alleen de objecten in de invoer- en overlaylaag bovenop geplaatst die binnen de huidige omvang van de kaart zichtbaar zijn. Als deze optie niet is aangevinkt, worden alle objecten in zowel de invoerlaag als de overlaylaag bovenop geplaatst, zelfs als deze zich buiten de huidige omvang van de kaart bevinden.

Choose input layer

The point, line, or area layer that will be overlaid.

Naast het kiezen van een laag uit uw kaart, kunt u Analyselaag kiezen selecteren onderaan de vervolgkeuzelijst. Met deze optie wordt een galerij geopend waarin u kunt bladeren naar en datasets van verschillende bronnen kunt selecteren die u in uw analyse kunt gebruiken.

Kies overlay laag

The layer to overlay on the input layer.

Naast het kiezen van een laag uit uw kaart, kunt u Analyselaag kiezen selecteren onderaan de vervolgkeuzelijst. Met deze optie wordt een galerij geopend waarin u kunt bladeren naar en datasets van verschillende bronnen kunt selecteren die u in uw analyse kunt gebruiken.

Kies overlay methode

The overlay method defines how the input layer and the overlay layer are combined.

  • Intersect—The result will contain only those features or portions of features in the overlay layer that overlap with features in the input layer. Bij kruisende lijn- of gebiedsobjecten hebt u de optie om het type objecten aan te geven dat u als uitvoer wilt.

  • Union—The result will contain features from both the input layer and the overlay layer. The Union option is only available if both the input layer and the overlay layer contain area features.
  • Erase—The result will contain features or portions of features in the overlay layer that do not overlap with the areas in the input layer. De overlaylaag kan gebieds-, lijn- of puntobjecten bevatten.

If Use current map extent is checked, only the features in the input and overlay layer that are visible within the current map extent will be overlaid. If unchecked, all features in both the input layer and the overlay layer will be overlaid, even if they are outside the current map extent.

Resultaat laagnaam

This is the name of the layer that will be created in My Content and added to the map. Deze zal hetzelfde objecttype (gebieden, lijnen of punten) hebben als de overlaylaag. De standaardnaam is gebaseerd op de overlaymethode en de naam van de invoerlaag. Als de laag al bestaat, wordt u gevraagd een nieuwe naam te geven.

Using the Save result in drop-down box, you can specify the name of a folder in My Content where the result will be saved.
