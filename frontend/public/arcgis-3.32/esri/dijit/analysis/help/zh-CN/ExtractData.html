提取数据

使用此工具可以选择并下载特定感兴趣区的数据。所选图层将被添加到 ZIP、CSV 或 KMZ 文件中。

待提取图层

它们是内容表中的图层，可从中提取要素。选中的图层将被提取到结果 ZIP 文件或 CSV 文件中。

为使图层显示在要进行提取的图层列表中，您必须拥有数据或数据的拥有者必须启用导出数据功能。要对拥有的数据启用数据导出：

  • 请打开数据的项目描述。
  • 单击 设置选项卡。
  • 向下滚动至 导出数据部分。
  • 选中 允许其他用户导出不同格式。完成后单击 保存。

如果您的分析图层包含附件，则请注意，仅当输出数据格式为文件地理数据库并且未选中 裁剪要素时才能提取附件。

研究区域

定义输出 ZIP 文件或 CSV 文件将包含哪些要素的区域。可选择的范围有：

  • 地图的当前可见范围（ 与显示相同）。
  • 内容列表中的一个图层。如果所选图层是点或线要素，则图层的范围将用作研究区域。
  • 绘制一个区域以定义研究区域。
将提取所有完全位于地图范围内的要素。对于跨越地图范围边界或与地图范围边界相交但没有完全位于范围之内的要素，您有两种选择：
  • 选择要素 - 将提取跨越研究区域边界的整个要素。
  • 裁剪要素 - 仅提取要素位于研究区域内的相应部分。

输出数据格式

将提取的数据格式。在将线要素或面要素导出为 CSV 文件时，只提取非空间属性。导出点要素时，会同时提取空间和非空间属性。

提取的数据将使用 Web Mercator 1984 投影。对于 CSV 文件，将采用 Web Mercator 1984 坐标将 Lat 和 Long 字段附加到数据。

支持以下输出格式：

  • 逗号分隔值 (CSV) - .csv 或 .zip
  • 文件地理数据库 - .zip
  • Keyhole 标记语言 (KML) - .kmz 或 .zip
  • Shapefile - .zip

输出文件名

此为将添加到 我的内容中的项目的名称。项目名称基于所选的 输出数据格式。如果项目已经存在，则系统会要求您输入其他名称。

使用 将结果保存在下拉框，可以指定 我的内容中的文件夹的名称，结果将保存到该文件夹中。
