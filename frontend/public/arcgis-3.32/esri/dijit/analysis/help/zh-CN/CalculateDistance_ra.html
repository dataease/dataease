计算距离

该工具用于根据单个源或一组源计算欧氏距离、方向和分配。

示例应用包括回答以下问题：

  • 到最近城镇的距离是多少？
  • 到最近道路的距离是多少？
  • 到最近水源的距离是多少？

如果选中 使用当前地图范围，则仅分析当前地图范围中可见的输入图层中的要素。如果未选中，则将分析输入图层中的所有要素，包括超出当前地图显示范围的要素。

选择要素位置图层栅格以计算相应距离

此图层用于定义计算距离的源。该图层可以是影像服务或要素服务。

对于影像服务，输入类型可以为整型或浮点型。

对于要素服务，输入可以为点、线或面。

最大计算距离

要计算的最大距离。例如，如果您提供了值 10,000 米，则将按照源位置到该值（距离）的距离计算欧氏距离。

如果未提供任何值，则将计算到定义范围的距离。

可用单位包括千米、米、英里和英尺。

输出像元大小

输出栅格的像元大小和单位。

单位可以是千米、米、英里或英尺。

结果距离图层名称

将在 我的内容中创建并添加到地图中的图层的名称。默认名称基于工具名称以及输入图层名称。如果该名称的图层已存在，则系统将提示您提供其他名称。

您可以使用 将结果保存在下拉框指定 我的内容中的文件夹名称，结果将保存到该文件夹中。

结果方向图层名称

将要在 我的内容中创建的图层的名称。必须提供 我的内容中当前没有的名称。

您可以使用 将结果保存在下拉框指定 我的内容中的文件夹名称，结果将保存到该文件夹中。

结果分配图层名称

将要在 我的内容中创建的图层的名称。必须提供 我的内容中当前没有的名称。

您可以使用 将结果保存在下拉框指定 我的内容中的文件夹名称，结果将保存到该文件夹中。

选择分配字段

输入源图层上用于保存定义每个源的值的字段。该字段可以是源数据集的整型字段或字符串型字段。

默认值为 Value。如果使用的是要素服务但没有可用的 Value 字段，则将使用第一个可用的整型字段或字符串型字段。如果不存在有效字段，则将使用 ObjectID 字段（例如 OID 或 FID，取决于要素输入的类型）。
