Supervisar vegetación

Esta herramienta realiza una operación aritmética en las bandas de un dataset ráster multibanda para mostrar la información de la cobertura de vegetación del área de estudio.

Seleccione los datos de entrada

Elija la capa ráster multibanda. Asegúrese de que la capa ráster tenga disponibles las bandas apropiadas.

Elija el método para supervisar la vegetación

Elija el método para crear la capa de índice de vegetación. Los diferentes índices de vegetación pueden ayudar a resaltar determinadas entidades o a reducir el ruido.

  • Global Environmental Monitoring Index — El GEMI es un índice de vegetación no lineal para la supervisión medioambiental global desde las imágenes de satélite. Es similar a NDVI, pero es menos sensible a los efectos atmosféricos. Lo afecta el terreno desnudo, por lo tanto, no se recomienda su uso en áreas de vegetación escasa o moderada.
  • Green Vegetation Index - Landsat TM — El GVI se diseño originalmente desde las imágenes Landsat MSS, pero se modificó para utilizarlo con imágenes Landsat TM. También se denomina índice de vegetación verde Landsat TM Tasseled Cap. Este índice de supervisión también se puede utilizar con imágenes cuyas bandas comparten las mismas características espectrales.
  • Modified Soil Adjusted Vegetation Index — El MSAVI2 es un índice de vegetación que intenta minimizar las influencias del terreno del método SAVI.
  • Normalized Difference Vegetation Index — El NDVI es un índice normalizado que le permite generar una imagen que muestra el verdor, o biomasa relativa. Este índice aprovecha el contraste de las características de dos bandas de un dataset ráster multiespectral: las absorciones de pigmento de clorofila en la banda roja y la alta reflectividad de los materiales de las plantas en la banda infrarroja cercana (NIR).
  • Perpendicular Vegetation Index — El PVI es similar a un índice diferencial de vegetación; sin embargo, es sensible a las variaciones atmosféricas. Al utilizar este método para comparar diferentes imágenes, sólo se debe utilizar en imágenes que se han corregido atmosféricamente. Esta información puede proporcionarla el proveedor de datos.
  • Soil-Adjusted Vegetation Index — El SAVI es un índice de vegetación que intenta minimizar las influencias del brillo del suelo utilizando un factor de corrección de brillo del suelo. Esto con frecuencia se utiliza en regiones áridas en donde la cubierta de vegetación es baja.
  • Sultan's Formula — El proceso Fórmula de Sultan utiliza una imagen de 8 bits de seis bandas a la que aplica un algoritmo específico para producir una imagen de 8 bits de tres bandas. La imagen que resulta resalta las formaciones de roca denominadas ofiolíticos en zonas costeras. Esta fórmula se diseñó basándose en las bandas TM y ETM de una escena Landsat 5 o 7.
  • Transformed Soil-Adjusted Vegetation Index — El SAVI transformado es un índice de vegetación que intenta minimizar las influencias de brillo del suelo al asumir que la línea del suelo tiene una intercepción y pendiente arbitraria.

Especificar los índices de la banda NIR y la banda roja

Especifique los índices de banda para las bandas NIR y roja.

Cada sensor de satélite y cámara aérea capturan información que se desglosa en índices de banda. Cada índice de banda contiene la información para una parte específica del espectro electromagnético. Para este método de supervisión de la vegetación, es necesario que especifique el índice de banda que capturó las longitudes de onda de NIR y roja.

Pendiente de línea de suelo

La pendiente de la línea del suelo. La pendiente es la relación lineal aproximada entre las bandas NIR y roja en un gráfico de dispersión.

Este parámetro solo es válido para el método Índice de vegetación ajustado al suelo transformado.

Interceptar

El valor del NIR cuando el valor de reflejo de la banda roja es 0 para las líneas del suelo concretas.

(a = NIR - sRed) , cuando Rojo es 0.

Este parámetro solo es válido para el método Índice de vegetación ajustado al suelo transformado.

Cantidad de cobertura de vegetación verde

Especifica la cantidad de cobertura de vegetación verde, requerida por el método SAVI.

Los valores válidos son los siguientes:

  • 1 = Áreas sin cobertura de vegetación verde
  • 0,5 = Áreas con cobertura moderada de vegetación verde
  • 0 = Áreas con cobertura alta de vegetación verde

Factor de ajuste

Especifica el factor de ajuste para ayudar a minimizar el efecto del suelo, lo cual se requiere para el método Índice de vegetación ajustado al suelo transformado. El valor predeterminado es 0.08.

Un valor bajo significa que está ignorando el efecto del suelo. Un valor alto significa que el suelo afecta al resultado.

Nombre de capa de resultados

El nombre de la capa que se creará en Mi contenido y que se agregará al mapa. El nombre predeterminado se basa en el nombre de la herramienta y en el nombre de la capa de entrada. Si la capa ya existe, se le pedirá que indique otro nombre.
