Convertir entidad a ráster

Esta herramienta convierte entidades en un dataset ráster.

Cualquier clase de entidad que contenga entidades de puntos, líneas o polígonos se puede convertir a un dataset ráster.

El centro de la celda se utiliza para decidir el valor del píxel de ráster de salida.

El tipo de campo de entrada determina el tipo de ráster de salida. Si el campo es entero, el ráster de salida será entero, si es punto flotante, la salida será punto flotante.

Elija la capa de entidades para convertir

La capa de entidades de entrada que se va a convertir a un dataset ráster.

Elegir campo

Elija el campo que se utilizará para asignar los valores al ráster de salida.

Tamaño de celda de salida

El tamaño de celda y las unidades para los rásteres de salida.

Las unidades disponibles son: millas, pies, kilómetros y metros. El valor predeterminado es pies.

Nombre de capa de resultados

El nombre de la capa que se creará en Mi contenido y que se agregará al mapa. El nombre predeterminado se basa en el nombre de la herramienta y en el nombre de la capa de entrada. Si la capa ya existe, se le pedirá que indique otro nombre.

Puede especificar el nombre de una carpeta de Mi contenido en la que se guardará el resultado con el cuadro desplegable Guardar el resultado en.
