Convertir le raster en entité

Cet outil permet de convertir un raster en jeu de données d'entité, tel que des points, lignes ou polygones.

Le paramètre Choisir un champ vous permet de choisir quel champ attributaire du jeu de données raster en entrée deviendra un attribut de la classe d'entités en sortie. Ce paramètre est disponible uniquement pour les rasters entiers qui possèdent des attributs supplémentaires dans la table associée.

Le raster en entrée peut avoir n'importe quelle taille de cellule.

Les rasters entiers peuvent être convertis en entités ponctuelles, linéaires ou surfaciques. Cependant, les rasters à virgule flottante peuvent uniquement être convertis en entités ponctuelles.

Si l'option Utiliser l'étendue actuelle de la carte est sélectionnée, seules les cellules raster qui sont visibles au sein de l'étendue actuelle de la carte seront converties. Si l'option n'est pas sélectionnée, toutes les cellules de la couche raster seront converties, même si elles se trouvent en dehors de l'étendue actuelle de la carte. Tenez compte de cette information si la couche raster est volumineuse ou si sa résolution est très élevée.

Voici certains cas d'utilisation de cet outil :

  • Créez des entités surfaciques à partir d’un raster de classification de l’utilisation du sol.
  • Les cours d'eau identifiés dans un MNT peuvent être convertis en entités linéaires.
  • Un MNT peut être converti en entités ponctuelles pouvant servir dans un workflow non raster.

Choisir la couche raster à convertir

La couche raster en entrée qui sera convertie en couches d'entités.

Choisir un champ

Indiquez un champ qui précise les valeurs à utiliser pour la conversion.

Il peut s'agir d'un entier ou d'une valeur textuelle.

Un champ contenant des valeurs à virgule flottante peut uniquement servir si la sortie est destinée à être un jeu de données ponctuelles.

Le choix par défaut consiste à utiliser le champ Value qui contient la valeur dans chaque cellule raster.

Choisir le type en sortie

Choisissez le type de géométrie parmi le point, la ligne ou le polygone.

  • Point : le raster sera converti en jeu de données ponctuelles.
  • Ligne : le raster sera converti en jeu de données d'entité linéaires.
  • Polygone : le raster sera converti en jeu de données d'entité surfaciques.

Simplifier des lignes ou des polygones

Indiquez si les entités doivent être simplifiées.

Si cette option est sélectionnée (par défaut), les entités linéaires ou surfaciques sont lissées pour produire un résultat plus généralisé. Si cette option n'est pas sélectionnée, les entités linéaires suivent les limites des cellules du jeu de données raster.

Nom de la couche de résultat

Le nom de la couche qui sera créée dans la page Mon contenu et ajoutée à la carte. Le nom par défaut repose sur le nom de l'outil et sur le nom de la couche en entrée. Si la couche existe déjà, vous êtes invité à fournir un autre nom.

Vous pouvez spécifier le nom d’un dossier dans My Content (Mon contenu), dans lequel le résultat sera enregistré via la zone de liste déroulante Save result in (Enregistrer le résultat dans).
