Monitorizare vegetaţie

Acest instrument efectuează o operaţie aritmetică a benzilor unui strat tematic raster cu benzi multiple pentru a afişa informaţiile privind acoperirea vegetaţiei în zona de studiu.

Selectare date de intrare

Alegeţi stratul tematic raster cu benzi multiple. Asiguraţi-vă că rasterul de intrare are benzile corespunzătoare disponibile.

Selectare metodă de monitorizare vegetaţie

Alegeţi metoda pentru a crea stratul tematic de indexare a vegetaţiei. Indecşii diferiţi pentru vegetaţie pot contribui la evidenţierea anumitor obiecte spaţiale sau pot contribui reducerea zgomotului variat.

  • Global Environmental Monitoring Index — GEMI este un index non-linear pentru vegetaţie destinat monitorizării globale a mediului prin imagini satelitare. Este similar cu NDVI, dar este mai puţin sensibil la efecte atmosferice. Este afectat de solul gol; prin urmare, nu se recomandă utilizarea în zonele cu vegetaţie rară sau moderată.
  • Green Vegetation Index - Landsat TM — GVI a fost iniţial proiectat din imaginile Landsat MSS însă a fost modificat pentru utilizare cu imaginile Landsat TM. Este cunoscut şi sub numele de index de vegetaţie verde Landsat TM Tasseled Cap. Acest index de monitorizare poate fi, de asemenea, utilizat cu imagini ale căror benzi au aceleaşi caracteristici spectrale.
  • Modified Soil Adjusted Vegetation Index — MSAVI2 este un index de vegetaţie care încearcă să minimizeze influenţele solului gol din metoda SAVI.
  • Normalized Difference Vegetation Index — NDVI este un index standardizat care vă permite să generaţi o imagine care afişează intensitatea verdelui sau biomasa relativă. Acest index profită de contrastul caracteristicilor celor două benzi dintr-un set de date raster multispectrale: absorbţiile pigmentului clorofilei din banda roşie şi reflectivitatea ridicată a materialelor plantelor din banda aproape infraroşie (NIR).
  • Perpendicular Vegetation Index — PVI este similar cu un index de vegetaţie de diferenţă; totuşi, este sensibil la variaţiile atmosferice. Atunci când utilizaţi această metodă pentru a compara imagini diferite, ar trebui utilizate doar imagini care au fost corectate atmosferic. Aceste informaţii pot fi oferite de furnizorul de date.
  • Soil-Adjusted Vegetation Index — SAVI este un index de vegetaţie care încearcă să minimizeze influenţele strălucirii solului utilizând un factor de corecţie a strălucirii solului. Acesta este adesea utilizat în regiuni aride în care acoperirea vegetaţiei este redusă.
  • Sultan's Formula — Procesul Formula lui Sultan foloseşte o imagine de 8 biţi cu şase benzi şi aplică un algoritm specific pentru a produce o imagine de 8 biţi cu trei benzi. Imaginea rezultată evidenţiază formarea de roci denumite ofiolite pe ţărmuri. Această formulă a fost concepută în baza benzilor TM şi ETM ale scenei Landsat 5 sau 7.
  • Transformed Soil-Adjusted Vegetation Index — Indexul de vegetaţie ajustat cu solul transformat (Transformed-SAVI) este un index de vegetaţie care încearcă să minimizeze influenţele strălucirii solului asumând că linia solului are o interceptare şi o înclinaţie arbitrară.

Specificarea indecşilor benzilor NIR şi Roşu

Specificarea indecşilor de bandă pentru benzile NIR şi roşii.

Fiecare senzor de satelit şi fiecare cameră aeriană capturează informaţii care sunt împărţite în indecşi de bandă. Fiecare index de bandă conţine informaţiile pentru o parte specifică a spectrului electromagnetic. Această monitorizare a vegetaţiei necesită specificarea indexului de bandă care a capturat lungimile de undă NIR şi Roşu.

Panta liniei solului

Panta liniei solului. Panta este relaţia liniară aproximativă dintre benzile NIR şi roşii pe un grafic de dispersie.

Acest parametru este valid doar pentru metoda Indice de vegetaţie ajustat cu solul transformat.

Interceptare

Valoarea pentru NIR atunci când valoarea de reflexie pentru banda roşie (Roşu) este 0 pentru liniile specifice ale solului.

(a = NIR - sRed) , când roşu este 0.

Acest parametru este valid doar pentru metoda Indice de vegetaţie ajustat cu solul transformat.

Valoare acoperirii cu vegetaţie verde

Specifică cantitatea acoperirii cu vegetaţie verde, care este necesară metodei SAVI.

Valorile valabile sunt următoarele:

  • 1 = Zone fără acoperire cu vegetaţie verde
  • 0,5 = zone cu acoperire moderată cu vegetaţie verde
  • 0 = zone cu acoperire ridicată cu vegetaţie verde

Factor de ajustare

Specifică factorul de ajustare pentru a ajuta la reducerea la minimum a efectului de sol, care este necesar metodei Indice de vegetaţie ajustat cu solul transformat. Valoarea implicită este 0.08.

O valoare mai mică înseamnă că ignoraţi efectul solului. O valoare ridicată înseamnă că solul va afecta rezultatul.
